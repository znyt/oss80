　　　　佘山高峰之上，两条虚影显化，对持了起来。
<br/>　　与此同时，两道强大气息直冲云霄，顿时林间群鸟腾飞而起，各种生物宛如世界末日来临一般的疯狂逃离，远离这片区域。
<br/>　　“不跑了？”
<br/>　　蛇精眼神含着高高在上的鄙夷，蛇眸灼灼发光的看着对方的那条金银蛇。
<br/>　　涂小安冷淡无波的开口：“这是我的地盘，我跑什么，到是你，怕是要客死异乡”
<br/>　　“嘎嘎...大言不惭，小土蛇，想必你还不知道本仙的手段”蛇精眼中透着强大的自信，它们巴蛇一族，在上古就是蛇族一脉的霸主，什么蛇敢与它们争锋，简直就是作死。
<br/>　　涂小安不想与它废话，这场仗避免不了，除非蛇精肯灰溜溜的滚出白镇的地界。
<br/>　　“来吧”
<br/>　　只见金银蛇气息暴涨，体型随之拔高，扶摇直上，顷刻间变成了十二米长的巨蟒，俯视蛇精。
<br/>　　蛇精看到此一幕，神情自若，眸中闪过轻蔑，讥笑道：“敢与我巴蛇一族比大小，你还是第一个”
<br/>　　涂小安自然不是跟对方比大小，他就想看看传说中的上古蛇种体型能达到何种的地步。
<br/>　　嗷...
<br/>　　一道似龙吟声冲天而起，惊的佘山的无数生物都惶恐不安，瑟瑟发抖。
<br/>　　霎时间，只见虚空中，一条三十米的蟒蛇显化，血盆大口张开，势要将上空的园月都给吞下，一股强风骤然席卷，搅动风云，此片林间仿佛引来了龙旋风，无数尘土飞扬，风吹过，卷起了漫天红叶，天地间充满了阴冷肃杀之意。
<br/>　　三十米，我的天。
<br/>　　涂小安呆若木鸡，完全始料未及。
<br/>　　他这条十二米的巨蟒与之一对比，就什么都不是了。
<br/>　　黑蛇，青首，食象。
<br/>　　这尼玛的何止可以吞象，吞恐龙都可以了吧。
<br/>　　太夸张了，整整三十米，这是一个何等的概念。
<br/>　　人类在这种生物面前，就跟蝼蚁似的。
<br/>　　果然不愧是上古蛇种，天赋异禀，涂小安试想，就算自己跟对方同等级，他的体型也不可能成长到三十米巨大。
<br/>　　而且他还是被注射了巨蟒基因的，不然现在连十二米都没有。
<br/>　　不论是等级，还是体型，涂小安这条蛇全方位被碾压。
<br/>　　可以说他一点胜算都没有。
<br/>　　要换平日，涂小安还真要跑路，但他现在还有一手依仗，他身负雷电能力，威力强大，未必就输定了。
<br/>　　况且白镇的地界上，他怎么能输。
<br/>　　“小土蛇，乖乖将我孩儿交出来，臣服于我，或许本仙还能大发慈悲的饶你一命”
<br/>　　三十米的巨蛇血盆大口一张，简直昏天黑地，发出了沙哑的声音，威势赫赫，佘山中的毒蛇，早就吓的匍匐在地，瑟瑟发抖。
<br/>　　涂小安一脸的严峻，周身细微电光窜出，在周身闪烁。
<br/>　　“废话真多，吃我一雷”
<br/>　　轰隆隆的惊雷声乍现，只见三十米蛇精的头顶光芒照耀，一道粗大的雷电狠狠的顺势而下，劈了下去。
<br/>　　又是雷？
<br/>　　蛇精心中鄙夷，它之前附身在人类的体内，雷电让他有点惧怕，因为人类的身体实在太弱了。
<br/>　　可此时此刻，它恢复本体，三十米巍峨的蛇躯，无坚不摧，无法无天，区区一道雷电，算的了什么。
<br/>　　蛇精连闪躲的念头都没有，任由这道雷电落到了自己庞大的蛇躯上，另外就算它想闪躲也做不到，蛇躯太大，岂能快的过雷电。
<br/>　　“滋滋...”
<br/>　　三十米的蛇精浑身一股电流流转全身，它青黑赤色的鳞片上徐徐白烟燎燎升起。
<br/>　　“就凭这雷的威力，也想伤到本仙，痴心妄想”
<br/>　　蛇精眼中一抹凶光，浑身一震，地面崩裂开一条缝隙，身上流转的雷电骤然溃散。
<br/>　　看来自己要携体内全部的电力同时释放，才有可能重伤的了对方。
<br/>　　涂小安当然不会天真的以为一道雷电就将蛇精伤到，不过试一试罢了。
<br/>　　“小土蛇，还是看看本仙的逆天手段吧”
<br/>　　“水龙”
<br/>　　蛇精发出山呼海啸的声音，只见它的血盆大口一张，一道滚滚滔天的水柱席卷而出。
<br/>　　滚滚水柱横穿虚空，形成龙形状，带着不可阻挡的威势吞噬而来。
<br/>　　“水系属性能力？”
<br/>　　涂小安瞬间想起洞察蛇精后的资料，觉醒生物，各自拥有各自的手段，但能拥有属性能力，是非常的了不得。
<br/>　　他就算身负系统，也是吞噬了变异鸡冠蛇，完成了第一阶段的进化，才赋予了雷电属性能力。
<br/>　　此水龙汹涌急速，眼看就要将十二米长的巨蟒吞噬，涂小安不傻，连忙选择躲闪。
<br/>　　可就在这时，蛇精的巨大瞳孔一抹狡黠闪过.
<br/>　　只见涂小安的地面猛然裂开，一道水柱冲天而起，直接将他给冲到了天上去.
<br/>　　“不好，糟糕”
<br/>　　刚被地表的水柱冲上去，迎面便是汹涌澎湃的水龙吞噬而来。
<br/>　　“轰...”
<br/>　　整整十二米的巨蟒瞬间被击飞出去，折断了林间不知道多少颗大树，最后轰然倒地。
<br/>　　兵无常势，水无常态。
<br/>　　霎时间，涂小安刚刚轰然倒地，附近十几道水柱冲天而起，猛烈上涌，声势浩大，不可阻挡。
<br/>　　还没等涂小安反应，十几道水柱汇聚成一道洪流，排山倒海的朝着地面的那条十二米长的蟒蛇淹没过去。
<br/>　　“我靠，好强的水系能力”
<br/>　　惊叹一声之后，十二米长的巨蟒被淹没了。
<br/>　　这片林间，风起水涌，大风刮起，水波汹涌，海啸山崩，仿佛经历了一场洪水洗礼。
<br/>　　“蛇王？嘎嘎，不过小土蛇而已，尔等米粒之光，也敢与日争辉”
<br/>　　三十米的蛇精巨大的头颅立于虚空之上，看着金银蛇被自己的洪流吞噬，轻蔑的说出。
<br/>　　可忽然，它...
<br/>　　嗯？
<br/>　　这时，它看到眼前一道影子显化，一条一米长的金银蛇凭空出现，霸气凌然，悬浮的高度，与它同等。
<br/>　　“呦，小土蛇，还没被淹死啊”蛇精一声戏谑，浑不在意。
<br/>　　金银蛇小小样子直视三十米蛇精，淡淡的说了一句：“多谢你帮我洗了一次澡，很舒服”
<br/>　　蛇精闻言勃然大怒，空间传出阵阵爆响声：“小土蛇，你是真想死吗，吞了你，你给本仙塞牙缝都不够”
<br/>　　“你被我吞了，刚好管饱”
<br/>　　“找死”
<br/>　　蛇精血盆大口一张，滚滚水柱席卷了出来，声势浩大。
<br/>　　就在这时，涂小安蛇眸猛然一亮，等的就是现在，全身电光涌现，大喝一声：“雷霆之怒”
<br/>　　顷刻间，他体内的雷电之力卸去了一大半。
<br/>　　只见金银蛇的身上亮的刺眼，一道庞大的雷电直径的劈到喷发而来的水柱上。
<br/>　　这滚滚水柱还没从蛇精的血盆大口中离体，雷电直接顺着水柱贯穿到了三十米蛇精身上。
<br/>　　水导电，威力更大，这也是涂小安故意激怒蛇精的原因。
