        &lt;h3&gt;第三卷第三百二十八章  联谊会&lt;/h3&gt;
<br/>
<br/>        推广会结束，陈子昂他们会搞一个联谊活动的，公司的高层和各界人士齐聚一堂，尤其是落家总裁和吴家发来的贺词让陈子昂水涨船高，很多人都围在陈子昂的身边。
<br/>
<br/>        陈子昂的推广会也引起了各方面的关注，陆家人也关注了陈子昂的推广会。
<br/>
<br/>        陆宇霆看着新闻里的陈子昂，心里说：“我知道你一定会成功的，你也会越来越好的，我也做到了不去想你，好好的爱着我的爱人，可是我的心里总是缺一点东西。子昂，我也想去祝贺你的成功，可是我不能，我只能在这里默默的祝福你，只要你安好，一切都值了。”
<br/>
<br/>        陆战君看着新闻里落家和吴家的祝词，冷冷一笑：“果然他们是一体的，落家和吴家真是好本事，看来他们是在培养他们的团队啊。为了霆儿我是不会让你们如愿的！”
<br/>
<br/>        他拨通了张沛尧的电话：“老张啊，看新闻了吗？”
<br/>
<br/>        张沛尧看着新闻叹了口气，落家和吴家这是将陈子昂推到了风口浪尖上啊。
<br/>
<br/>        “新闻怎么了？我这今天忙的不行，还没有来得及看呢。”
<br/>
<br/>        “你那好前儿媳妇真的是好本事啊，你看看她身边聚集了多少年轻人，都是世家豪族啊！而且落家和吴家都发祝贺词了。”陆战君语气不善。
<br/>
<br/>        “啊，是吗？那我们是不是也应该祝贺一下啊，现在还来得及吗？”张沛尧说。
<br/>
<br/>        “你？”陆战君生气了。
<br/>
<br/>        “我说陆兄，据我所知，宇霆和陈子昂也是朋友，只是宇霆结婚之后他们就没有来往了，小函又是孩子的爸爸，其实你可以想一想，我们还是可以争取的。从亲情上来说，我们要比他们近一点，陈子昂那姑娘还算善良，这几年本本分分，你说呢？还有啊，我们去了，吴家和落家是不是也认为陈子昂和我们在一起？”
<br/>
<br/>        “老张啊，还是你考虑的周到，那我们下班之后过去。”
<br/>
<br/>        “下班？”
<br/>
<br/>        “是啊，让宇霆、润竹和小函下班之后过去参加他们的联谊会。”陆战君笑了。
<br/>
<br/>        “行，你给他们交代一下。”张沛尧看着窗外的秃枝，烦躁的挂了电话。
<br/>
<br/>        张函一直闹着要去甸城，他们压下来了，弥青云将张函的身份证偷偷的收起来，要去补办身份证也得一个过程，所以这几天张函在和他们置气中。
<br/>
<br/>        他们知道，迟早会要有面对的一天，但不是在现在啊，他和张函的关系一直都没有缓和，反而越来越僵了。
<br/>
<br/>        焦虑的张沛尧给弥青云拨了个电话：“青云啊，陆战君要小函和陆宇霆夫妇去参加陈子昂晚上的联谊会，你叮咛一下小函，千万不要表现的太过明显啊。”
<br/>
<br/>        “这样不是将他们的关系公布于众吗？”弥青云担心的说。
<br/>
<br/>        “所以我提醒你让小函不要表现太过明显啊。”
<br/>
<br/>        “行，我给他打电话。”
<br/>
<br/>        .......
<br/>
<br/>        陈子昂的联谊会在万华酒店举行，除了白天的人员外，还有一些人也来参加，联谊会顾名思义就是大家在一起看看彼此有没有商机合作。
<br/>
<br/>        晚上的酒会陈子昂做了简短的发言，在发言的时候陆宇霆挽着吴润竹和张函一起走进大厅。
<br/>
<br/>        陈子昂看了一眼进来的三个人，有一瞬间的失神。
<br/>
<br/>        张函在西城分手之后他们就没有联系，张函看起来越发的消瘦了。
<br/>
<br/>        陆宇霆更是从分手之后再也没有联系，挽着他的人就是京城四美之一的吴润竹，吴润竹看起来确实很漂亮。
<br/>
<br/>        陆宇霆也是意气风发的样子，张函和陆宇霆从进门之后眼神就落在了陈子昂的身上。
<br/>
<br/>        陈子昂头发盘起来了，露出了白皙的脖子，长长的吊坠耳环垂在两侧，v领的玉白色毛衣得体穿在她的身上，直筒裤、低跟黑色皮鞋，棕色的长款大衣，显得时尚而干练。
<br/>
<br/>        江俞轩也看到了张函和陆宇霆，心里立马拉起了警戒线，他对寒伯安说；“大哥，这两个人好像没有邀请，这不请自来怕没安好心。”
<br/>
<br/>        “我和长卿去应付陆家夫妇，你去应付张函。”寒伯安低声和江俞轩商量着。
<br/>
<br/>        “文文，你和思宇还有安男、宫阳帮着招呼一下别人。”寒伯安对郝景文说并深深的看了一眼宫阳。
<br/>
<br/>        “欢迎今天到来的贵客，我的朋友陆宇霆夫妇及张函，大家给一点掌声，虽然他们没有参加今天的推广会，晚上能来我们的联谊会，子昂非常开心！”陈子昂在话筒前大声的说，并且带头鼓掌！
<br/>
<br/>        江俞轩和寒伯安、李长卿迎了过去。
<br/>
<br/>        江俞轩：“张总，别来无恙？欢迎来我们的推广会！请跟我来！”
<br/>
<br/>        寒伯安：“陆总，什么风把你吹过来了，蓬荜生辉啊，你看这个大厅立马亮如白昼。”
<br/>
<br/>        李长卿压着嗓子说：“吴姐姐，真是很想念你呢，没有想到你今天能来这里，我太开心了。”
<br/>
<br/>        “请，我们这边！”寒伯安做了一个“请”的手势。
<br/>
<br/>        寒伯安并没有过多的和张函打招呼，因为他们的交往都是私底下的，明面上就算不认识，但江俞轩不同，江俞轩的身份陆家人应该知道。
<br/>
<br/>        “江总，好久不见啊！”陆宇霆和江俞轩打着招呼。
<br/>
<br/>        “是啊，好久不见，承蒙陆总照顾多年，陆总结婚的时候，俞许因为有事，没有去参加你的婚礼，这是嫂子吧，嫂子真漂亮，比我们家倩楠还要好看。”江俞轩抬出了张倩楠，似乎这个时候张倩楠才起到了作用。
<br/>
<br/>        张函看了一眼江俞轩，露出了一丝若有若无的苦笑。
<br/>
<br/>        “你今天没有带倩楠来吗？”吴润竹看了一眼江俞轩，他们是知道张倩楠和江俞轩一直在谈着，但是并不知道俩个人已经领证了。
<br/>
<br/>        在他们的这些关系中，陆家既希望江家和张家能联姻，又不希望张家和陆家联姻，主要还是要看张函及张家的态度。
<br/>
<br/>        “倩楠这两天身体不适，在家休养着呢！再说，他们俩人还没有结婚，这种场合出面不太好。”张函替江俞轩回答了。
<br/>
<br/>        “结婚还不是迟早的事情？”吴润竹微微一笑。
<br/>
<br/>        陈子昂看着张函和陆宇霆他们被带到了座位上，心里一暖，她的这些朋友真的是很暖心。
<br/>
<br/>        陈子昂也过来了，她和大伙坐着说了一会话，然后江俞轩起身去接了个电话，回来对陈子昂说：“子昂，你二嫂冯睿出车祸不在了，我定了10点钟的飞机，你得回西城。”
<br/>
<br/>        “你说什么？”陈子昂吃惊的站起来。
<br/>
<br/>        “你七哥打电话了，昨天的事情，他没有告诉你，怕耽误推广会的召开。”江俞轩解释着。
<br/>
<br/>        “子昂，那你赶快回去吧。逝者为大，这里有我们了！”寒伯安开口了。
<br/>
<br/>        宫阳和安男在别的座位坐着，也在招呼着旁边的人。
<br/>
<br/>        “抱歉，大家，我先走一步。”陈子昂的泪花在眼眶里打转，虽然二嫂经常利用她，但是她从来都没有恨过她，她知道冯睿就是想过得比别人好一些，风光一些。
<br/>
<br/>        这个世界上有很多人，原本初心都是很好的，只是走着走着就偏离方向！
