　　“呵呵，你是说萧家萧炎吧，那小子可是大名鼎鼎啊，现在不过是一个废物，提他干嘛？”

<br/>　　毕竟，那家伙可是拥有主角光环的，既然两大异火就在眼前，自然没有舍近求远的必要。
<br/>　　骨灵冷火在药尘身上，剧情刚开始，自然不可能传给萧炎。
<br/>　　而金帝焚天炎乃是古族传承之火，如今就在古熏儿身上，古族的小公主有斗皇强者在暗中保护，但斗皇又如何，阻挡他完成任务的，皆是敌人。
<br/>　　两种异火他势在必得，骨灵冷火在异火排行榜上排名十一，金帝焚天炎排名第四，若是能够吞掉两种异火，九霄琉璃焱肯定能够提升，自己的修为也能随之更上一层楼。

<br/>　　竖日清晨，萧家测试广场上，周围早已围满了人群。

<br/>　　望着测验魔石碑上面闪亮得甚至有些刺眼的五个大字，少年面无表情，唇角有一丝自嘲，紧握的手掌。
<br/>　　因为力气有些大，而导致略微尖锐的指甲深深的刺进了手掌之中，带来一阵阵钻心的刺痛......
<br/>　　“萧炎，斗之力，三段，级别：低级。”
<br/>　　测试魔石碑前，一位中年男子，看了一眼石碑上所显示出来的信息，语气漠然的将信息公布了出来。
<br/>　　中年男子话语刚落，便是不出意外，在人头攒动的广场上带起了一阵嘲讽的骚动。
<br/>　　“三段？果然不出我所料，这个名动一时的天才，这一年又是在原地踏步啊！”
<br/>　　“哎，这废物真是把家族的脸都给丢光了。”
<br/>　　“要不是族长是他父亲，这种废物，早就被驱赶出家族了，任其自生自灭，哪还有机会待在家族中白吃白喝。”
<br/>　　“唉，昔日那名闻乌坦城的天才少年，如今怎么落魄成这般模样了啊？”
<br/>　　“谁知道呢，或许是做了什么亏心事，惹得神灵降怒了吧...”
<br/>　　周围传来的不屑嘲笑以及惋惜轻叹，落在那如木桩待在原地的少年耳中，恍如一根根利刺狠狠的扎在他的心脏上，使得少年呼吸微微急促起来。
<br/>　　这名少年不是别人，正是萧炎。
<br/>　　慕清霄站在人群的最前方，容貌与人群格格不入，可却不曾有人注意到他，视线落在萧炎身上，神色有些古怪，这小子曾经也是地球人，只是灵魂来到此处而已。
<br/>　　此刻，两人都身在他乡，但以这种方式站在同一个广场上，慕清霄略微觉得有些奇妙。
<br/>　　“真是有意思。”
<br/>　　听着周围的冷嘲热讽，慕清霄负手而立，嘴角牵起一丝笑容，倒不是他幸灾乐祸，而是这种身临其境的感觉，与看书的时候完全不同。
<br/>　　萧焱抬头向四周扫视一眼，将众人冷嘲热讽的嘴脸看在眼中，心中越发的苦涩。

<br/>　　回过神来，撇了撇嘴，暗道：“穿的真是骚包，这家伙真是男的么，怎么长的如此俊美？”
<br/>　　两人的视线在空中交汇，慕清霄报以微笑，萧炎怔了怔，因为他没有从面前的青年眼中看到嘲讽与不屑，而是一丝莫名的古怪，这让他二丈摸不着头脑。

<br/>　　萧炎则站在最后面，眼中满是自嘲，面对族中长老分配资源问题也充耳不闻。
<br/>　　“家主，外面有客人求见。”就在这时，萧家的门卫快速来到大厅禀报。
<br/>　　“客人？”

<br/>　　“......是一位风度翩翩，俊美绝伦的青年，小的将他拦在门外，可他非要见家主，否则就堵在门口不肯离去。”

<br/>　　“原来门卫说的是真的，世间真有如此俊美的男子。”
