        郎安……
<br/>
<br/>        该不会也是狼人吧？
<br/>
<br/>        连星儿一想到这里，莫名其妙地感到一阵毛骨悚然。
<br/>
<br/>        明司寒隐隐察觉到坐在自己身旁的连星儿的身子在打颤，于是偏过头去，看着连星儿的侧颜，关心地问道：“你没事吧？是不是冷？电影院里的空调温度太低了吗？”
<br/>
<br/>        “没有。”连星儿睐向明司寒，咧嘴干笑。
<br/>
<br/>        当她睐着明司寒的同时，目光的焦点顺道落在了坐在明司寒另一侧的郎安身上，身子也在不经意间地跟着微微向前倾了倾。
<br/>
<br/>        只见郎安目不转睛地盯着荧幕上的那个狼人，脸上一如既往的没有任何神情。
<br/>
<br/>        明司寒见连星儿将目光锁在郎安的身上，于是故意往前挪动了一下坐姿，挡住了连星儿的视线。
<br/>
<br/>        连星儿这才回过神来，继续看向前方的大银幕。
<br/>
<br/>        其实，她对郎安的感情比较复杂。
<br/>
<br/>        有忌惮、有同情，也有说不出来的感觉，总而言之，是让她无法做到对视他不见。
<br/>
<br/>        四个人看完电影，从电影院里出来时，已经晚上九点半了。
<br/>
<br/>        回到明司寒的跑车上后，连星儿提出要回祖爷爷家去住。
<br/>
<br/>        而这个时候，明司寒忽然想到了什么，便问道：“你祖爷爷喜欢养花吗？”
<br/>
<br/>        “嗯，喜欢。”连星儿点了点头。
<br/>
<br/>        明司寒随即转头看向后座上的郎安和裴萱妍，淡淡地说道：“我得用这车的后座去拖几盆花，二位，你们就自己拦出租车，各回各家吧！”
<br/>
<br/>        郎安第一个识趣地从敞篷跑车内起身跳下来，裴萱妍倒是打开了车门走了下去。
<br/>
<br/>        明司寒的用意再明显不过了。
<br/>
<br/>        连星儿侧过身去，看了看郎安，又看了看裴萱妍，微笑着打趣道：“你们俩个要相亲相爱哟！不可以再打架了！”
<br/>
<br/>        听着连星儿这话，裴萱妍忍不住对一旁的郎安翻了个白眼。
<br/>
<br/>        郎安依旧面无表情，只是转身离开。
<br/>
<br/>        待连星儿回过头来时，明司寒看向裴萱妍，对她使了个眼色。
<br/>
<br/>        裴萱妍会意地点了下头，也跟着转身离去。
<br/>
<br/>        现在跑车里就只剩下明司寒和连星儿了。
<br/>
<br/>        临海城的夜晚有些凉，明司寒按了开关，将跑车的敞篷合了起来。
<br/>
<br/>        “你怎么突然想起要去买盆栽？”连星儿坐在副驾驶座上，看向明司寒，好奇地问道。
<br/>
<br/>        明司寒微微一笑：“之前不是答应过你，要给你买鲜花的吗？只是刚刚想起，买鲜花容易凋谢，买带盆的鲜花，说不定来年还能再开。”
<br/>
<br/>        “那我们去买牡丹花吧！祖爷爷很喜欢牡丹。”连星儿拍手道。
<br/>
<br/>        明司寒点了点头，突然朝连星儿凑了过去，将她吓了一跳。
<br/>
<br/>        “嗖”地一声，他只是拉过她身侧的安全带，替她系上。
<br/>
<br/>        而刚刚，他凑近她的那一瞬间，差点让她忘记了呼吸。
<br/>
<br/>        “坐稳了，我们要出发了！”明司寒温和地说道。
<br/>
<br/>        连星儿讷讷地点了下头。
<br/>
<br/>        将数十盆五颜六色、花盘硕大、花瓣层叠有致的牡丹花塞满整个跑车的后座，那比满载一车的玫瑰花还要格外引人瞩目。
<br/>
<br/>        明司寒开车将连星儿送到了傅家大庄园的门外。
<br/>
<br/>        他们俩到的时候，因为连星儿事先打了电话，所以门外管家早已安排了家佣候着。
<br/>
<br/>        家佣们将明司寒跑车后座上的牡丹花一一搬进了大庄园内。
<br/>
<br/>        进门前，连星儿还不忘跟明司寒再次说了一声：“生日快乐。”
<br/>
<br/>        明司寒欣喜地扬起嘴角，回了一声“谢谢”后，又说了一句“晚安”。
<br/>
<br/>        就在连星儿转身之际，明司寒忽然要求道：“给我一个拥抱如何？”
<br/>
<br/>        “什么？”连星儿怔愣了一下，以为自己产生了幻听。
<br/>
<br/>        明司寒会心一笑，大步上前，直接将连星儿拥入了怀中。
<br/>
<br/>        他只是单纯地想要抱抱她。
<br/>
<br/>        在连星儿还会缓过神来时，明司寒便已经放开了她。
<br/>
<br/>        “那你一个人开车回学校的路上要注意安全。”连星儿叮嘱道。
<br/>
<br/>        明司寒点了下头，又向大门口迎了迎手：“你先进去吧！我看着你进去。”
<br/>
<br/>        “好。”连星儿抬起手来跟明司寒挥了挥，转过身去后，脸上挂满幸福的微笑。
<br/>
<br/>        连星儿进了大门后，管家将大门合上了。
<br/>
<br/>        此刻，明司寒的手机响了起来。
<br/>
<br/>        他从裤兜里直接掏出手机按下接听键。
<br/>
<br/>        “跟丢了！”听筒里传来了裴萱妍的汇报。
<br/>
<br/>        明司寒不禁皱了下眉头，若有所思地问：“那你之前有查到他的来历吗？”
<br/>
<br/>        “我在你们学校的学生档案室里，并未查到他的学籍资料。而且，他跟连星儿亲近，傅夜沉的手下对他却没有任何动静。或许，是我想多了。他真的只是连星儿的贴身保镖。”裴萱妍揣测道。
