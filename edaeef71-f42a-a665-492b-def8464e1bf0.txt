<br/>　　“混蛋！”
<br/>　　仅剩的两名血瞳族的强者，顿时起的怒吼连连。
<br/>　　但是，两人却是没有丝毫办法，他们根本察觉不到隐藏之人在何地。

<br/>　　老者厉喝一声，浓郁的血光，形成了一柄巨大的血色神剑，撕裂虚空，洞穿虚无，爆射而去。

<br/>　　血色神剑从天穹之上斜斜暴刺而下，速度无与伦比，恐怖的威势，让得下方的空间为之一分，下方的大地瞬间裂开，出现了一条深不见底的大裂缝。
<br/>　　这一剑威势惊天动地，击向一处空无一人之地。
<br/>　　众人见此大惊失色，知道这名血瞳族的武尊境老者，发定然是发现了隐藏之人的藏身之地。
<br/>　　众人暗暗的捏了一把汗，不知道这名隐藏之人，能不能挡住这一击？
<br/>　　此人既然隐藏身形，想必不太可能是武尊境武者，因为若是武尊境武者的话，何必如此畏首畏尾。
<br/>　　但是，若是只是武帝境的武者，怕是很难挡住这一击了。
<br/>　　“不好！”任云都见此，心中暗道不好，但他此刻已然来不及出手阻挡了。
<br/>　　虚空深处，苏莫眸光如电，望着那急速爆刺而来的血色神剑。
<br/>　　杀死众多血瞳族的人，自然就是他。
<br/>　　这柄巨大的血色长剑，虽然威力巨大，但因为对方距离他非常的远，足足数万里，他应该是能躲避开来。
<br/>　　但是，他却感觉自己被血色神剑锁定住了，仿佛无论他如何躲避，都躲避不开此剑。
<br/>　　这种感觉充斥在心底，挥之不去。
<br/>　　毫不犹豫，苏莫身形顿时暴退，向下方不停的暴退，企图尽量的拉开距离。
<br/>　　嗖！
<br/>　　苏莫的速度，亦是奇快无比，瞬息间便进入了大地深处。
<br/>　　与此同时，他立刻展现出了巅峰的战力，准备迎接这一击。
<br/>　　尊者吗？
<br/>　　如此远的距离，他未必挡不住一击！
<br/>　　特殊体质催动了起来，苏莫全身白光、紫光、琉璃之光等各色光芒闪耀而起。
<br/>　　全身变得犹如钢铁浇筑，又如翡翠般晶莹剔透，更有刺目的白光伴随着恐怖的高温升腾而起。
<br/>　　不仅如此，一瞬间，苏莫收起了隐身战魂，动用所以精神力量，催动了十五种剑之战魂，剑威无可匹敌。
<br/>　　他右手中握着金色的尊级宝剑，左手中出现了一块巨大的黑色盾牌。

<br/>　　当他退入地下三万里之时，血色神剑终于临近了他的头顶，狂暴的威势让得大地不断的崩塌。
