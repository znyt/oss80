        &lt;h3&gt;第三卷第二百零五章  温馨的江家&lt;/h3&gt;
<br/>
<br/>        江俞轩在家里呆了几天，陪着父母亲和姐姐，还有那个总是爱吃他“豆腐”的小外甥，一家人也是其乐融融，在忙碌之余有这样的休闲时间享受一下天伦之乐人生何其有幸。
<br/>
<br/>        这几天除了和父亲聊天讨论工作规划就是被小外甥拉着看动画片，不是海底总动员就是喜羊羊和灰太狼，再就是蜡笔小新。
<br/>
<br/>        天空就想是一批锦缎散发着温柔的蓝色，太阳温暖而慵懒的挂在空中，半醉半醒，江俞轩和自己的老父亲在自家的后花园看着草长莺飞，品着香茗。
<br/>
<br/>        江爸：“西城的项目快要进入尾声了吧？”
<br/>
<br/>        江俞轩：“估计还得一两年，杨总催得紧，我们的工人都是加班加点。我后天就得飞西城。”
<br/>
<br/>        “不去北京吗？”江爸低下头喝了口茶。
<br/>
<br/>        江俞轩也拿起茶杯轻轻的吹了一下，浅浅的抿了一口，过了好一会放下茶杯：“不去了，从西城回来的时候杨总就给我三天假，我还是多申请了两天，那个老头子啊，整个就是一个工作狂，西城新区的规划耗费了他全部的心血，两年之后，新区将会大放异彩。”
<br/>
<br/>        江爸：“这么说，我还得好好感谢一下杨总，给了你坚持下来的动力啊。”
<br/>
<br/>        “你这意思对你的儿子不自信。”
<br/>
<br/>        “之前是有点不自信，不过这些年你已经很厉害了，环境还真是能改变一个人，江家大少经过几年的卧薪尝胆已经完全能够独当一面了，我的担子可以卸下来了。”江爸笑着说。
<br/>
<br/>        “别，你还年轻，正是干事业的时候，我还要学习的地方很多，你还是让我多学习一些经验。”
<br/>
<br/>        江爸：“等你西城项目完工了，去别的地方拓展拓展业务。我们的眼光不能只盯着内地，有一些二三线城市的发展空间还是很大的。”
<br/>
<br/>        “我也有同感，一线城市总有饱和量的，城市的发展都是从内向外延伸，我们还是要将投资扩大化，那你什么打算什么时候启动你的计划？”江俞轩想到了陈子昂。
<br/>
<br/>        江爸：“一看你就心不在焉，我都说了等你西城项目结束了就去别的地方拓展业务，你还要问我第二遍。”
<br/>
<br/>        “这脸打的，你能不能给你自己留点面子，说话留一线，下次好相见！”江俞轩哈哈大笑。
<br/>
<br/>        “你这臭小子，你说你心里刚刚在想什么？”江爸爸紧追不舍。
<br/>
<br/>        “什么都瞒不过你，陈子昂今年要去甸城发现信息化，已经谋划了好几年了，而且那边的项目资金申请也下来了，你一说去别的地方拓展业务我自然而然就想到了她。”江俞轩不打算隐瞒父亲，反正眼前的这个人对他的心思是知道的透透的。
<br/>
<br/>        “这么说，你是想追着陈子昂去甸城发展？”江爸爸眉头紧皱。
<br/>
<br/>        “张家和陈子昂渊源颇深，我记得张倩楠也说过，今年他们兄妹有可能会去甸城。”江俞轩解释着。
<br/>
<br/>        “那么你告诉我，你是追随陈子昂去的还是追随张倩楠去的？我们江家这么些年并没有依附任何势力，对于你的婚事，我也不强迫你，但是我不希望你摇摆不定!”江爸语气已经可以用严厉来形容了。
<br/>
<br/>        “每个地方都有其发展的独特性，甸城也不是不可以去，那里独特文化氛围和地理位置也是可以尝试的，我们是生意人，哪里有生意就往哪里去，谈何追随？对于婚姻之事，我并不着急，是你要一而再再而三的催促我结婚。”江俞轩沉默了一会：“和谁结婚不是结呢？”
<br/>
<br/>        “那么你到底爱不爱张倩楠？”
<br/>
<br/>        “和张倩楠谈了这么多年，她无可挑剔，我想她应该就是最合适做妻子的人选吧，余生有她相伴，虽然平淡但是也相安。”江俞轩突然发现，他真的和张倩楠的感情平平淡淡不温不火，就连拉手都很少，感情升温的那一段时间也就是张倩楠在西城，回到家里俩个人最亲密的举动就是偶尔蜻蜓点水的在各自的脸颊上碰一下。
<br/>
<br/>        虽然他有那种自家的水蜜桃挂在树上吃不到嘴的遗憾，但是他从来都没有动过别的心思，他们都是成年人了，这样真的好吗？
<br/>
<br/>        “你和张倩楠有没有在一起？”
<br/>
<br/>        “还没有。”江俞轩老实的回答。
<br/>
<br/>        “外界传言是真的啊，没想到你这个臭小子还很痴情。”
<br/>
<br/>        “万花丛中过，片叶不沾身！”
<br/>
<br/>        “姥爷，舅舅！”远处跌跌撞撞跑过来一个三岁大的小男孩。
<br/>
<br/>        “宝贝，快到姥爷这里来。”江爸伸开双手准备迎接跑过来的小男孩。
<br/>
<br/>        “舅舅。”小男孩一头扎进了江俞轩的怀里。
<br/>
<br/>        “姥爷，我要和舅舅看动画片去。”小男孩的头埋在江俞轩怀里，奶声奶气的说着话。
<br/>
<br/>        “你又将鼻涕蹭我一身。”
<br/>
<br/>        “谁让你是大灰狼，你抓的羊呢？”小男孩扭动着身子，抬起头，明亮的眼睛看着江俞轩，完全无视旁边的姥爷。
<br/>
<br/>        “那我去给你抓羊去？”小男孩及时的到来打断了江俞轩父子的谈话，也打断了江俞轩向更深层次的思考。
<br/>
<br/>        “舅舅，我是小宝宝，你是大灰狼，你的红太狼呢？”小男孩拉着舅舅的手，俩个人华丽丽的离开，留下江爸眼巴巴的看着一大一小走远。
<br/>
<br/>        “还灰太狼呢，连个红太狼都没有！”江爸爸嘀咕着，江俞轩回头看了一眼江爸，微微一笑。
<br/>
<br/>        “舅舅，抱。”走了几步，小男孩就不愿意挪动脚步，定定的看着江俞轩。
<br/>
<br/>        “自己走，你才走了几步路？”江俞轩迈开长腿，小男孩抱住他的腿，“舅舅，抱，舅舅抱，抱，小宝好累啊，小宝才三岁。”
<br/>
<br/>        “三岁不是更应该走路吗，小宝乖，小宝要好好锻炼怎么走路。”江俞轩蹲下身子，抱起耍赖的小宝。
<br/>
<br/>        小宝将眼睛一眨，立马满脸的泪水，伸出小胖手，环住江俞轩的脖子，将满脸的泪水蹭了江俞轩一脸。
<br/>
<br/>        “舅舅，羞羞，弄丢了红太狼，哭唧唧。”
<br/>
<br/>        “小心红太狼来抓你。”江俞轩威胁小宝，那一脸严肃的表情让小宝“哇”的一声哭出来。
<br/>
<br/>        这一次是真的哭了，眼泪鼻涕一大把，蹭在江俞轩的肩头。
<br/>
<br/>        “不准哭了，在哭我就不陪你看海底总动员了。”
<br/>
<br/>        “不，我要看蜡笔小新、我还要看猫和老鼠，我还要看柯南。”
<br/>
<br/>        “你还要看柯南？你看得懂吗？”
<br/>
<br/>        “有舅舅在，小宝都能看得懂！小宝都要看！”
<br/>
<br/>        “好，陪你看！”
<br/>
<br/>        江俞轩回到了童年，看了两天动画片，又踏上了去西城的路。
