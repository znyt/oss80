　　村子中剩下的那些毒贩眼看大势已去，直接选择了投降，他们明白继续抵抗已经毫无意义，只能是白白送命。
<br/>　　先前双方在交火的时候，现场一度非常混乱，司徒空也只是看到了佟千雪被布克尔特所带走，但是被带去了哪里却没有发现。
<br/>　　搜！就是挖地三尺也要把人给找出来！
<br/>　　找不到布克尔特，找不到佟千雪，这次的行动就不算是成功的。
<br/>　　比洛多的精锐机动部队和战术中队的人员加起来总共超过百人，不过想要将这个村子彻底搜遍，还是需要一点时间的。
<br/>　　村子后面的矿洞并没有做什么伪装，所以很容易就被在村子中搜索的机动部队队员发现。
<br/>　　三名机动部队队员互相看了眼，迈步朝着矿洞走去。这里的矿藏丰富，锂矿存储量更是位居世界第一，全国有数不清被开采过的矿洞，如果一个村子中能发现金矿或者银矿，那么这个可能原来只是依靠种植烟草、咖啡、棉花作为主要经济收入的村子，可以在一夜之间发生翻天覆地的变化；当然也有可能会给这个村子惹来灾祸。
<br/>　　所以在这里看到这么一个矿洞，三名机动部队队员并不感到意外。
<br/>　　矿洞内黑漆漆的一片，进入矿洞的时候，三名机动部队队员眼睛没能立刻适应矿洞里黑暗的环境。
<br/>　　进入矿洞的机动部队队员便没有留意到那根横在距离地面只有三厘米高的细钢丝，打开了安装在M16A3自动步枪战术导轨上的战术手电，观察起矿洞中的情况，很快就注意到了角落中还处于昏迷中的娜塔莉娅。
<br/>　　“那里有人！”
<br/>　　左侧的机动部队队员在看到娜塔莉娅后，立刻朝着倒在地上的娜塔莉娅走去，因为一直没有注意到地面上那根钢丝绊线的关系，所以这名这名机动部队队员一脚迈出去的时候，踢到了地面上的钢丝绊线。
<br/>　　察觉到自己脚上阻力，低头观察的时候，看到的是反步兵地雷的安全插销从撞针保险孔中飞出，以及失去安全插销的撞针撞击下去。
<br/>　　“快走！”
<br/>　　那名机动部队队员大惊失色，提醒身边的其他队员撤离出矿洞，可是在时间上，根本不允许他们这么做。
<br/>　　轰——！
<br/>　　矿洞中的地雷、手雷和炸药被引爆，矿洞发生了剧烈的爆炸，整座矿山和地面都在晃动，矿山上的石块滚落下来。
<br/>　　娜塔莉娅一心帮助佟千雪为的就是要获得自由，可是她最后却还是没能得到她想要的自由……
<br/>　　“是矿山那边，过去看看！”
<br/>　　矿山那边发生的爆炸令得罗昊心头“咯噔”一跳，立刻带着战术中队的人赶往矿山的方向，可是赶到矿山的时候，看到的是被石块堵住了洞口的矿洞。
<br/>　　整个矿洞都被炸碎的石块堵得严严实实，根本看不到里面的情况，短时间内是不可能清除干净的，而且从刚才爆炸的情况来看，很有可能矿洞内部都被石块给堵死了，清理的工作所需要的时间可能更长。
<br/>　　“布克尔特不会是躲在这个矿洞里被炸死了吧？那佟医生……”
<br/>　　周立看了眼站在旁边慕容飞，看到他的脸色并不算好看，把后面半句话又给重新咽了回去。
<br/>　　“你是不是蠢？！”陈应天翻了个白眼，走到周立旁边，在他头上拍了一下，说道：“说你是笨牛，你还真是笨。如果是你躲在矿洞里面，你会在矿洞里埋炸药和地雷，然后等着别人冲进来引爆地雷和炸药把自己活埋在一堆乱石之下？”
<br/>　　“有道理。”周立挠了挠头，点头道。
<br/>　　“这个矿洞应该是被打通的。”罗昊看着那堆把矿洞堵死的石头，说道：“布克尔特应该是早就有所准备，他在这里准备了交通工具，他埋下地雷和炸药后，然后再带着佟千雪使用交通工具离开。如果没有猜错，这个矿洞应该是通道这座矿山的背后，上直升机到这座矿山背后去，找到这个矿洞的出口。”
<br/>　　“明白！”
<br/>　　战术中队朝着停在村子空地上的直升机跑去。
<br/>　　直升机起飞，带着战术中队朝着矿山背后飞去。
<br/>　　齐凡坐在直升机上，探头往外看了眼，按住耳麦对罗昊说道：“狂龙，矿山背后是一片山林，根本看不到山林中的情况。”
<br/>　　“让直升机降落吧。”罗昊按着耳麦对战术B组和战术C组的人吩咐道：“我们准备下去！所有人注意，进入山林一定要注意安全，布克尔特是前‘绿色贝雷帽’特种部队队员，而且根据凤凰姐的情报，布克尔特在‘绿色贝雷帽’的时候，在只负责美国政府‘黑活湿活’的部队服役，表现很出色，大家一定要提高警惕。”
<br/>　　三架“黑鹰”直升机开始下降高度，悬停在山林上空，每架直升机上各抛下两根索降绳，战术中队的队员抓着索降绳索降进山林，并且在山林中构建起防御圈。
<br/>　　战术中队进入山林后，三架“黑鹰”直升机重新拉升高度，从山林上空离开，他们还需要把机动部队送到这片山林中来。
<br/>　　罗昊做个手势，示意战术中队队员散开搜索，很快齐凡就找到了在矿山背后开出来的洞口，立刻通过耳麦通知了罗昊。
<br/>　　齐凡指了指那个黑漆漆的洞口，对罗昊说道：“我进去那里看过，里面被石块堵死了，这个洞口应该就是跟村子里那个矿洞连通的，地面上有轮胎印，应该是越野摩托车留下的，留下的时间不超过一个小时，有两辆。现在看来，从矿洞中离开的有两辆越野摩托，从村子里逃跑的，除了布克尔特和佟医生以外，应该还有其他人。”
<br/>　　“沿着轮胎印追，一定要追到布克尔特！”罗昊对着通向山林深处的越野摩托轮胎印，拳头缓缓握紧。
<br/>　　汽车不能在茂密的山林中行驶，但是越野摩托可以，罗昊明白他们必须要加快速度追击，不然很有可能就被布克尔特带着佟千雪跑得无影无踪。
