　　萧逸看着廖灵儿离去的身影，皱了皱眉。
<br/>　　不过，他的身影可没有停下，而是继续前行。
<br/>　　一路上，击杀火焰妖兽皆是轻而易举。
<br/>　　而击杀火焰妖兽所得的奖励，则动动手指，随意收走。
<br/>　　半个时辰后。
<br/>　　萧逸来到了火牙宗峰顶。
<br/>　　而此时，到达峰顶处的武者，已有过万之数。
<br/>　　萧逸是随意行走，边前行边通过考核。
<br/>　　而其余武者，除了这里的奖励外，可是冲着排名奖励而去的。
<br/>　　自然的，这些武者都是全力行走，全力通过考验，故比萧逸更早来到峰顶。
<br/>　　另外，赤色火焰，是火牙盛事中危险程度最低的考验。
<br/>　　本身死亡率就不高，难度也并不多大。
<br/>　　故，来到峰顶这里的武者，多打过万之数，差不多有一半的参加者都能来到这里。
<br/>　　廖灵儿，赫然也在人群之中。
<br/>　　廖灵儿，自然也看到萧逸。
<br/>　　嗖…廖灵儿脸色一惊，连忙闪身而来。
<br/>　　“前辈，我不是跟你说了不要继续前行？”廖灵儿满脸气急败坏之色。
<br/>　　萧逸眼神冰冷而淡漠，不语。
<br/>　　廖灵儿见状，脸上的急色，戛然而止，取而代之的，是一抹苦笑。
<br/>　　“也是，前辈与我非亲非故，怎可能信我。”
<br/>　　“再者，在前辈眼中，我不过是个小人物罢了，怎可能听我的。”
<br/>　　说罢，廖灵儿再也不语。
<br/>　　萧逸宽大袍子之下，看不到脸色，但实则，嘴角咧过一道笑容。
<br/>　　虽然不知道廖灵儿什么意思，但如无意外，应该便是和火牙宗有关，不，准确来说，是和火牙宗的邪修有关。
<br/>　　萧逸是来查火牙宗的，怎可能退去。
<br/>　　当然，他也不会解释。
<br/>　　数分钟后。
<br/>　　峰顶处，大片火焰席卷而起。
<br/>　　火焰，渐渐化作一个巨大的熔浆巨人。
<br/>　　周遭武者，当即脸色大变。
<br/>　　“是九牙火兽，该死，火牙大阵怎会幻化出这等妖兽？”
<br/>　　“明明只是赤色火焰，难度应当是最低的才对。”
<br/>　　周遭武者，纷纷惊呼。
<br/>　　九牙火兽，皇境中最凶名赫赫的凶兽。
<br/>　　其等级，只属圣皇境巅峰妖兽，但其实力，就是武道大能遇到，都得慌忙逃命。
<br/>　　虽然现在这头九牙火兽，只是火焰幻化；但这就是火牙大阵的厉害之处，所幻化之物，与真实的妖兽、傀儡等等，有着一模一样的实力。
<br/>　　“不管了，和它拼了。”一个个武者，纷纷大喝出言。
<br/>　　“对，我记得九牙火兽，是只有火牙盛事难度最高时才会出现的妖兽。”
<br/>　　“但若是遇到了，即便不敌，火牙大阵内的禁制最后也会将我们轰出火海，不会有生命之虞。”
<br/>　　说罢，一众武者，同时出手。
<br/>　　九牙火兽，只有绝世强者有那个能力应付；若是单打独斗，武道大能也只有逃命的份儿。
<br/>　　自然的，一众武者联手而出。
<br/>　　嗖…嗖…嗖…
<br/>　　一道道身影，激跃而出。
<br/>　　大战，瞬间在这火海内爆发。
<br/>　　萧逸，并无动作，只是淡漠看着。
<br/>　　这头巨大的火兽，身躯巨大无比，有着九双狰狞獠牙，一看便知非善类。
<br/>　　但这等妖兽，即便是真正的九牙火兽来了，萧逸也不惧之。
<br/>　　轰…轰…轰…
<br/>　　九牙火兽与过万武者的战斗，激烈至极。
<br/>　　整个火海之内，处处是震耳的元力爆响。
<br/>　　轰…
<br/>　　忽然，九牙火兽巨大的火焰手臂，一拳轰中了一个黑袍武者。
<br/>　　这个黑袍武者，当即衣衫碎尽，露出了真实面容。
<br/>　　“嗯？”
<br/>　　几乎是这个黑袍武者露出面容的瞬间，周遭武者便已认了出来。
<br/>　　“那不是火牙城主府的火长老？”
<br/>　　“原来如此，是来暗中保护少城主的吗？”
<br/>　　轰…轰…轰…
<br/>　　仅仅是十数分钟后，便有近10个黑袍武者露出了真容。
<br/>　　“那不是三大家族的长老们？”
<br/>　　“还有遁火门的大长老…”
<br/>　　一声声惊呼出现。
<br/>　　另一边，萧逸观看着战斗。
<br/>　　那一个个黑袍武者露出真容，虽让人惊讶，但似乎也在情理中。
<br/>　　这次来此参加火牙盛事的，多的是火牙地域内的大人物。
<br/>　　火牙城少城主、三大家族的少家主、遁火门的少门主等等，可都是地域内一等一的天骄。
<br/>　　除了他们明面上带的强者外，自然也有别的强者暗中保护。
<br/>　　萧逸观看着战斗，蓦地点了点头。
<br/>　　战斗爆发至今，九牙火兽虽强，却也奈何不了过万武者。
<br/>　　毕竟这次来此参加盛事的，都非地域之内的泛泛之辈，有大量强者存在。
<br/>　　当然了，一众强者，也奈何不了九牙火兽。
<br/>　　九牙火兽，由火牙大阵凝聚而出，集全阵之力，岂是那么好对付的。
<br/>　　这一战，若无一方拥有碾压性的实力，怕是短时间内难以结束。
<br/>　　萧逸暗暗皱眉，他可没兴趣浪费这些时间。
<br/>　　尽快结束了这一盛事，还得看看火牙宗内集聚的大量邪修怎么回事。
<br/>　　不过有一点，萧逸可以确定。
<br/>　　这些邪修，应该不是为了对付来这里参加盛事的武者而集聚的。
<br/>　　这过万武者，一个个都来自于地域内的大势力，且皆是重要人物。
<br/>　　若这些人物全出事了，怕是整个火牙地域都要暴走。
<br/>　　火牙宗，没这个胆子。
<br/>　　起码，单单这里的邪修，不敢如此做。
<br/>　　否则，火牙宗和这里的邪修，将承受整个火牙地域的怒火。
<br/>　　火牙宗，显然没有这个本事，也不敢如此。
<br/>　　若他们真有这个胆子的话，除非他们拥有着碾压整个火牙地域的实力吧。
<br/>　　当然，若真如此，火牙地域早便被攻陷了，哪里还需要故意在此设下盛事，多此一举。
<br/>　　故，邪修集聚，应该和这场盛事无关。
<br/>　　但，也正是如此，萧逸才更加疑惑。
<br/>　　这群邪修，到底想干什么？
<br/>　　萧逸摇了摇头，没有再多想，事情如何，应该要不了多久就能明了。
<br/>　　现在，还是先击杀了这头九牙火兽，结束这场盛事。
<br/>　　萧逸刚要出手，然而，廖灵儿却闪身而来。
<br/>　　“前辈。”廖灵儿沉声道。
<br/>　　“你现在走，还来得及。”
<br/>　　“此时整个火牙大阵，都在为了凝聚操控九牙火兽，大阵本身的力量，已无比虚弱。”
<br/>　　“一众武者，也是全力应付着九牙火兽。”
<br/>　　“你若现在离去，足可神不知鬼不觉，无人知晓你临阵脱逃。”
<br/>　　萧逸不语。
<br/>　　廖灵儿脸色一急，脸色，头一次认真到极点。
<br/>　　“再不走，真的要丢命了。”
<br/>　　萧逸，仍旧不语。
<br/>　　“好。”廖灵儿药咬了咬牙，迟疑地看了眼萧逸，下一秒，脸色一冷。
<br/>　　一柄匕首，凭空而现。
<br/>　　正是之前‘捡到’的那把下品圣器。
<br/>　　此刻，下品圣器上，光芒涌动。
<br/>　　“阵法？”萧逸皱了皱眉。
<br/>　　不知什么时候，廖灵儿在匕首上烙印了阵法。
<br/>　　“是空间类阵法？”萧逸宽大黑袍之下，蓦地脸色一变。
<br/>　　这是他来到火牙宗后，头一次变色。
<br/>　　“你早便完美掌控了空素风狐的空间能力。”萧逸双眼一眯。
<br/>　　“前辈，现在，你后悔也来不及了。”廖灵儿叹了口气，但语气冰冷。
<br/>　　周遭空间，开始扭曲。
<br/>　　火焰，渐渐消散。
<br/>　　而那头过万武者围攻下仍旧奈何不得的九牙火兽，竟在空间扭曲之下，被强行绞成了齑粉。
<br/>　　“怎么回事？”周遭武者，脸色大变。
<br/>　　“空间扭曲？这是传奇人物才可能掌控的手段，怎么可能…”
<br/>　　周遭武者，虽惊呼出言，但声音，已再也发不出。
<br/>　　空间，蓦地凭空而变。
<br/>　　待得一切恢复正常时，周遭，已是邪修环伺。
