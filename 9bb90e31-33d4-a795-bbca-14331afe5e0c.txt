　　第561章
<br/>　　天将亮未亮的时候，程大雷和华正则找到一个汤池。交待伙计去外面买两套干净衣服，再捎回来两碗热汤面饱肚。
<br/>　　直等到日上三竿，程大雷在水池里伸个懒腰，才感觉体力慢慢恢复，疲惫感被热水一点点洗去。
<br/>　　二人填饱肚子，换上干净衣服，离开了汤池。
<br/>　　这时候，整座琅琊城也苏醒了过来。街道两旁的商铺打开门做生意，小商小贩推车挑担在街道上来来往往，佩剑的书生背刀的武士走在街头，青楼的姑娘们的打开门，红袖招摇。发出阵阵清脆的笑声。
<br/>　　程大雷和华正则并肩走在一起，托伙计买来的衣物，自然都是书生长袍，恰好一黑一白，只不过二人的相貌……穿在身上显得有些不伦不类。
<br/>　　华正则很明显没意识到这点，手中摇着折扇，对路上穿红挂绿的姑娘流口水。治病救人时他像个高手，放下医囊，他是随时可以送进大牢的主儿。
<br/>　　二人是向书院走去，一边走程大雷心里一边琢磨着。等回到书院，若没有其他事情耽搁，自己就可带着华正则和刘发财离开。至于那《山水经》，程大雷是很希望搞到手的，但如果真的找不到，他也没有什么法子。
<br/>　　这样走着，程大雷突然一拽华正则，拐进一个死胡同中。
<br/>　　华正则正盯着一个姑娘的腰身浮想联翩，不知道程大雷为何突然要这么做。
<br/>　　“欧公子，你……”
<br/>　　程大雷皱起眉头，透过胡同口盯着人来人往的街道，眼睛眯成一道线。
<br/>　　今天的琅琊城……有些不对。
<br/>　　在街上的书生、游侠、商贩和姑娘中间，多了些其他什么人。他们行色匆匆，表情严肃，在街上找来找去，似乎寻找着什么人。
<br/>　　当然，程大雷目前还不能确定对方的目标就是自己，但毕竟做贼心虚，程大雷的身份如果暴露出来，怕整个东海的高手都会陷入亢奋。
<br/>　　“欧公子，是不是发生什么事了？”华正则看程大雷表情严肃，心眼转得却是很快。
<br/>　　“嗯呐。”程大雷点点头：“我们就此分开，别忘了你答应过我什么。”
<br/>　　声音落地，程大雷一点脚尖，跃上房顶走了。他必须尽快搞明白发生了什么事，确认自己是安全的。
<br/>　　华正则仍旧是一头雾水，他摸摸鼻子，感慨道：“这高来高去的本事，还真是不错呐。”
<br/>　　之后，他走出胡同，独自一人返回书院。
<br/>　　华正则也并非傻瓜，他一回到书院，就感觉今天的书院有些不太对。山上山下，弥漫着一种严肃的气氛。那朗朗的读书声今日也听不见了，给人的感觉，昨天这里还是一座教书育人的书院，今日已变成一座军事堡垒。
<br/>　　宋寒之接到了华正则，惊喜道：“华神医是怎么回来的，可是让我等好生担心？”
<br/>　　华正则倒也没什么好隐瞒的，将燕不归和程大雷的事情都说了。最后感慨道：“多亏了欧公子出手相救，不然你们就见不到我了。”
<br/>　　“华神医和那姓欧的刚刚分开？”宋寒之认真问。
<br/>　　“是啊，怎么了？”华正则觉得宋寒之表情有些不对，道：“那欧公子可是个厚道人呐。”
<br/>　　“他……厚道？”宋寒之愣了愣，道：“华神医可知道那人是谁？”
<br/>　　“谁？”
<br/>　　从宋寒之口中吐出三个字，听到后华正则表情就是一呆。道：“这不太可能吧？”
<br/>　　“怎么不可能，那贼人隐姓埋名来到书院，把我们都骗了过去。幸亏夫子老人家灵台明鉴，识破贼人身份，否则还不知他会造出怎样的恶孽。现在，我们已经抓住了他的同伙，定要审出他来东海做什么。”
<br/>　　宋寒之咬牙切齿，华正则却不太能理解他的情绪。当然，对这种江湖斗争，他也并不关心。
<br/>　　“华神医可知道他去了哪里？”宋寒之问。
<br/>　　华正则摇摇头：“他既然已经走了，想要再找到他，怕就不容易了。”
<br/>　　“未必。”宋寒之冷冷道：“他的同伙还在我们手里，我就不信他不回来搭救。”
<br/>　　……
<br/>　　张明理领着楚云生，穿过长长的走廊，再次进入书院的地牢。楚云生捂住鼻子，一走进地牢，光线便暗了下来，空气中似乎夹杂着什么别的味道。
<br/>　　“楚公子，你来看。”张明理停下脚步，手往前伸。
<br/>　　楚云生顺着他手指的方向望过去，目光一下子呆住。
<br/>　　只见刘发财被吊在半空中，身上伤痕累累，血水顺着头发衣服一滴滴落在下面。
<br/>　　楚云生不再怀疑，刚才自己闻到的血腥味就是从他身上传来的。只不过一夜不见，他竟变成这般模样。如果不是他仍睁大双眼，楚云生都要想他是不是死了。
<br/>　　“乖儿子，还有什么招数，都冲老子使出来。老子若皱皱眉头，便算没生过你这么个儿子。”
<br/>　　李亚子坐在一张太师椅上，翘着二郎腿，身体斜在一边。
<br/>　　“不着急，不着急。”李亚子道：“时间还有很长，我们可以好好聊聊。”
<br/>　　“哇。”刘发财发出一声尖叫，大喊道：“我服了，我服了，你们想知道什么，我统统告诉你们。”
<br/>　　“说，程大雷来东海想要做什么，你们有什么阴谋诡计！”
<br/>　　“这个我真不知道啊。”刘发财道：“我和你们说过多少遍了，我和他真的不熟。”
<br/>　　“呵，嘴巴倒还挺硬，我倒要看看你们的江湖义气能撑到什么时候。”李亚子嘴角的笑容越来越冷：“继续动刑。”
<br/>　　“去你巴拉的江湖义气。”
<br/>　　刘发财歪着一张嘴，他当然没有要为程大雷保守秘密的觉悟，也没有这个必要。事实情况是，他不知道程大雷任何秘密。
<br/>　　所以，在刚一用刑，他就把能说的全都说了。然而，在李亚子眼里，他什么都没说。
<br/>　　作为一个骗子，有时候最可悲的是：你好不容易说次真话，偏偏没人相信。
<br/>　　张明理摆摆手：“楚公子，非礼勿视，我们出去谈话。”
<br/>　　大牢内的气氛的确有几分压抑了，楚云生咳嗽一阵，跟在张明理身后，一瘸一拐的向外走去。
<br/>(本章完)
