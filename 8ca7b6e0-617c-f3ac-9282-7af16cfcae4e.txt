　　“小子，本尊乃是北邙学院的内门弟子杨岳！死前你也要记得本尊的名号，免得做个不明不白的无头尸！”
<br/>　　那跟班杨岳，看着楚风眠，大笑连连道。
<br/>　　别看他在那年轻男子面前只是一个跟班，在这武胜国度，他的身份简直是一尊无敌人物。
<br/>　　北邙学院的任何一位弟子，放在武胜国度都要比起武胜国度皇帝的地位都要高。

<br/>　　他的境界虽然只是神力境六重的地步，但是出手瞬间爆发的力量，足以是跟神力境八重所抗衡。
<br/>　　足足是超越了自己两个境界的实力。
<br/>　　“这北邙学院的弟子，果然都是天才中的天才，一个小小的内门弟子，跟班一样的人物，都可以跨越两个境界来战斗。”
<br/>　　看着杨岳出手，楚风眠心中暗自想到。
<br/>　　这样的资质，放在秦皇国度都可以算是第一的天才了，不过若是在北邙学院中，也最多只能算是中游，不然也不至于沦落成一个小跟班。
<br/>　　“好强的力量。”
<br/>　　“这便是北邙学院的弟子吗？”
<br/>　　雷澈，还有其他三大宗门的圣者，看到这杨岳出手，都是脸色微变。
<br/>　　在神力境，能够跨越两个境界对敌，已经称得上是天才中的天才了，这北邙学院随意一尊弟子，居然都可以有着如此可怕的资质。
<br/>　　杨岳出手。
<br/>　　群山之力，便是连绵不断一般，向着楚风眠轰击而来，如同无穷无尽连绵不绝的山脉，无穷无尽，令人毫无喘息的时间。
<br/>　　这一招武技，霸道无比，就算是让一尊神力境九重巅峰的圣者来，都要被死死压制住，无法还手。
<br/>　　他这一招出手，显然便是要将楚风眠彻底灭杀，不给楚风眠任何机会。
<br/>　　不过就算是在天才，在楚风眠的面前，却都是如同蝼蚁。
<br/>　　楚风眠冷哼一声，大手一挥，便是随意的一掌对着那杨岳打了过去。
<br/>　　这一掌之下，看起来虽然只是轻松许多，但是两股力量相撞，这杨岳的灵力，居然是被一下子打的涣散，一招被逼退了。
<br/>　　“什么？这样一个破落国度的，居然还有着这样的人物？”
<br/>　　一招被逼退，那杨岳的脸上都是不由的露出几分震惊，楚风眠的境界可是只有御风境巅峰，居然是有着足以跟他抗衡的力量。
<br/>　　这简直不可思议到了极点。
<br/>　　“不过就算是你的实力不错，也无法跟我们抗衡！力拔千钧！”
<br/>　　杨岳一招败退，又是一招猛然打出，他的脸上都是露出几分怒意，动用了全力。
<br/>　　他可是北邙学院的弟子，如今对付一个御风境武者，居然都被一招逼退，这简直耻辱到了极点。
<br/>　　这一招的力量，比起刚刚还要强大，这力量范畴已经是接近到了神力境九重的地步。
<br/>　　这一招出，宛如一座巨大无比的山峰，要彻底的压迫楚风眠，将楚风眠碾碎。
<br/>　　“滚！”
<br/>　　楚风眠瞥了一眼那杨岳，猛然大手一挥，全力之下，一股强大到极点的力量爆发了出来。
<br/>　　顿时将那杨岳打出的无尽灵力，彻底轰碎，这一掌，更是直接砸在了他的胸口之上，将他砸飞出去。
<br/>　　这杨岳身上的灵脉，都被这一掌之间，硬生生的砸碎。
<br/>　　这杨岳，没有几年的功夫，是绝对无法恢复了，而且就算是恢复，境界也无法再进一步，这一掌，几乎废了杨岳的未来。
<br/>　　“放肆！”
<br/>　　“该死的小子，居然敢打伤我们北邙学院的弟子！”
<br/>　　“今日你必须要给一个交代才行！作为我们的奴才！为我们终生效力才行！”
<br/>　　“对，镇压了他，他的资质不错，在他的身上刻下禁制，让他成为我们的奴隶！”
<br/>　　在年轻男子身边，另外的两名随从，看到杨岳被打飞，统统怒喝一声道。
<br/>　　两道身影，同时出现，这两人的实力，丝毫不比那杨岳要差，瞬息之间，便是向着楚风眠镇压过来。
<br/>　　这一个冲突，便是就要将楚风眠给镇压，要将楚风眠当做奴隶。
<br/>　　这些北邙学院的弟子，一个个心高气傲到极点，更是冷酷无情，在他们看来，除非是他们北邙学院的人，不然任何人都是蝼蚁，奴隶，想要当做奴隶，就可以当做奴隶。
<br/>　　楚风眠眼神冰冷道了极点，他一直克制，没有全力出手，正是不想要跟这些北邙学院的弟子起什么太大的冲突。

<br/>　　楚风眠不主动惹事，可也不会惧怕了什么，七大宗门他都不怕，这万年间新崛起的北邙学院，楚风眠更不会怕。
<br/>　　“放肆？你们这些北邙学院的人，上来就想要夺取北邙圣牌，还想要打我女人的主意？看来你们都是无法无天了，那好，那你们今日都得死！”

<br/>　　“哼，蝼蚁一般的人物，也敢反抗我们北邙学院？你敢反抗我们！不服从我们的命令，便是有着大罪恶，我们将你镇压，让你当做奴隶，也是你应得的！”

<br/>　　“你现在反抗，更是罪加一等，不止是你，你的一切朋友，部下，都要当做我们的奴隶才行！”
<br/>　　那两尊北邙学院的弟子，同时出手，这二人之间，居然是习得了一种合击之术。

<br/>　　一齐出手，便是化为了一座巨大无比的禁制。
<br/>　　要将楚风眠给彻底的镇压在其中。
<br/>　　“滚开！”
<br/>　　楚风眠头也不抬，猛然一拳砸了过去，这一道巨大无比的禁制，便是在楚风眠的一拳之下，被打的崩塌离析，瞬间破碎。
<br/>　　其中的余波，都将那两名北邙学院的弟子，一下子给镇飞了出去，口中吐出了无数鲜血来。
