　　远处，数道人影快速袭来。
<br/>　　正是慕容山以及慕容家的几位长老，还有两个陌生中年男子，全是先天武者。
<br/>　　慕容墨大手一挥，轻松将灵气牢笼打破，不屑道，“萧离火，半步终究是半步，对灵气的掌控力，你远远不如我。”
<br/>　　说罢，慕容墨凌空一掌打出。
<br/>　　包括大长老在内的所有萧家族人瞬间重伤。
<br/>　　这时，慕容墨看了眼另一边快速逃跑的萧逸，咬了咬牙，自语道，“一旦让这小子跑了，他日必定后患无穷。”
<br/>　　“慕容山，萧家的人已经全部重伤，交给你们了。”
<br/>　　说罢，慕容墨没有任何停留，立刻凌空飞起，迅速朝萧逸追击而去。
<br/>　　.....
<br/>　　前方远处，萧逸的速度飙升到极致。
<br/>　　嘭、嘭、嘭....
<br/>　　一阵阵剧烈的破空声从后方传来，比萧逸快了数十倍不止。
<br/>　　萧逸蓦然一笑，他早就知道自己放下那种狠话，慕容墨会不惜一切马上追来的。
<br/>　　他的意图本身就是吸引慕容墨追他。
<br/>　　若留着慕容墨对付大长老等人，大长老他们必死无疑。
<br/>　　但只要引开慕容墨，就算慕容家有追兵，大长老他们也能有一线生机。
<br/>　　“灵气牢笼。”
<br/>　　萧逸已经跑到数里之外，但还是被御空飞行的慕容墨快速追上了，一记灵气牢笼困在了原地。
<br/>　　“该死。”萧逸试图挣扎，却徒劳无功。
<br/>　　而且，慕容墨为防止有意外发生，竟将灵气牢笼的范围扩散到方圆数十米。
<br/>　　这代表着，数十米范围内，萧逸都寸步难行，甚至只要慕容墨一个念头，他就必死无疑。
<br/>　　当时逃走的那紧急关头，萧逸没有太多时间，只来得及考虑引开慕容墨，让大长老他们能有一线生机。
<br/>　　却没有时间考虑自己该如何脱身。
<br/>　　嗖，慕容墨从半空中降落，冷冷地盯着萧逸。
<br/>　　“小子，挺能跑的，继续跑阿。”慕容墨得意一笑，“区区后天境，想从我手上逃走？真是笑话。”
<br/>　　萧逸咬咬牙，暗中施展了升龙，自身实力提升到极致，试图破开灵气牢笼。
<br/>　　可惜，他全力挣扎，却徒劳无功，撼动不了灵气牢笼半分。
<br/>　　“小小年纪，竟有如此天赋和修为，老夫活了大半辈子，从未见过如此天才。”慕容墨赞叹了一声，但脸色随即变为冰冷。
<br/>　　“你这种天才，哪个强者见了都会眼热，想收为徒弟，培养一番，老夫也不例外。可惜，你是我慕容家的敌人，你必须要死。”
<br/>　　慕容墨说道。
<br/>　　萧逸冷笑一声，“你早知我与你们慕容家水火不容，今日必杀我，何必还说那些冠冕堂皇之话。”
<br/>　　萧逸试图拖延时间，看能否想到脱身的办法。
<br/>　　慕容墨一眼看穿了萧逸的想法，嗤笑道，“小子，收起你那可怜的念头吧，你逃不掉的。别以为我不知道你的目的，不就是想引开我，好让你萧家的长老有逃命的机会吗？”
<br/>　　“既然你这么想死，我便成全你。”
<br/>　　慕容墨对自己自信十足，并不急着杀萧逸，反而想好好折磨这个他生平所未见过的绝世天才。
<br/>　　以前年轻时，他便被萧离火压了一辈子。
<br/>　　没想到如今突破到洞玄境了，回到紫云城，萧家又出现了一个绝世天才，且天赋更为恐怖。
<br/>　　他要让萧逸绝望，以满足他那畸形的妒忌心。
<br/>　　灵气牢笼内，萧逸握紧了拳头，倔强的目光盯着慕容墨，“慕容墨老鬼，别太得意了。你唯一比我强的，只是你那多活了数十年的日子。若我是你这般岁数，却只是个区区洞玄境，想必会羞愧得撞墙自杀。”
<br/>　　“你...”慕容墨顿时气急，他想通过言语打击萧逸的心理，让萧逸绝望，没想到反被激怒。
<br/>　　一个人，越是愤怒生气，越容易出错，甚至露出破绽。
<br/>　　萧逸虽然不认为慕容墨露出破绽，自己就能逃脱；但终归能通过言语拖延一下时间，再不济，也不要看到慕容墨那高高在上的得意面孔。
<br/>　　“好，很好，我看你的实力是否也像嘴巴那么硬。”慕容墨面容冰冷，“听慕容蝎说，你似乎能破我的黑墨曼陀罗武魂力量。”
<br/>　　“呵呵，我倒要看看，你是否真有那本事。”
<br/>　　“到底是你破了我的武魂力量，抑或你被黑墨曼陀罗腐蚀，皮肉、骨头、内脏化为一滩血水凄惨而死。”
<br/>　　慕容墨挥了挥手。
<br/>　　一阵黑气从他手上激射而出。
<br/>　　霎那间，数十米范围的透明的灵气牢笼内，立马被一层黑气所充斥。
<br/>　　远远看去，透明牢笼，仿佛成了一个黑色的剧毒之地。
<br/>　　“额。”萧逸立马感到一股窒息之意，随即皮肤上出现了剧烈的刺痛。
<br/>　　“不好。”萧逸反应极快，立刻调动体内真气，一层‘火焰外衣’覆盖在身上。
<br/>　　但，火焰外衣却几乎以肉眼可见的速度被黑气快速溃散。
<br/>　　“糟了，要抵挡不住了。”萧逸暗暗咬牙，脑海中快速思索应对之法。
<br/>　　当初慕容墨仅仅是提前凝聚出武魂力量，且由五长老通过水源稀释，却已经能让四六长老功力全失。
<br/>　　如今慕容墨亲至，黑墨曼陀罗在他洞玄境的浑厚真气加持下，其威力强了百倍都不止。
<br/>　　太阴太阳之眼确实能吸收这些武魂力量，但吸收速度是和萧逸的实力挂钩的。
<br/>　　当初在矿脉为四六长老吸收黑墨曼陀罗武魂力量，尚且要数分钟时间。
<br/>　　现在的黑气由慕容墨亲自发出，一个时辰都不一定能吸收完。
<br/>　　一个时辰，萧逸早就化为一滩血水了。
<br/>　　说到底，还是自身的修为与慕容墨相差太多了。
<br/>　　“嘶哑”一声，火焰外衣仅仅支撑了几秒便被完全腐蚀。
<br/>　　黑气再度开始腐蚀他的身体。
<br/>　　衣服已经溃烂，皮肤也出现了融化的迹象。
<br/>　　而且，灵气牢笼内漆黑一片，深处黑暗中，又饱受剧烈疼痛的折磨，这种感觉会让人抓狂。
<br/>　　当然，萧逸心智过人，仍旧强撑着脑海清灵，不断思索。
<br/>　　他不是没有想过‘冰鸾剑’，那是他最大的保命底牌。
<br/>　　冰鸾剑的强大，胜在他足够锋利，任何东西都抵挡不住它的锋芒。
<br/>　　只可惜，现在自己被灵气束缚住，动弹不得。
<br/>　　当初在陨星山脉能打败银背妖蟒，是因为银背妖蟒不擅速度，虽是三级妖兽，先天实力，但速递只与二级妖兽差不多，这才被萧逸击杀。
<br/>　　可如今慕容墨乃是洞玄境武者，别说自己靠近不了他。
<br/>　　就算能靠近，以洞玄境武者的反应，一瞬间就能将自己再次禁锢住，自己根本没有机会。
<br/>　　黑墨曼陀罗的毒气快速地腐蚀着他的皮肤，他已经全身发黑，这是身体即将被融化的征兆。
<br/>　　后天武者的肉体还是相当强悍的，就算肉体受了极严重的伤势，只要不死，便还有一线生机。
<br/>　　可是，黑气果真霸道。竟已经通过皮肤深入他的体内。
<br/>　　毒气开始在内部腐蚀他的经脉、内脏。
<br/>　　不出数十秒，他便会由内而外，彻底化为一滩血水。
<br/>　　“要死了吗？”萧逸心头着急想着，意识已经开始模糊。
<br/>　　半空中，慕容墨得意地看着自己的‘杰作’，灵气牢笼内虽然已漆黑一片，但实则一切都在他掌控当中。
<br/>　　他可以清晰看到萧逸的一举一动。
<br/>　　他已经看到萧逸倒下，再无反抗之力，生机也在快速流逝，离死不远。
<br/>　　“萧家最出色的天才，马上就要死了，哈哈哈哈。”慕容墨得意地大笑着。
<br/>　　只是，他没有看到，黑气进入萧逸体内，在腐蚀萧逸经脉内脏的同时，竟也开始腐蚀萧逸的武魂。
<br/>　　控火兽武魂‘呆头呆脑’地被腐蚀着，完全无法反抗。
<br/>　　但，黑气来到冰鸾剑附近时，冰鸾剑却忽然光芒大作。
<br/>　　一股骇然而散发着古老气息的力量忽然觉醒。
<br/>　　同一时间，萧逸体内的黑气，竟被一股‘绝对力量’瞬间冰封，再也无法腐蚀萧逸的内脏和经脉。
<br/>　　“嗯？”那一瞬间的冰冷，让萧逸打了个激灵，模糊的意识也瞬间清醒。
<br/>　　半空中，慕容墨正在放生大笑，忽然，笑声戛然而止。
<br/>　　“怎么回事？周围的温度怎么在急速骤降？”慕容墨皱了眉头。
<br/>　　下一秒，一股冰凉的感觉出现在慕容墨额头。
<br/>　　慕容墨摸了摸，额头上竟有一朵雪花。
<br/>　　“下雪了？”慕容墨眉头皱得更紧。
<br/>　　几乎在他还未反应过来的时间内，漫天大雪倾盆而下。
<br/>　　方圆数里地，在短短几秒内化为了一片冰天雪地。
<br/>　　“异象，这是异象，到底发生了什么？”慕容墨脸色大变。
<br/>　　他瞥了眼灵气牢笼，原本漆黑一片，此刻竟忽然变为白茫茫一片。
<br/>　　他的黑墨曼陀罗黑气，已经全部化为冰屑。
<br/>　　那里，萧逸傲然地站了起来，手中握着一把剑，一把全身紫芒，气息骇人的神剑。
<br/>　　“这小子，怎么还能站起来？”慕容墨大惊失色。
<br/>　　“灵气牢笼，给我禁锢。”慕容墨再也不敢怠慢，洞玄境的一身功力全部调动起来，而后一掌打出。
<br/>　　这一掌，就算十个先天九重都会被震死，他相信萧逸必死无疑。
<br/>　　然而，正当他这一掌凌空打出时...
<br/>　　萧逸的剑上，忽然发出一声震啸九霄的啼鸣。
