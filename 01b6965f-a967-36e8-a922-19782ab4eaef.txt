　　宫殿中，苏莫和吕罡相对而坐。
<br/>　　“吕罡，问道山方圆百万里之内，有没有大的坊市或者拍卖会？”苏莫沉声问道，若是问道山附近有的话，他还是不去古桐城为好。
<br/>　　毕竟，在问道山方圆百万里之内，他能有绝对的安全，若是去古桐城，说不定会遇到天虚宗之人的截杀。
<br/>　　“方圆百万里之内？”
<br/>　　吕罡闻言，立刻摇了摇头，道：“方圆百万里之内，只有几座小城和小镇，去了也是浪费时间！”
<br/>　　“是吗？”苏莫闻言皱眉，看来他要想将身上的宝物全部脱手，还要远离问道山了。
<br/>　　“苏莫，你是不是担心出去以后，会被天虚宗之人暗算？”吕罡不傻，他心念一转，便猜到了苏莫的想法。
<br/>　　“不错！”苏莫点了点头，大方的承认了。
<br/>　　“这的确是要注意！”
<br/>　　吕罡了然的点了点头，道：“不过，只要小心一些，别被天虚宗之人发现，应该不会有大碍！”
<br/>　　苏莫闻言沉默不语，陷入了沉思之中，思量了片刻之后，他还是决定要出去一趟。
<br/>　　现在他身上灵晶太少，说是一穷二白都不为过，只有出去赚些灵晶，才能增强实力。
<br/>　　当然，因为天虚宗之事，他必须要小心，说不定道宗之外，就有天虚宗的高手，在等待他离开问道山。
<br/>　　“吕罡，我们一起去古桐城，不过我需要你帮个小忙！”苏莫目视吕罡说道。
<br/>　　“什么忙？”
<br/>　　“我藏身在空间宝器之中，你带我离开宗门！”苏莫说道，这算是一个比较好的方法，若是问道山外有天虚宗之人，他以此离开宗门，也不会引起注意。
<br/>　　毕竟，问道山上每天都有数以千计的弟子进进出出。
<br/>　　当然，苏莫也不担心吕罡会害他，毕竟两人也没有大仇，之前还有合作。
<br/>　　而且，以对方的实力，想要害他也不太可能。
<br/>　　“没问题！”
<br/>　　吕罡闻言点头，对于苏莫的要求也能理解，这样的确比较安全。
<br/>　　“既然如此，我们事不宜迟，现在就走吧！”
<br/>　　苏莫沉声道，言罢，他将空间玉镯取了出来，交到了对方的手上。
<br/>　　少倾之后，吕罡大步的走出了宫殿，随即离开了问道山，一路向南飞去。
<br/>　　吕罡一路不停，身形如同划破长空的流星，速度奇快无比。
<br/>　　飞行了三个时辰之后，吕罡便远离了问道山无数万里，这个时候，他便停了下来。
<br/>　　“苏莫，你可以出来了！”吕罡取出空间玉镯，轻声说道。
<br/>　　他话音刚落，空间玉镯上光芒一闪，苏莫凭空出现。
<br/>　　“多谢！”苏莫向吕罡道谢，旋即便将空间玉镯收了回来。
<br/>　　“小事而已！”
<br/>　　吕罡不在意的笑了笑，道：“我们走吧，去古桐城！”
<br/>　　“吕兄稍等！”苏莫轻言一声，随即面部肌肉蠕动，相貌缓缓的改变。
<br/>　　少倾之后，苏莫变成了一名国字脸的青年。
<br/>　　吕罡见此恍然，苏莫倒是够谨慎，不仅出门藏在空间玉镯之中，连面目都改变了。
<br/>　　不过，改变面貌用处并不大，只能瞒过那些很低阶的武者，根本无法瞒过武尊境以上武者的查探。
<br/>　　略一沉吟，吕罡道：“苏莫，我有一件异宝，可以送予你，对你或许有帮助！”
<br/>　　“什么异宝？”苏莫疑惑问道。
<br/>　　旋即，便见吕罡手掌一番，掌中出现了一张面具。
<br/>　　这是一张银色的面具，古朴无华，现在颇为平凡，并没有任何特殊之处。
<br/>　　“苏莫，这张面具名为银缤面具，可以隐藏气息，只要修为不是超过你四重，以及四重以上，都无法看出你的真实身份！“吕罡说道。
<br/>　　“哦，如此不凡？”苏莫闻言，眼眸顿时一亮，这种宝物倒是对他很有用处。
<br/>　　“是啊！这面具非常的玄妙，是五年前我在一处古遗迹中所获，不过，这种宝物对于我来说，没有半点用处！”吕罡点头说道，言罢，他便将面具递向苏莫。
<br/>　　“这……吕罡，我不能平白无故收你的宝物，这样吧，这面具值多少灵晶，我支付与你！”苏莫没有接过面具，沉声说道。
<br/>　　他不想欠对方人情，这面具如此不凡，他岂能直接收取！
<br/>　　“苏莫，你说这话就见外了，上次在魔渊，若不是你，我也无法赚到这么多灵晶！”
<br/>　　吕罡摇了摇头，随即直接将面具塞到了苏莫的手中，道：“走吧！”
<br/>　　言罢，吕罡不由分说，身形一展，直接向前方飞去。
<br/>　　苏莫见此，面现无奈之色，既然对方不要灵晶，他也不好再多言。
<br/>　　随即，他便快速的追了上去。
<br/>　　飞行之中，苏莫带上了银缤面具，顿时便感觉到了此面具的奇妙之处。
<br/>　　这面具，并不是遮盖了他的气息，而是将他的所有气息，全部都收敛在了面具之中。
<br/>　　若是他不内视丹田，连自己都看不出自己的修为，当真奇妙无比。
<br/>　　“不错！”苏莫面上露出了一丝笑容，有了这张面具，必要的时候或许能有大用。
<br/>　　随即，苏莫便将面具收了起来，继续保持着改变后的容貌。
<br/>　　虽说这面具能隐藏气息，但平常若是都戴着一张面具，也太过扎眼了。
<br/>　　“苏兄，你是何时突破的武尊之境？”
<br/>　　飞行之中，吕罡和苏莫闲聊，他对于苏莫的修炼速度，心中颇为好奇。
<br/>　　“半个多月之前！”苏莫说道。
<br/>　　“哦，那看来你并不是在宗内突破，半月前并未听说有谁渡劫！”吕罡笑着说道。
<br/>　　“渡劫？”闻听此言，苏莫顿时神情一凝，因为他自从离开苍穹世界之后，压根就没有任何天劫。
<br/>　　当初，阎魔鬼圣告诉他，他的命数不在天道规则之中，所以不会有天劫。
<br/>　　对于此事，苏莫完全搞不清楚，但是对于他而言，没有天劫反而更好。
<br/>　　只不过，别人好像都有天劫，而他没有，不知道以后会不会有什么变数？或者留下某种隐患？
<br/>　　“是啊！我上次在星空渡劫，宗内的几名长老为我守护，你若是在外渡劫的话，危险性会大增！”吕罡说道，他的眼眸中依旧有后怕之时。
<br/>　　天劫，是任何生灵都惧怕的存在，那是天道的惩罚，一个不慎就是生死魂灭，永世不得超生。
<br/>　　在吕罡看来，苏莫在外渡劫，没有宗内长老的指点，是非常危险的事情。
<br/>　　而且，越是逆天的天才，天劫便越是恐怖，危险性更大。
<br/>　　“还好吧！一切都顺利！”苏莫点了点头，并未告诉对方，他并没有天劫。
<br/>　　两人一边闲聊一边赶路，速度却是丝毫不慢，近两日时间之后，前方出现了一座古老的雄城。
