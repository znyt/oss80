　　第304章
<br/>　　三军未动，情报先行。程大雷将这一方针贯彻到底，战争究竟打的是什么，打的是经济，打的是情报，至于战场上真正的动刀动枪，反而是次要的。
<br/>　　派出一支人马去浩甲城经营酒楼，听着城里的风吹草动，程大雷给他们的命令，是每三日汇报一次。在『江湖传言』技能的加成下，情报问题算是解决了。
<br/>　　程大雷亲自负责经济的问题，现在蛤蟆寨至少也有小一千人口，每天人吃马嚼不是一个小数目。程大雷再如何，也得先把这些人的肚子填饱，不然他们凭什么陪着自己玩命。
<br/>　　这一日，程大雷携着徐神机、张肥，乘一叶扁舟，顺水而行离了山寨。他今日换了一席秀士儒袍，手中一柄折扇，腰间佩着剑，不像个杀人放火的山贼大王，倒像一个风度翩翩的读书人。
<br/>　　船在浩甲城的码头靠岸，三人弃船登岸，程大雷摇着折扇，踱入浩甲城。
<br/>　　前面街上出现一生药铺，药铺东家陪着一妇人坐在柜台后，秋高气爽，妇人看上去已怀了身孕，一动一行身体都露出些不方便。东家陪着小心，不敢露出丝毫怠慢。
<br/>　　林掌柜招呼伙计干活，偷眼看着这一幕，心中感慨万分：自家掌柜自从上次事情后，就好像换了个人似的，整天大门不出二门不迈，这浩甲城内，都快没了西门大官人的艳名。
<br/>　　“林掌柜，这笔药材能不能加急几天结账，我们急着拿这笔款回去度冬。”
<br/>　　“老李啊，你是第一天做生意么，加急也要等药卖出去，你这笔药材都受了潮，我给你往外卖，还得陪着笑脸……”
<br/>　　正这时候，一个书生打扮的携着两人走进药铺，正在侃侃而谈的林掌柜无意间瞥了一眼，脸上表情突然僵住，滔滔不绝的嘴巴闭上。
<br/>　　“林掌柜，家里实在是转不开了，您行行好，还指着这笔钱还帐呢。”
<br/>　　“好好好，你明天过来，我把银子给你备着。”林掌柜忙不迭口的答应。
<br/>　　读书人进来后也不说话，自顾自的找个位置坐下，随之，林掌柜吩咐了伙计几句，药铺里买药的客人都被不动声色的请了出去。
<br/>　　很快，药铺内就只剩下几个人，大门关上，光线变得昏暗起来。
<br/>　　林掌柜和西门大官人都苦着一张脸，垂手立在读书人面前。
<br/>　　“程当家，您怎么又来了，这次您要点什么？”西门春长
<br/>　　“什么话，我特意来找你买东西的。”程大雷品着林掌柜送上来的香茶，说实话，这茶真不错，比蛤蟆寨的茶不知好到哪里去。
<br/>　　“哎呦呦，您可别说这话，您要什么，直说就可以了，我肯定差人送过去。”西门春长心想，你可别像上次一样，船到河心，问我吃板刀面还是馄饨。
<br/>　　“真是买药的。”程大雷递过来一份单子：“上面的药材给我备齐，自己送到我家里，按市价结账。”
<br/>　　唉，还要什么钱啊，惹上这样的煞星，不出人命就烧高香了。
<br/>　　程大雷递过去的单子忽然收了回来：“敢耍花招，你妻儿老小的性命，可得好好掂量掂量。”
<br/>　　“哎呦，我哪儿敢呐，您就是我的活祖宗。”
<br/>　　程大雷这才把药单子给他，程大雷的确是来买药的，山寨那么多人，多多少少也有些头疼脑热，备些药材算是战略物资。
<br/>　　“其实……我还有件事……”
<br/>　　我说什么来着，我说什么来着，山贼上门，哪会有好事。
<br/>　　“我最近缺钱，像学着做生意。”
<br/>　　“哎呦呦，您老爷还做什么生意，什么生意比您老爷做的生意赚钱。”
<br/>　　“抢……也不能天天抢。”程大雷道：“你市面上熟，现在什么东西挣钱？”
<br/>　　和程大雷聊了这么久，西门春长的惧意稍减，说话也有条理起来。
<br/>　　“如果说一本万利，自然是茶叶，药材，盐巴，戎族不产这些，送过去都是暴利。只不过，这生意做不了。”
<br/>　　“为何做不了？”
<br/>　　“朝廷不许和戎族做生意，抓到就是砍头的大罪。当然，也不缺铤而走险的人，关键是这条路现在走不通，沿路十八家首领，层层剥削，这条路如今已经没人敢走。”
<br/>　　程大雷寻思着这件事，别人不敢走，未必我走不通呐，没有商业，靠什么繁荣。他多看了这西门春长一眼，这小子是个人才呀，生意上的事情说得头头是道，要胆色也有胆色，虽然有些好色……但对于一个男人而言，好色能算毛病么。
<br/>　　程大雷微微倾身，认真的问：“你……想做山贼么？”
<br/>　　西门春长噗通一声跪倒，磕头如捣蒜。
<br/>　　“程当家，您饶了我吧，我上有老下有小，还有没出娘胎的孩儿，您高抬贵手，放我一马吧。”
<br/>　　程大雷离了药铺，继续往前走，拐入一家酒楼的后院。
<br/>　　“大当家，大当家……”厨子掌柜都围了上来。
<br/>　　这里的掌柜姓王，名叫钦州，因为会打一手算盘，心也细，所以被程大雷派过来支应酒楼的事情。
<br/>　　“好了，都做自己的事去。”程大雷挥挥手：“人都来了么？”
<br/>　　“也都是刚到不久，在里面候着呢。”王钦州道。
<br/>　　程大雷被引着走入后院堂屋，王钦州打开门后立在外面，只有程大雷、徐神机、张肥三人走了进去。
<br/>　　屋内也有三人，同时站了起来，脸上神色变幻，似乎有些不知该如何面对程大雷。
<br/>　　“坐，坐，都拘着做什么，又不是不认识。”
<br/>　　这三人分别是云中龙，乔鹤，游九楼，云中龙和程大雷照过面，甚至称得上熟悉，而其他二人算是和程大雷第一次照面，相同的是，他们都被程大雷打败过，原先的十八家头领之一。
<br/>　　“莫鸣米把你们划给我了，知道么？”程大雷问。
<br/>　　三人同时点点头。
<br/>　　“好处是，你们以后不需要向莫老大交保护费，坏处是，他也不会出头保护你们。”程大雷道：“不过事情，都得分两面看，你们可以选择以后跟我混，今天叫你们来，就是聊聊这个事儿。”
<br/>
