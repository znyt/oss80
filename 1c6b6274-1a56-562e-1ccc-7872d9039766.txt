　　第189章
<br/>　　山崖倒悬一道瀑布，从高空坠下来，跌落在河水中，白色的河流将山谷分为两段。
<br/>　　河水冲刷出来的滩涂是最肥沃的土地，往年这个时候，已该播种，占着这样天造地设的家园，只靠自耕自种，蛤蟆寨也能活得不错。
<br/>　　奈何，戎族打了过来，莫说种地，很多人整夜连个囫囵觉都睡不好，大好的田地只能荒废。
<br/>　　程大雷带着徐神机在山谷内转悠，他已经准备走了。
<br/>　　只要程大雷没疯，就不会觉得自己现在还能守得住，城破只是时间的问题，必须早做准备。
<br/>　　从城门突围几乎想也不要想，程大雷的意思是，在谷内开掘出一条山道，五千人一起撤，也花不了太多时间。
<br/>　　程大雷有一份青牛山地图，因为是系统奖励的，所以精度比戎族大王子手上那张好很多，程大雷可以从地图上，找到山谷那里适宜开掘。
<br/>　　当然，困难还是很大的，不过程大雷有火药，可以用来爆破。虽然有难度，但并不是不能做到。
<br/>　　只要有了这条向外的山道，蛤蟆寨这盘死棋就盘活了，打得过就打，打不过就撤。
<br/>　　咚咚咚！
<br/>　　忽然城门响起急促的鼓声，有一个喽啰飞奔过来。
<br/>　　“报！大当家，戎族打过来了。”
<br/>　　“又打过来了。”徐神机一惊，他也明白如今情况的严峻性。
<br/>　　“算着也该打来了。”程大雷摇摇头：“让大家上城。”
<br/>　　“大当家，还守得住么？”徐神机心惊胆战。
<br/>　　“守守看。”程大雷道。
<br/>　　这边守守看，那边打打看。等程大雷来到城门上时，只见己方已经布置好防御，人人目光坚定，再没有第一次守城时的慌乱和惶恐。
<br/>　　不得不说，蛤蟆寨这些山贼越来越像一支真正的精兵了，连续的坚守给了他们自信，自信又让意念凝聚，如今他们不再是五千个体，而是一个整体。
<br/>　　趟着尸山血海走过来的男人，自然要比普通人更坚硬。
<br/>　　戎族大军进发，兵丁推着投石机前进，此时投石机的数目已达到可怕的一百架。
<br/>　　一进入射程，投石机便一起发动，落石如雨，纷纷砸在城墙上。
<br/>　　同时，蛤蟆寨开始反击，蛤蟆寨的城头上也出现小型抛石机，这种机械没什么技术难度，大概知道个样子，就能琢磨着制造出来。
<br/>　　当然，出现在城墙上的抛石机要比地方小很多，射出的石弹大小不过和足球一样。
<br/>　　但蛤蟆寨射出的不是石弹……
<br/>　　轰隆隆……
<br/>　　落地时发出剧烈的爆炸声，铁片溅飞，一旦爆炸，就是成片的死亡。
<br/>　　这是程大雷研究的出来的，外层是铁皮，中间填着火药，铁钉，石子，大概就相当于将两口铁锅扣在一起，往里面填充爆炸物。大概可以命名为……铁锅弹？
<br/>　　不算太有技术含量的东西，投掷之前还需先点燃引线，不过程大雷之前试验过，威力倒是还行。
<br/>　　所以，战争果然是科技的第一推动力。无论在这个时代，还是在程大雷的前生……坦克、轰炸机，自动步枪等等普通人都很熟悉的武器，其实都是一战二战时发明的，出现历史不足百年。
<br/>　　这种爆炸物一出现，就炸得戎族魂飞魄散，瞬间，程大雷就收获了大波恐惧值。
<br/>　　而程大雷这面也十分危险。纵然城墙可以自动恢复耐久度，也挡不住投石机的狂轰乱炸。
<br/>　　程大雷和戎族的指挥官都明白，戎族已可以攻破这座蛤蟆寨，究竟能不能打破，取决对方愿意为此支付多大代价……
<br/>　　蛤蟆寨城门前，是一个三角形空地，十万大军根本排不开，每次进攻至多几千人而已。而在程大雷的狂轰乱炸下，这片空地完全成了一架用来屠杀的绞肉机。
<br/>　　你可以打下这座城，但你准备为此付出多少人命……五千，一万，还是五万，或者更多……
<br/>　　很显然，敌人的指挥官没有如此大的魄力，在付出几千条人命的代价后，望着摇摇欲坠的城门，对方还是无奈的选择了退兵。
<br/>　　对方刚走，程大雷就立刻令人修复城墙，固防技能激活，很快城墙就完好如初，就像没有遭受过攻击一般。
<br/>　　而程大雷则立刻将今天收获的恐惧值兑换成火药，守城和开掘山道都要靠这些火药。
<br/>　　戎族大营，死气沉沉。
<br/>　　大兵集结时的意气风发已经不见了，所有人都低垂着头，想起今日攻城的惨状，不觉心有余悸，当然，这又为程大雷提供了一些恐惧值。
<br/>　　金问道心中是得意的：让你们狂，让你们嘲笑我，现在知道厉害了吧。
<br/>　　“再继续打下去，伤亡太大，诸位可有什么高见？”大王子问。
<br/>　　“强攻是不行了，实在不行，只能派一队人爬上山峰，从里面攻破。”
<br/>　　“给山谷里投几具尸体，让他们染上瘟疫，此城不攻自破。”
<br/>　　“山峰陡峭，大军运不上去，兵力太少进去了也是送死。”大王子否了这些法子。
<br/>　　“那就只剩下围城一个办法了，困他们半年，困到他们没粮，自然城就破了。”
<br/>　　“不行，我们没有这么长时间在这里耗，再说困半年，他们活不活得下去我不知道，我们肯定先没粮了。”
<br/>　　众人议论纷纷，却也没人能拿出一个既能获得胜利，又不用付出代价的办法。
<br/>　　突然，此时有一个人冒出一句话：“我们是不是一定要打蛤蟆寨？”
<br/>　　众人一愣，忽然安静下来，所有人目光看向其他方向，没人吭声。
<br/>　　攻打幽州，是不是一定要打下蛤蟆寨？
<br/>　　答案是……不需要。
<br/>　　青牛山属于战略要地，如果帝国在青牛山屯一支重兵，进可攻，退可断道而守，若戎族继续南下，就要小心青牛山的背后偷袭。
<br/>　　但蛤蟆寨一共五千人，拥有战斗力的才一二千人，就算这一二千人都是精兵，也发挥不了太大作用。
<br/>　　而且，无论是断道而守还是背后偷袭，这都要出城，而在野地，那时程大雷或许就会明白，为何戎族骑兵被称作天下无双。
<br/>　　所以，要攻打幽州，并不需要打下蛤蟆寨。
<br/>　　大家完全可以绕过去嘛。
