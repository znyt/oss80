　　这剑鼎的作用只有这两个。
<br/>　　如今楚风眠唤醒了剑鼎，却是在他的面前，突然摆着一个难题，便是这剑鼎如何处置？
<br/>　　镇压在剑道门之中？镇压剑道门气运？
<br/>　　曾经剑道之主炼制剑鼎的确是为了这一点，但是现在的剑道门，却是不需要剑鼎来镇压气运。
<br/>　　楚风眠凝炼圣地，使得这剑道门，已经是有着对抗外敌的能力了。
<br/>　　而且现在剑道门主要还是依靠楚风眠支撑下去，提升实力，才是楚风眠最为重要的事情。
<br/>　　可这剑鼎的两大能力，镇压对敌，楚风眠并不缺少对敌的手段。
<br/>　　他掌控无数剑术，龙皇血脉，不管是始祖九式，还是楚风眠的无数剑术，都令他不缺少对敌的能力。
<br/>　　他手中的祖龙至尊剑，巨阙神剑，对敌的能力也不亚于这剑鼎，这剑鼎让楚风眠拿来对敌，没有太大的用处。
<br/>　　楚风眠也可以像是那荒鎏金一样，将剑鼎融入到他的身躯之中去，来提升楚风眠的实力。
<br/>　　楚风眠感觉到，他将剑鼎融入到自身之后，实力至少足以攀升一倍。
<br/>　　不过这力量的增长，只是外力，终究不是楚风眠的力量，而且借助外力来提升实力，对未来楚风眠的提升，也有着阻碍。
<br/>　　荒鎏金是自知他自己没有踏入半步天人境的希望，才会投机取巧，将剑鼎融入到身躯之中，晋升为半步天人境。
<br/>　　可楚风眠可不愿意做这种竭泽而渔的事。
<br/>　　“剑鼎，如果将剑鼎，融入到神树之中如何？”
<br/>　　楚风眠思考之中，脑海之中突然浮现出了一个想法。
<br/>　　楚风眠身躯之中的神树跟楚风眠力量同源，这神树的力量越是强横，也代表着楚风眠的力量强横。
<br/>　　之前楚风眠就是让神树吞噬了那青铜天魔骸骨，才让神树成长到了如此的地步，更是借助青铜天魔骸骨的力量，凝炼出了血肉晶体，化为了半步不朽真身的肉身。
<br/>　　青铜天魔骸骨，实际上也称得上是一件灵器了，而且一旦是催动起来，只怕是要比起剑鼎的威力还要强横。
<br/>　　这神树就连青铜天魔骸骨都能够吞噬，那这剑鼎，也应该不在话下。
<br/>　　想到这里，楚风眠的背后便是浮现出了一根根树枝，将神树凝聚出来，随即将剑鼎放在了上面，顿时被神树枝叶，卷入到了其中。
<br/>　　楚风眠清晰的看到，在他的身躯之中，那剑鼎，已经是被无数的神树枝叶所包裹这，被神树所炼化这剑鼎的力量。
<br/>　　“果然可以炼化。”
<br/>　　楚风眠看到这一幕，心中暗叹到。
<br/>　　这神树不愧是曾经建木的一部分，有着无比强横的威能，哪怕是剑鼎，他都可以吞噬。
<br/>　　曾经炼制剑鼎的材料，许多都都是价值奇高的至宝，在神树的炼化之下，这剑鼎逐渐的开始分解。
<br/>　　从其中源源不断的灵力，被融入到了神树之中去，楚风眠看到，这神树以肉眼可见的速度在生长了起来。
<br/>　　吞噬了这剑鼎，让神树的力量都是提升了无数倍。
<br/>　　楚风眠感受到，他身上的力量，也随着神树力量的提升，也在节节攀升这。
<br/>　　虽然提升的力量，也许是比起剑鼎直接镇压在楚风眠身躯之中的提升要少。
<br/>　　可是这神树加持的力量，却是属于楚风眠自己的力量，这神树扎根于楚风眠的身躯之中，已经是相当于楚风眠的一部分了。
<br/>　　而剑鼎镇压在楚风眠身躯之中的力量，却是属于外力，一旦是碰到强横的人物，就可以像是楚风眠剥夺了荒鎏金身上的剑鼎一样，剥夺楚风眠身上的剑鼎。
<br/>　　神树却是跟楚风眠血肉相连，哪怕是在强横的人物，都不可能剥夺出楚风眠身上的神树来。
<br/>　　不到半个时辰的时间。
<br/>　　在楚风眠的身躯之中，那剑鼎已经是彻底消失不见了。
<br/>　　不剩下一丝一毫，已经全部是被神树吞噬了。
<br/>　　现在一眼看过去，神树上的气息，都是强横了，足足长高了数百米，看起来更加雄伟庞大，已经是有着几分那荒古门神树秘境之中那一株神树的模样了。
<br/>　　单单是神树之中蕴含的力量，楚风眠便是感觉到，至少相当于三四个五行刀圣那个级别的半步天人境强者了。
<br/>　　这种力量，恐怖到了极点，楚风眠一身的实力，也足足提升了五成左右。
<br/>　　楚风眠的境界，已经是向着九劫古帝巅峰大步迈进这。
<br/>　　当然神树的成长，带给楚风眠的好处可不止是力量的提升，而是方方面面的。
<br/>　　楚风眠身上那十二万九千六百枚血肉晶体，如今似乎都是在神树力量的增强之下，开始蜕变这，更加向着不朽真身成长着。
<br/>　　而楚风眠身躯之中的神脉，都是粗壮了许多，其中凝聚的灵力，已经是越发的精粹了。
<br/>　　楚风眠缓缓的抬起手，催动神脉的力量在他的手心之中凝聚，一枚几乎无色的灵石，浮现在楚风眠的手心之中，
<br/>　　这无色灵石，并非是没有蕴含灵力，而是其中的灵力太过精粹了，以至于没有任何的杂质，看起来都没有多少的颜色了。
<br/>　　“神品！这就是神品灵石！”
<br/>　　楚风眠看到他手中凝聚出的那一枚灵石，心中也是狂喜，这就是传说之中灵石品级的巅峰，圣品灵石。
<br/>　　楚风眠之前凝聚出来的灵石，一直都是圣品灵石，哪怕是在纯粹的圣品灵石，也没有达到神品的地步。
<br/>　　要知道神品灵石，可是号称神石，已经不再是一种货币了，而是一种至宝。
<br/>　　布置阵法，凝炼丹药，炼制灵器等等都必须要的至宝，价值连城。
<br/>　　哪怕是在无比古老的时代，这种神品灵石都是无比稀少的，寥寥无几。
<br/>　　每一次出现神品灵石，都会引起一阵血雨腥风的。
<br/>　　现在楚风眠将剑鼎融入到了神树之中去，他的神脉再次蜕变，终于是可以凝炼出来了这传说之中的神品灵石。
<br/>　　跟寻常灵石不同的是，这种神品灵石，可是就连九劫古帝，甚至是半步天人境强者，都无比渴望之物。
<br/>　　楚风眠拿出去，那是绝对的硬通货，不会有任何人不接受。
<br/>　　甚至是在一些古籍的记载之中，这种神品灵石，就算是在天人境强者之中，都有流通，足以证明其价值。
