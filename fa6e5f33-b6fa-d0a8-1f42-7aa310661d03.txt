
<br/>　　<br/>　　    从耿明的嘴中描述出了一个东方庞贝城，水下亚特兰蒂斯的壮观景象，抚仙湖底的古城面积差不多相当于现代城市一个县城的规模了，要知道这种规模的城市哪怕放在古代的话，都相当于是现在的省城了，这个发现绝对非常惊人。<br/>　　<br/>　　    王惊蛰问道：“那你在湖下，有没有发现什么异样？”<br/>　　<br/>　　    耿明眨了眨眼睛，摊着手问道：“发现一座古城和数不尽的尸体，这还不算异样？”<br/>　　<br/>　　    “呃……”<br/>　　<br/>　　    黄九郎这时插嘴接上话头说道：“水下的发现没有异样，但是等他上来之后就有了，他当时回来后就去了附近的村子打听，问湖底古城的事，绝大多数人都是不知道的，只有一些上了年纪的老人，听老一辈人讲过，可能湖底是有座城市的，可也只是听说而已，却没有任何的证据，而我们后来也查过此地的地方志，县地方志的历史最久远的是到唐代，没有任何记载，不过抚仙湖的历史超过两三千前肯定是有的了，查不到也正常，耿明从湖底回来后我们部门联合了一个考古研究所进行了一段时间的推论还有调查，于是在零六年的时候进行过一次具有科学性质的湖底探测，当时还动用了一艘潜水器，你猜这回发现了什么？”<br/>　　<br/>　　    王惊蛰说道：“不管发现什么，依旧还是解不开的谜题”<br/>　　<br/>　　    “扎心了，呵呵”黄九郎指了指他，点头说道：“对，依旧没有解开，不过还是有些发现的，古城四周是一堵南北走向的石墙，其中有道城门，往里是一条由石板铺成的类似于街道的建筑物，随后就是一座气势恢宏的高台式建筑，有十层楼高，从外观来看很像古玛雅人的金字塔，城中还有的建筑类似古罗马的斗兽场，这些建筑都是石料的，规模很宏大，也很难想象出当时的人是怎么建造出来的，后来考古队返回经过论证后，据初步考证，这座古城大致存在于2000年前，极有可能是历史上记载的古老滇国的国都，这是根据湖底那些尸体上衣服的穿着推论出来的”<br/>　　<br/>　　    古滇国的历史相当悠久了，并且由于地处滇西，跟中原隔着几千里呢，这个国度基本是很少被发现的，探知的非常少，所以在历史学术的角度来讲，古滇国的一切都属于被推测和论断的，基本没有什么认知。<br/>　　<br/>　　    但是有一点不可否认，就是越久远，越神秘的国度，就有着越多的隐秘。<br/>　　<br/>　　    几个人聊了很久，最后是把能聊的都聊完了，王惊蛰就看着黄九郎说道：“黄老板，那我想问问，我要是想一探抚仙湖的话，是不是还得要经过你们批准啊？”<br/>　　<br/>　　    黄九郎淡淡的说道：“普天之下莫非王土么……”<br/>　　<br/>　　    “那您是让呢，还是不让呢？”<br/>　　<br/>　　    黄九郎拍了下他的肩膀，笑道：“我主动接近你，其实也是这个意思，我们特别事务处理办公室，虽然权利挺大，涉及面也很广，但毕竟人员的局限性也有，就比如你这种吧，我们就没有囊获其中，多年前还有个古井观的向缺，也照样不搭理我们这茬，所以呢现在策略也改了，不一定要把你们招揽到麾下，有时候合作一把也是可以的……就比如这一次，我就很感兴趣了，因为对于抚仙湖湖底的研究，我们始终都没有什么进展，既然你有兴趣的话，彼此大可以合作一回么，对不对？”<br/>　　<br/>　　    “诚心的？”<br/>　　<br/>　　    “呵呵，事上见……”<br/>　　<br/>　　    关于一探抚仙湖，这是个相当繁琐和危险还有复杂的活动，光是前期准备就得要几天的时间了，当下，王惊蛰和黄九郎定下了调子以后，计划就已经开始运转了起来。<br/>　　<br/>　　    耿明跟王惊蛰讲述着水下注意事项，还有潜水的一些经验，同时黄九郎一个电话打到了军区，调了几个经验丰富的蛙人还有装备过来，甚至他都已经做好准备了，如果有需要的话，都可以动用潜水器了。<br/>　　<br/>　　    黄九郎的力度很大，权限也不小，有他从中运筹帷幄的话，确实比王惊蛰单打独斗要轻松多了。<br/>　　<br/>　　    这要是他自己来的话，估计光下水就能把他给难住了。<br/>　　<br/>　　    王惊蛰去外面上厕所的时候，陈三岁也跟了出来，跟在他身后，小声问道：“王哥，有个事你怎么没跟他们说呢？”<br/>　　<br/>　　    “啥啊？”<br/>　　<br/>　　    “阴雨天的时候，会有状况发生……”<br/>　　<br/>　　    王惊蛰尿完了，抖了抖，然后提着裤子斜了着眼睛说道：“还是年轻，这种合作大家可能都是各怀鬼胎的，他们未必对我推心置腹，我呢也得留一手，你把自己所有的底子都透给人家了，那不是赤果果的一点底牌都没有了么？哥们，以后别和尸体玩了，人容易太单纯！”<br/>　　<br/>　　    这天，黄九郎也在村里租了个民房，就在王惊蛰和陈三岁住的地方附近，他们至少在半个月内都得要安顿在这里了，毕竟前期的准备还得要一阵。<br/>　　<br/>　　    一夜无话，到了隔天，王惊蛰醒了的时候，就一把拉开窗帘，天气预报上讲今天是有中到大雨的。<br/>　　<br/>　　    清晨的窗外，天气已经阴了下来，云层有点低，树杈子已经被风给吹的开始摆动起来，抚仙湖那边的波浪已经渐渐的开始翻滚了。<br/>　　<br/>　　    “要下雨了，起来收拾下，吃完饭我们出门”<br/>　　<br/>　　    半个多小时后，王惊蛰和陈三岁随便吃了口饭，外面就开始掉雨点了，一人拿了一把伞就从家里出去了，路过黄九郎那栋房子的时候，王惊蛰转头看了一眼，就看见对方拢着胳膊站在床边正看向自己。<br/>　　<br/>　　    王惊蛰脚步顿了顿，随即转过头，就接着朝湖边走了过去。<br/>　　<br/>　　    黄九郎看着他的身影，淡淡的笑了笑，低声说道：“还是年轻人的心性啊……”
