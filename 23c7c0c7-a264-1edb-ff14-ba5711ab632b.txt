　　“这，怎么可能？”

<br/>　　一尊尊魔尊，看着眼前的一切，脸庞都僵住了，一个个眼珠子都要掉了下来。
<br/>　　楚风眠的身形，傲立在众人面前。
<br/>　　在他身上，唯有衣衫，稍微狼狈了些，但是他的身躯上，却是没有任何一点伤势。

<br/>　　一尊生死境四重的武者，足以做到这一切？
<br/>　　别说是生死境四重，就算是生死境巅峰，甚至是超越生死，让一尊碎命之境的武者前来，遭遇这样的围攻，都不可能像是楚风眠这样，毫发无伤。

<br/>　　令这本命剑元的威力，提升了十倍不止，他刚刚一次，催动造化之力，便是感受到了其中的可怕。

<br/>　　至于那些魔尊，在楚风眠看来，一个个更是跟蝼蚁，没有任何区别。

<br/>　　楚风眠嘴角露出几分冰冷，这众人的围攻，正好是让楚风眠熟悉这突破的力量。
<br/>　　现在这些人，也该死了。

<br/>　　楚风眠凝聚灵力，位于剑锋之上，突然之间一道剑芒，瞬间斩出。

<br/>　　在其中最为强大的一道剑芒，正是向着小魔主，斩杀而去。
<br/>　　这小魔主，今日必杀，他拥有天魔宝车，对楚风眠有着最大的威胁。

<br/>　　天魔宝车，乃是一件攻守兼备的灵器，这罗伞，单独拿出来都是一件天级灵器，丝毫不比楚风眠曾经遇到过的那先天玉如意要差，甚至还要强上不少。
<br/>　　不过这罗伞在强，楚风眠的剑锋，却是要更加锋利。
<br/>　　他的剑锋，充满剑意，所向睥睨，任何灵器，都无法抵挡。

<br/>　　这一道剑锋，斩在上面七彩霞光上的一刻，顿时这七彩霞光，都是抵挡不住，直接破碎。

<br/>　　就连抵挡楚风眠的剑锋都远远做不到。
<br/>　　这小魔主被逼退，仅仅是楚风眠反击的开始。
<br/>　　他的无数剑芒，已经是向着所有魔尊，斩杀而去，在一瞬间，其中的两位魔尊，他们的身躯便是就被这剑芒贯穿。

<br/>　　一尊魔尊，大声的吼道，这剑芒，无穷无尽，他打破一道，便是就有十道剑芒再次斩杀过来。

<br/>　　楚风眠冷喝一声，他手中的剑锋，再次动了，一道剑光，冲天而起，直接向着这“死”字，斩杀过去。
