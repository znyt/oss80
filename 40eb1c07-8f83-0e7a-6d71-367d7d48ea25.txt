　　“这是？”
<br/>　　人德宫主面带疑惑，看向苏莫身前的棺椁，以及其内如画般绝美的白衣女子。
<br/>　　“宫主，这棺椁内的女子，乃是晚辈至亲之人，不知宫主能否救活她？”苏莫向人德宫主抱拳，郑重的行了一礼。
<br/>　　“老夫看看！”
<br/>　　人德宫主点了点头，长身而起，从主座之上走了下来。
<br/>　　既然已经给予了苏莫庇护，在力所能及的范围之内，他自然不会拒绝。
<br/>　　随后，人德宫主打开了棺椁，手指捏在了芊寻月手臂的脉门之上。
<br/>　　苏莫神情有些许忐忑，人德宫主，在神域都已经算是大能了，想必应该能救活芊寻月吧！
<br/>　　人德宫主查看了少许，随即松开了手掌，面带一丝凝重之色。
<br/>　　“宫主，怎么样？”苏莫紧张的问道。
<br/>　　“想要救活她，不简单！”
<br/>　　人德宫主叹了口气，继续说道：“此女肉身生机已绝，但神魂尚存一丝执念，这丝执念很弱很弱，按理来说，早该消散于天地之间，居然能稳住不散，也是奇迹！”
<br/>　　“宫主，我当年在她体内放置了一枚恢复神魂的石头，应该是此物起的效用！”苏莫道，他的面色凝重不已，连宫主都说不简单，那就真的是困难了。
<br/>　　好在，对方只是说很难，并不是说没有办法。
<br/>　　“不，那石头早已失去了效用！”人德神王摇了摇头，她自然发现了芊寻月口中的小石头。
<br/>　　这的确是一种滋补神魂的宝物，但是，早已没有了任何效果。
<br/>　　“啊！那她怎么坚持到现在的？”苏莫闻言心中一惊，自己真是太大意了。
<br/>　　若是芊寻月连这最后一丝执念都消散了，他就真的无力回天了。
<br/>　　“这丝执念虽然很弱，那是却颇为顽强，说明，有什么在支撑着她，支撑着她存在下去，或许是某个人，也或许是某件事，又亦或是某种未了却的心愿！”人德宫主道。
<br/>　　苏莫闻言沉默，是什么在支撑着芊寻月呢，是因为自己吗？
<br/>　　“宫主，那有什么办法可以救活她？”，少倾，苏莫再次问道。
<br/>　　“老夫有三种办法，但是，均是不太容易，而且目前无法完成。”人德宫主道。
<br/>　　略一沉吟，人德宫主继续道：“第一个办法，在此女陨落之地，动用一种特殊的秘法，于无尽的虚空之中，重新凝聚溃散的神魂，这个办法若是老夫亲自动手，成功率约为两成。”
<br/>　　“只有两成！”苏莫闻言，面色凝重，这个几率未免太低了吧！
<br/>　　况且，芊寻月陨落之地，在苍穹大陆，虽然苍穹大陆已经被他重新组合，但在下界古周星，现在根本回不去。
<br/>　　“两成已经是最乐观的了！”人德宫主摇头说道，其实，他只有一成的把握，想要达到两成的把握，他必须自损修为和精元才行。
<br/>　　想把一个基本已经死亡之人复活，谈何容易，他虽然修为高达高级神王，但也不能逆天改命。
<br/>　　“那第二种和第三种办法呢？”苏莫再次问道。
<br/>　　第一种办法，明显不太现实，不仅成功率太低，而且现在无法下界。
<br/>　　“太古武魂榜，排名第七的武魂名为生死武魂，你若是能找到拥有生死元神之人帮助，则有一定的几率复活此女！”人德宫主面色凝重的道。
<br/>　　“多大的成功率？”苏莫急忙问道。
<br/>　　“成功率的高低，取决于拥有生死元神之人的修为，若是没有神王境的修为，那么，成功率最多一成，若是有高级神王的修为，成功率可达五成！”人德宫主徐徐说道。
<br/>　　苏莫闻言，怦然心动，五成的的成功率，这已经非常的高了。
<br/>　　“第三种呢？”苏莫继续问道。
<br/>　　“太古大道榜，排名第四的大道，名为生死，也就是生死法则，太古大道榜排名第十一的大道，名为生，也就是生之法则，你若是能将这两种法则，参悟到五阶以上，就有救活此女的希望；当然，法则等级越高，成功率自然也越高。”人德宫主详细的讲解。
<br/>　　“生死法则？生之法则？”苏莫闻言沉思，五阶以上才有希望，估计成功率也是不大，必须六阶、七阶、乃至八阶，才能有比较大的成功率。
<br/>　　“嗯，这两种法则，都有起死回生之效，当然，你无需自己去参悟，因为此女可能等不到那个时候了，你可以寻找精深这两种法则之人！”人德宫主道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，沉声问道：“宫主，您见多识广，可知在神域之中，何人有生死元神？何人又精深生死法则或生之法则？”
<br/>　　他自然清楚，凭自己去参悟，还不知道要参悟到何年何月。
<br/>　　而且，他并没有太多的时间，去再参悟一种新的法则。
<br/>　　“参悟生死法则和生之法则之人，不在少数，毕竟这是太古大道榜排名前列的大道，不过，轮回神朝之中，无人在这两种法则之上有太高的成就，天池神朝倒是有几位比较适合。”人德宫主沉吟着说道。
<br/>　　“还请宫主告知！”苏莫恭敬的说道。
<br/>　　“在天池神朝在生死界域，参悟生死法则之人较多，有几位的生死法则，已经达到了极高的境界，不过，老夫向你推荐一人，此人不是生死界域之人，但是最为合适。”人德宫主徐徐说道。
<br/>　　“此人是谁？”苏莫急忙问道。
<br/>　　“终极剑域紫霄神王！”人德宫主郑重的说道。
<br/>　　“紫霄神王？”苏莫闻言一怔，心中不禁有些愕然，此人不就是绫裳所在的终极神王殿的强者吗？
<br/>　　而且，他以前听闻，紫霄神王和绫裳神王，并成为终极剑域绝代双骄。
<br/>　　只不过，当初他只是打听绫裳，倒是没有具体打听这个紫霄神王。
<br/>　　难怪！
<br/>　　难怪当年，绫裳斩钉截铁的告诉他，在神域可以复活芊寻月，原来就是因为这个紫霄神王。
<br/>　　“不错，紫霄神王和天尘宇一样，同时是神域十大无上神王之一，不仅拥有生死元神，更精深生死奥义，只要此人出手，救活此女把握极大。”人德神王点头说道。
<br/>　　“原来如此！”苏莫心中突然敞亮了起来，有绫裳这层关系，找这个紫霄神王帮忙，对方应该不会拒绝吧！
<br/>　　只是让他没有想到的是，这个紫霄神王居然是神域十大无上神王之一，但绫裳好像并不是。
<br/>　　不过，绫裳当年能和紫霄神王，并称为终极剑域绝代双骄，想必其当年的天赋，也不会比紫霄神王差多少。
<br/>　　“宫主，多谢你的指点，那我就先回去了！”苏莫收起棺椁，向人德宫主提出告辞。
<br/>　　“回去吧，明日再来见我！”人德宫主挥手道。
<br/>　　之所以要让苏莫推迟一天离开，不仅是要准备源石，还是为苏莫的安全提前安排。
<br/>　　“是！”苏莫点头，随即告退离开，回到了自己的宫殿之中。
<br/>　　回到宫殿之后，苏莫坐在密室之中沉吟，这一次离开人王宫，他本来是准备去时空界域的，但现在看来，还要去终极剑域。
<br/>　　“先去终极剑域吧！”沉吟了片刻，苏莫做出了决定。
<br/>　　时空界域和终极剑域，都在天池神朝，先去一趟终极剑域，也耽搁不了多长时间。
<br/>　　待救活了芊寻月，他再前往时空界域，进入时空之城，借助其内的时间流速，加速修炼。
