　　赤红色的岩浆世界中，慕清霄闭目盘坐。
<br/>　　由于火鬼王利用了火灵珠中的能量，整个岩浆世界中，火属性能量极其的精纯，而且非常浩瀚。
<br/>　　心法运转过后，慕清霄就感觉到，四面八方的精纯能量都朝自己汇聚过来。
<br/>　　浓郁的能量肉眼可辨，源源不断的汇聚而来，最终将慕清霄包裹，顺着肉身窍穴融入他的体内。
<br/>　　随着心法的运转，精纯的火属性能量也渐渐产生变化，强化慕清霄肉身的同时，不断被阴阳灵气同化，汇入阴阳湖中。
<br/>　　周围的能量疯狂的汇聚过来，在岩浆世界中已经产生了不小的动静。
<br/>　　慕清霄如同老僧入定，而在他的上方早已形成一个大型旋窝，汇聚过来的能量则顺着旋窝中心，灌入他的体内。
<br/>　　不管多少能量，灌入他体内后，都如同石沉大海，根本不起一丝波澜，如同无底洞一样。
<br/>　　强烈的波动，使得周身范围粘稠的岩浆都动荡起来。
<br/>　　随着时间的流逝，两天时间一晃而过。
<br/>　　此刻，整个岩浆世界，岩浆的色泽貌似比原先暗淡了许多，不知何时，已经从赤红转化成了红色，似乎少了一丝色泽。
<br/>　　甚至，原本高的可怕的热量也下降了不止一个层次。
<br/>　　如果火鬼王在此，肯定能感觉出，岩浆世界中的精纯能量已经完全消失不见，取而代之的是普通的岩浆。
<br/>　　与此同时，慕清霄上方的旋窝也渐渐消散，动荡的岩浆世界也在下一刻恢复平静。
<br/>　　剑眉一抖，慕清霄逐渐睁开双眼，星眸中光泽流转。
<br/>　　此时，他只觉得通体舒畅，全身经历了又一次的升华，忍不住深吸一口气，随后轻轻的吐了出来。
<br/>　　原本归为平静的岩浆世界，再一次剧烈动荡起来，粘稠的岩浆都已肉眼可见的程度荡起一丝丝涟漪。
<br/>　　要知道，这可是在地底六千多米的地方，更何况周围全是岩浆，可见其重压是何其的恐怖。
<br/>　　但是，慕清霄仅仅只是吐出一口气，便令岩浆世界都荡起波澜，可见大乘期修士的恐怖。
<br/>　　大乘期四重！
<br/>　　毫无疑问，炼化地狱冥火，并且将岩浆世界中的精纯能量吸收后，慕清霄直接突破了两个小境界。
<br/>　　但这两个小境界，对于常人来说，是一辈子都无法跨越的沟壑。
<br/>　　突破境界，慕清霄受益良多，除了体内的阴阳灵气变得更加的磅礴浩荡外，最主要的还是肉身的力量。
<br/>　　修为强大很重要，但肉身却更重要，对于大部分修士而言，注重练气，反倒是忽略了肉身。
<br/>　　肉身的强大，可让慕清霄在同境界上碾压对方，甚至越级杀敌，配合魂剑与神通，轻而易举的就能将击杀。
<br/>　　也正是因此，慕清霄才更加注重肉身的强大。
<br/>　　为此，他在平日里可没少炼制锻体的丹药，甚至搜集各种灵药炼制高品质淬体液。
<br/>　　除了阴阳灵气对肉身的改造，在魂剑吸收各种力量后，反哺回来的力量也会改造肉身。
<br/>　　现在，慕清霄的肉身已经强大到了一定的程度，修炼《御女心经》，他注定会在这条道路上越走越远。
<br/>　　回过神来，慕清霄敏锐的发现，自己对道的感悟更深了一些，这种感觉很模糊，但却真的存在……
<br/>　　微微摇头，不再胡思乱想，慕清霄相信，只要实力越强，这种感觉估计就会越发的清晰。
<br/>　　至于对道的感悟，估计自己也只是触及到了皮毛而已。
<br/>　　长身而起，地狱冥火已经炼化，岩浆世界中的精纯能量也已经吸收，火灵珠也已经拿到手，他已经没有逗留在鬼界的必要。
<br/>　　回头看了一眼极乐世界，可能是因为火鬼王已经死亡的缘故，空间结界似乎变得薄弱了许多。
<br/>　　或许普通的合道修士都能将结界轻易击碎。
<br/>　　慕清霄嘴角牵起一抹笑意，双手结印，牢固的结界扩展而出，不多时便将整个极乐世界层层包裹住。
<br/>　　界中界！
<br/>　　除了火鬼王原本布下的结界，慕清霄另外添加了一层。
<br/>　　如此一来，极乐世界中的鬼族无法从其中出来，而外面的鬼修也无法进去，除非有鬼圣出手击碎结界。
<br/>　　只是随意布置了个结界，慕清霄也没花心思，到时候他早已不在鬼界，哪怕鬼界高层也干涉不到他。
<br/>　　突破境界总共用了两天的时间，也是时候离开了。
<br/>　　想到此处，慕清霄就向极乐世界上方掠去，犹如追星赶月，风驰电掣间，便携着破空声掠出火山。
<br/>　　终于告别一成不变的赤红色世界，抬头望着昏沉的天空，鬼界这种地方，没有太阳与月亮，真不适合人族居住。
<br/>　　哪怕是他也极其的不喜欢这里，微微摇头，慕清霄就转身向冥河岸边掠去。
<br/>　　在慕清霄前脚刚离开，便有一股能量波动自火山山体下升腾而起，随着一个诡异图案的出现，几道身影就凭空而现。
<br/>　　其中一名身穿麻布衣，背负黑色长剑的青年，望着周围景象，啧啧称奇。
<br/>　　“这里就是极乐世界啊，除了温度高点，天色暗点与世俗也没什么不同嘛，就是不知道那火鬼王漂不漂亮？”
<br/>　　“死菜牙，我可警告你，你待会可别在美色面前迷失了心智。”
<br/>　　对于两人斗嘴吵架，其他人已经见怪不怪。
<br/>　　“哥哥，赵无延大哥瞒着其他阴差，利用传送阵将我们传送到这里，我们还是快点去找火灵珠吧？”
<br/>　　另一名身穿白袍，背负长剑的青年微微点头。
<br/>　　“景兄弟，龙葵说的有道理，难道你忘了？在古藤林的时候，土灵珠就被人捷足先登，等获得火灵珠后，在欣赏这里的景色吧。”
<br/>　　“切。”
<br/>　　景天撇了撇嘴，不屑道：“我们能进入鬼界，并且来到极乐世界，可都是因为赵无延，本大爷可不信，别人也能来鬼界。”
<br/>　　如果慕清霄在这里，肯定能认出，这一行人正是剧情的主角。
<br/>　　不过，火灵珠已经被他拿走，景天等人注定要白跑一趟。
<br/>　　甚至，极乐世界外有他布置的结界，他们能不能进入其中都是两说。
