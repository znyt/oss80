　　怎么可能！

<br/>　　铜钱尽数落在那僵尸身上，结果……直接就被腐蚀了，液化成汽，消失在空气中。

<br/>　　“我要是逃出去，你就不好逃了。你先帮我撑一会。”

<br/>　　那边，僵尸爬到离毛小方不到几米远的时候，猛然间腾空而起，朝毛小方抓过来。

<br/>　　“咣”的一声巨响，听在人的耳朵里还不算什么，但是对邪物来说，却无异于一道奔雷霹雳。
<br/>　　僵尸抽搐了一下，被震开了几米远，摇头晃脑地，口中流出了粘液，看上去更加地癫狂了，抖动着双臂，再度爬了过来，这家伙也有灵智，感觉毛小方手中的道钟不好对付，不太想跟他正面刚，试图攻击躲在后面列阵的叶少阳。
<br/>　　它也看出毛小方在尽力保护着叶少阳，于是绕着圈子，想要先拿叶少阳开刀。
<br/>　　毛小方也只好转着圈子，挡在叶少阳身前，说道：“少阳子，你在做什么，还不快跑？”
<br/>　　“快好了。”
<br/>　　叶少阳刚说完这句话，那僵尸急不可耐地扑了上来，毛小方再度敲响了道钟，结果这一次僵尸有了前车之鉴，道钟的声音刚响起，僵尸就大吼一声，口中喷吐一股强烈的尸气，生生将钟声造成的灵气波动震碎，手一抬，朝毛小方脑门扇过去。
<br/>　　毛小方情急之中也不管道钟了，直接用敲钟的桃木棍挡了一记。
<br/>　　与僵尸的爪子撞在一起，毛小方立刻感到手臂一麻，人也向后倒退了几步，抬头看去，僵尸已经越过了他，朝叶少阳扑过去。
<br/>　　糟糕！
<br/>　　毛小方刚要作法提防，叶少阳从后面一把推开了他，直面僵尸。
<br/>　　僵尸双手搂起，拍叶少阳两边太阳穴。
<br/>　　在他两条筋肉相连的手臂接触到叶少阳脑袋的一刹那，叶少阳欺身后退，双手结印，念道：“太上老君急急如律令！天授神威，撒豆成兵！”
<br/>　　僵尸落下的地面上，发出炒豆子一般的爆炸声，接着灵光乍现，组成了一个太极图，叶少阳的七星龙泉剑就反插在太极图的中间，剑身发出夺目的紫光，汇入双鱼图中。
<br/>　　强大的灵力，如同无数道尖刀，对着僵尸的身体切割起来……
<br/>　　毛小方看到这一幕，总算明白了叶少阳刚才在干什么：他用剑在地上刻下的是符文，埋进铜豆子，同时用七星龙泉剑定阵，之后引僵尸过去，用撒豆成兵的法术引燃阵法，之后又借助七星龙泉剑的灵力，将阵法发挥到了极致……
<br/>　　真是……有想法。
<br/>　　实际上，叶少阳也是不得已才这么做，自己现在只有真人的实力，不管什么样高明的阵法，在自己手中都发挥不出什么作用，只有借助七星龙泉剑的灵力，来激活高等阵法，达到自己期望的效果。
<br/>　　在阵法之力的绞杀之下，那僵尸立刻一身是伤，血肉模糊起来，身体扭曲着，试图从阵法之中冲出去，但根本爬不起来。
<br/>　　叶少**本不想去管它的死活，从地上拔起七星龙泉剑，唤了毛小方一声，朝谷口方向冲过去。
<br/>　　冲出了阴云，两人也到了谷口，转头看去，那团阴云再度凝聚起来，形成了细长的一团，再度狂卷而来。
<br/>　　两人不敢怠慢，一路飞奔，毛小方一路跑，突然叶少阳拉了他一把，“毛师兄，你看！”
<br/>　　毛小方转头看去，只见那团黑影，跟那些失去的人魂的鬼魂一样，停留在了谷口的位置，那些鬼魂一个个张牙舞爪，对着自己这边发出吼叫，但是宛如被一层看不见的玻璃挡住，不敢离开谷口一步，那团阴云也是一样，在谷口位置停留不前。
<br/>　　怎么会这样！
<br/>　　叶少阳和毛小方互相看了一眼，两人心中都是感到惊讶不已，停留了片刻，叶少阳大着胆子走回到谷口边上，离那些鬼魂只有十来步远的距离。这些鬼魂仍然踟蹰不前。
<br/>　　毛小方也走过来，叶少阳招呼他离开这里，往前走了一会，再回头看，那些鬼魂不再观望他们，而是回到之前的正常的轨迹上，继续在山谷里绕圈子。

<br/>　　“这是怎么回事，他们为什么不追出来？”毛小方惊奇地说道。
<br/>　　“那阴云里，还有别的邪物，那个长的像僵尸的不是唯一的邪物。”叶少阳道，“如果真的追过来，以我们两个，真的不一定跑的掉。”
<br/>　　毛小方点点头，道：“我明白你的意思。”
<br/>　　邪物杀人，根本不会留情，尤其是针对法师，只要双方实力相差不多，基本上都会拼死一战，不然放走一个法师，说不定会引来一群。
<br/>　　就拿眼下来说，既然对方还有后手，那不应该就这么放任他们离开。
<br/>　　这有悖常理的事情，让叶少阳和毛小方实在不解，坐在地上冥思苦想。
<br/>　　毛小方道：“只有一种可能，那山谷里的邪物，或者说是什么邪恶力量，没有办法离开山谷。”
