　　对于自己的弟弟被人残害这件事，屠苏元让其实并不怎么在意。
<br/>　　他和屠苏元庆乃是异母兄弟，并没有太深的感情，反而是竞争家主之位的对手，巴不得对方早点死呢！
<br/>　　不过，等他朝画像上瞄了一眼，两眼顿时便再也挪不开了。
<br/>　　“竟然是他！”
<br/>　　一旁的屠苏玄眉头一皱，
<br/>　　“怎么？元让，你认识这个凶手？”
<br/>　　屠苏元让深吸了一口气，道：
<br/>　　“父亲大人，可还记得我刚刚向您说的那个叶尘吗？此人就是叶尘！”
<br/>　　“什么！”
<br/>　　屠苏玄顿时两眼一瞪，直接霍然站起身来，
<br/>　　“你说的是那个，一口气通过了四场丹师考核，而且是一连炼制出三枚完美品阶丹药的叶尘？”
<br/>　　原来，
<br/>　　叶尘在考核大会上的表现，已经第一时间，在整个云丹城的上层传递开来。
<br/>　　现如今，云丹城的各方势力，都想将叶尘才招入麾下，他已经是炙手可热的人物。
<br/>　　之前，屠苏玄和屠苏元让，两人就是在商议此事。
<br/>　　然而他们却怎么也没有想到，打伤屠苏元庆的凶手，居然就是这个叶尘！
<br/>　　“你确定就是此人吗？”
<br/>　　屠苏玄微微愣了片刻，露出一脸凝重之色。
<br/>　　一方面，他也想将这个新崛起的丹道天才，招入屠苏家的阵营。
<br/>　　可是另外一方面，看到自家儿子被人打成这幅凄惨的模样，他也不可能无动于衷。
<br/>　　屠苏元让点了点头，
<br/>　　“父亲大人，我确定，就是此人！”
<br/>　　“轰！”
<br/>　　屠苏玄一掌拍在旁的石桌上面，那用灵石打造的石桌，顿时出现了一个清晰的掌印，
<br/>　　“妈的！原本老子还打算招揽此人，可是他竟然将元庆打成这幅模样，我岂能容他！”
<br/>　　一旁的屠苏元让，顿时心头一喜，
<br/>　　他原本就对叶尘嫉妒的发疯，根本不想将此人招入屠苏家，只是苦于没有合适的理由，去说服自己的父亲。
<br/>　　如今好了，出了这档子事，却正中他的下怀！
<br/>　　“父亲大人，此人既然已经和我们屠苏结怨，纵然您既往不咎，他只怕也不会加入我们屠苏家，既然我们得不到，那也不能让其他势力得到，必须尽快杀之，以除后患！”
<br/>　　“否则，若是让其他势力招揽了去，到时候我们再动手，可就要顾忌重重了！”
<br/>　　“再者，此人的炼丹手法颇为玄妙，应该是有过什么奇遇，若是我们能率先擒获自然，用搜魂之术，将他的炼丹手法逼问出来，必然可以提高咱们屠苏家整体实力！！”
<br/>　　屠苏玄听完儿子这番话，顿时两眼一亮，眼眸中闪过一抹炙热之色，
<br/>　　“好！就这么办！你速速去请三位长老前来，一同商议此事，顺便再派人去云丹宗一趟，将此事告知你爷爷知晓！”
<br/>　　屠夫元让顿时大喜，连忙道：
<br/>　　“父亲大人英明！”
<br/>　　……
<br/>　　却说叶尘离开飘香苑之后，动用神念之力，很快便锁定了灵山等人的气息。
<br/>　　大约半个小时之后，双方汇合在了一处，灵山看到两个女儿没事，自然是欢喜不已，
<br/>　　“叶先生，我们接下来怎么办？”
<br/>　　众人纷纷望向叶尘。
<br/>　　叶尘沉吟了片刻，方才缓缓道：
<br/>　　“如今我们已经得罪了屠苏家，据我所知，这屠苏家是云丹城的五大家族之一！”
<br/>　　说到这里，众人的脸色顿时全都无比难看。
<br/>　　说起来，他们也真是够悲催的，好不容从巨门星域逃出来，来到着云丹城，结果这才刚一进城，就得罪了当地的地头蛇。
<br/>　　“叶先生，您就直接发话吧！怎么做，我们全都听您的！”
<br/>　　灵山再次开口，其余众人也都纷纷点头。
<br/>　　叶尘深吸了一口气，道：
<br/>　　“两条路！”
<br/>　　“第一，要么直接离开云丹星域，我可以带你们去其他星域，彻底避开屠苏家！”
<br/>　　“第二，继续留在云丹城，不过你们今后行事，必须要万分小心，不能让屠苏家的认出你们来！”
<br/>　　众人听到这话，顿时面面相觑，一时间全都没了主意。
<br/>　　离开云丹星域，貌似是一个不错的选择，可是彻底避开屠苏家。
<br/>　　可是想要再找到一个，像云丹城这样安逸的地方，实在太不容易了！
<br/>　　就像他们之前所在的巨门星域，常年战争不断，朝不保夕，说不定叶尘一离开，他们就要死在战乱之中了。
<br/>　　“整个云丹城，有上亿人口，屠苏家再厉害，想要找到我们，也没有那么容易吧？”
<br/>　　有人忍不住嘀咕了一句，顿时得到其他众人的相应。
<br/>　　叶尘想了想也对，
<br/>　　屠苏元庆那帮人，虽然和他们照过面，可是想要在人海之中找出他们，却也不是件容易的事情。
<br/>　　大不了，他离开之前，再去一趟屠苏家，把之前那些人全部杀光！
<br/>　　想到这里，叶尘点了点头，
<br/>　　“既然你们决定了，那我就不多说什么，不过你们今后在云丹城如何生存，可曾想过吗？”
<br/>　　提及这个，灵山顿时笑道：
<br/>　　“叶先生放心！我们都已经打听过了，按照云丹城的规矩，只要在云丹宗设立的事务处，办理一道居住手续，便能在云丹城永久居住，不会被执法队驱逐！”
<br/>　　“而且，云丹城灵气充沛，丹药也十分便宜，十分适合修炼，我们平时可以出城采药，或者猎杀妖兽，来换取灵石……”
<br/>　　灵山侃侃而谈，显然已经做好了不少功课。
<br/>　　叶尘点了点头，
<br/>　　“既然如此，那我们就先去办理居住手续吧！”
<br/>　　叶尘让人雇了一辆车子，是用灵兽驱使的那种，可以承载上百人，在上千米宽的街道上，可以肆意奔腾。
<br/>　　半个多小时之后，
<br/>　　众人来到一座庄严、大气的高楼之前，有点类似于地球上的政府大厦。
<br/>　　这便是灵山口中的事务处了。
<br/>　　然而，
<br/>　　等灵山进去问了一下情况之后，顿时所有人全都傻眼了，
<br/>　　原来，办理居住手续不难，可是每人要缴纳十万块上品灵石！
<br/>　　他们一行人共二十多人，那就是足足二百多万块上品灵石！
<br/>　　“太贵了！我们根本住不起啊！”
<br/>　　灵山等人，全都沮丧不已。
<br/>　　叶尘其实早就料到，想要成为云丹城的永久居民，肯定不会那么容易。
<br/>　　否则，云丹城这么一个绝佳的修炼之地，恐怕在就被挤爆了！
<br/>　　“灵石的事情，你们不用操心，我来想办法！”
<br/>　　叶尘心里早就有了主意。
