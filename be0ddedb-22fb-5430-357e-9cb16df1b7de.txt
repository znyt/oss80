
<br/>　　<br/>　　    陇西是座古城，秦汉至隋唐时期叫陇西郡，包括天水兰州一带，是古代丝绸的必经之路也是兵家必争之地，这里出李姓名人，李白，李渊，李广都出自陇西郡。<br/>　　<br/>　　    如果你问问寻常人国内的几大家族都是哪几个，除了几个不能说的以外，有人可能会讲了应该是曲阜孔家，传承千年孔姓子弟遍天下，名人数之不尽。<br/>　　<br/>　　    也有人会说满洲皇室爱新觉罗家族或者周庄沈万三世家，可能还会有人讲港岛被称为李家的城，这也能算一个。<br/>　　<br/>　　    但要说到五姓七族世家门阀恐怕知道的人就不多了，比如陇西李氏，清河崔氏还有太原王氏，因为据说武则天掌权之后这些世家大门阀的势力就被削弱了，从而成为了历史，到如今还哪有什么陇西，太原，清河等世家门阀的传说啊，常人大部分都是这么认为的，但事实呢？<br/>　　<br/>　　    就说陇西李氏这乃是正经的皇族，哪怕是大唐已成烟云消散成为了过去，但李家的皇族子弟你知道都在哪里么？<br/>　　<br/>　　    王惊蛰是在离开京城的三天之后，一路辗转折腾抵达陇西的，但他要去的地方却不在陇西城内，而是一处挺偏的老城地带。<br/>　　<br/>　　    这片老城年代挺老，很多房子还保留着历史的面貌和现代建筑一眼就能看出区别来。<br/>　　<br/>　　    王惊蛰站在一处大宅门的前，这门确实挺大的，至少得比冯宅的四合院院门大了两倍不止，特别是门上的那块牌匾看着更是给人生出一股压迫感了，除了李氏两字以外，落款处是一个小篆，四周全都是雕龙刻凤，中间是个篆体的李字。<br/>　　<br/>　　    “贵气扑面来啊······”王惊蛰嘀咕了一句：“到底是皇族底蕴，这宅子啧啧，还真是不错呢”<br/>　　<br/>　　    “铛，铛”王惊蛰敲了下门环，片刻后一个穿着唐装佝偻着肩膀的老人将大门开了一条缝隙。<br/>　　<br/>　　    王惊蛰整理了下长袍，挺直了脊梁，一字一顿的说道：“朝歌王家一门，王惊蛰来访陇西李氏”<br/>　　<br/>　　    朝歌是个小地方，一个名不见经传的小城，如果往前推千年的话，这个地方倒算是鼎鼎大名。<br/>　　<br/>　　    商朝国都，朝歌。<br/>　　<br/>　　    “李伯，照你看这青年是个什么样的人呢？”几分钟后，李氏大宅门内正厅，两张太师椅上坐着一对中年男女，手里捧着青花瓷的茶杯，有点皱眉的问了一句。<br/>　　<br/>　　    李伯就是刚才为王惊蛰开门的佝偻着肩膀的老人，他是李氏一族的老仆人。<br/>　　<br/>　　    “挺年轻的，大概二十三四岁左右，从面相看也很沉稳，不像是浮夸满面的青年，说话的时候中气十足······”李伯中规中矩的点评了几句，最后想了想又说道：“挺普通的一个人”<br/>　　<br/>　　    这对中年男女对视一眼，都没有说话，看出来是什么反应，老仆李伯在李家的地位虽然是是个仆人，但主人可是他一手带大的，就连后来跟主母生的少爷和小姐也是他带成人的，所以地位比较超然。<br/>　　<br/>　　    这对男女就是现今陇西李氏的家主和家母了，男的叫李兰荣，女的叫茅清水。<br/>　　<br/>　　    李伯看两人都没说话，就又问了一句：“这婚约·······”<br/>　　<br/>　　    李氏大小姐有婚约的事知道的人不多，大概也就族内一些老人知道，这婚约源自于二十几年前，李氏当时的家主也就是李兰荣的父亲定下的。<br/>　　<br/>　　    据闻，李家大小姐定的婚约是和一王姓子弟，但对方具体什么身份却不太了解。<br/>　　<br/>　　    李兰荣和茅清水也不太清楚。<br/>　　<br/>　　    李伯说道：“老太爷当初怎么会给大小姐定下了这么个婚约”<br/>　　<br/>　　    “如果老太爷还在世的话，你倒是可以去问问他是出于什么考虑给小姐定下了这样一门婚约，反正我是不太清楚的”茅清水说这话的时候颇有些不满，父母之言媒妁之命什么的在她这里好像不太好使，因为当年老天爷在李氏族内有着绝对的决定权，他说有婚约，婚约就有了。<br/>　　<br/>　　    李伯又皱眉说道：“可是，清河崔氏不是也要来提亲了么？”<br/>　　<br/>　　    李氏一族的大小姐已经到了待嫁之年，这几年都有人来上门提亲，不过李兰荣和茅清水一直都在压着没提这一茬，但是大概在一年前左右，清河崔氏派人来上门了，他们家的少爷也到了婚嫁的年龄，想和陇西李氏结亲。<br/>　　<br/>　　    说白了这就是一场联姻，两个曾经的世家门阀要结合在一起，然后想更上一层楼。<br/>　　<br/>　　    关于这门亲事，李兰荣和茅清水都是颇为满意的，据说这崔家的少爷十分优秀，年纪轻轻就从国外留学归来，年少的时候又拜了一位大师学习风水堪舆之术什么的。<br/>　　<br/>　　    简单一句话来形容，就是此人实乃人中龙凤啊。<br/>　　<br/>　　    清河崔氏提亲的日子就在最近，但没想到的是，李氏老爷子在二十多年前定下的一门亲事，在今天也登门了，这一下就有点尴尬了。<br/>　　<br/>　　    “人还在外面等着呢·····”李伯提醒了一句。<br/>　　<br/>　　    李兰荣皱了皱眉，茅清水则笑了：“你说老爷子定的亲事，是给谁定的？”<br/>　　<br/>　　    “我们家大小姐”<br/>　　<br/>　　    “咱家大小姐应该姓什么？”茅清水又问道。<br/>　　<br/>　　    “自然应该是姓李的”李伯说完这话，顿时就恍然了。<br/>　　<br/>　　    茅清水笑了笑，拍了下手后说道：“对啊，当年这门亲事是给李家大小姐定的，来提亲的这姓王的青年，娶的也应该是李姓大小姐······但是，这姓李的大小姐在哪呢？”<br/>　　<br/>　　    李兰荣瞥了老婆一眼，皱眉说道：“这可有点胡搅蛮缠了”<br/>　　<br/>　　    “那我的话有没有道理呢？有姓李的大小姐我自然可以许配给他，但是······要没有呢？”<br/>　　<br/>　　    李伯恍然问道：“主母，您这是早就料到会有这么一天了？”<br/>　　<br/>　　    茅清水说道：“我可不想我的女儿随随便便的就嫁给一个不知道从哪冒出来的不知名的野小子，他和清河崔氏的少爷，哪里有什么可比性？我女儿，要嫁就得嫁人中龙凤才行！”<br/>　　<br/>　　    是的，早年老爷子死后，茅清水就对这门亲事反悔了，她不想把李家大小姐嫁给这姓王的。<br/>　　<br/>　　    所以，李家大小姐如果不姓李，那是不是就不用嫁了？<br/>　　<br/>　　    毕竟，当初老爷子答应下来的是李姓的大小姐。<br/>　　<br/>　　    呵呵，一个文字游戏罢了。
