　　“额，明天回去再分呗。”
<br/>　　谢雨晴没说什么，把东西放下，找到叶少阳的背包，有一根背带断了，叶少阳简单打了结，勉强能用。
<br/>　　“我找老板刚要了针线，给你缝上吧。”说着坐在沙发上缝起来。
<br/>　　叶少阳看她笨拙的样子，笑道：“给我自己来，你不擅长这个。”
<br/>　　“为你缝一次吧，以后也没机会了。”
<br/>　　“啥意思啊。”
<br/>　　见她不理会，叶少阳挠着头，纳闷道：“我怎么觉得你今天怪怪的。”
<br/>　　谢雨晴不说话，给他缝好了，坐了一会，说声去洗澡，就钻进了浴室。
<br/>　　出来的时候，身上裹了浴巾，有点羞涩地让叶少阳把灯关了。
<br/>　　“你……干啥？”
<br/>　　“睡觉啊。”谢雨晴自己把灯关了。
<br/>　　“在这睡？”
<br/>　　谢雨晴爬上了床，推倒叶少阳，爬到他身上去，叶少阳挣扎了几下，被他堵住了嘴巴……叶少阳心中惶然不已，这连着好几天，两人虽然有点小暧昧，但都是秋毫无犯的，今天谢雨晴也没喝酒啊，怎么这么主动……
<br/>　　衬衣被扒了，谢雨晴自己也脱了浴袍，全身贴上来。
<br/>　　“这！”
<br/>　　叶少阳不淡定了，反抗的动作也越来越小，但关键时刻还是用力推开了她。
<br/>　　窗外有路灯和月光斜射进来的光，叶少阳看到谢雨晴坐在床上，一片白花花的，于是拿起浴袍给她披上，突然又被扑倒了，在他耳边吐气如兰：
<br/>　　“少阳，我只问你一次，你要不要我？”
<br/>　　这……
<br/>　　“我知道你是第一次，也对得起我了。”她埋头在他胸前，“哪怕只有今晚，你是我一个人的……”
<br/>　　她没什么动作，叶少阳也就没防备了，双手按着她的肩膀，一时不知道该说什么，该做什么。
<br/>　　谢雨晴突然就哭了，趴在他胸前大声哭起来，双手紧紧搂着他的脖子，叶少阳也不知道她是怎么了，只好安慰她。
<br/>　　谢雨晴哭了一会，慢慢冷静下来，手在他胸口轻轻抚摸着，静静说道：“明天你拿好东西，自己回去吧，我很早就走了，你也不用送我。”
<br/>　　叶少阳一怔，“你不跟我一起走？”
<br/>　　“我要去云南了。”
<br/>　　“去干啥？”
<br/>　　“工作。”
<br/>　　“什么！”叶少阳隐隐然意识到什么。
<br/>　　“嗯，你还记得上次你去警局，我爸爸去我办公室催我的事吗，因为我申请了调去云南，他当时是来确定，因为名额有限。”
<br/>　　“为什么啊，为什么要去那么远？”
<br/>　　“我想去缉毒，这算是我的梦想吧，就是但跟一个缉毒特警，战斗在第一线，而且我很喜欢云南。”谢雨晴一只手摸着他的脸，笑了笑，“我不回来了。”
<br/>　　叶少阳心头一紧，停顿了一下，问道：“是因为我吗？”
<br/>　　谢雨晴没作声，算是默认了。
<br/>　　叶少阳内心紧张起来，失声喃喃：“静茹走了，现在你也要走，为什么你们都要这样，啊，静茹出国，你是知道的？”
<br/>　　“知道，她走了之后，跟我一直有联系，我劝过她回来，但她后来说了很多，我就在想，或许我也应该这样，正好又有一个机会可以外调，我犹豫了几天，就答应了……我本来不想告诉你的，想走之后给你发个微信说一声，但我怕你太难过了，还是当面跟你说了。”
<br/>　　“这样我就不难过了吗？”叶少阳眼角流下泪来。
<br/>　　谢雨晴笑笑：“傻瓜，我会好好的，我还年轻，有的是时间去调整，只是，我可能不会再爱上任何人了……”
<br/>　　叶少阳还想说什么，谢雨晴往上凑了凑，搂住他的脖子，说道：“我想跟你睡一夜，万一能怀上孩子，我就生下来，自己去抚养，我不会让他见你的，也不会打扰你的生活，少阳……”
<br/>　　她凑上来，吻住他的嘴，眼泪全都流在了叶少阳的脸上。
<br/>　　第二天，叶少阳醒来，已经日上三竿，谢雨晴不见了。想到昨晚的事，叶少阳一下子跳起来，穿上衣服冲出房门，去敲谢雨晴的门，敲了半天没人理，正想回去拿手机看看时间，门打开了，谢雨晴已经穿戴整齐，提着她的行李箱，冲叶少阳抿嘴笑起来，“干啥，你要送我吗？”
<br/>　　“我……我想请你不要走了。”
<br/>　　谢雨晴脸上闪过一丝不自然，又笑了笑，叶少阳看她这样子，眼泪一下子就下来了。
<br/>　　“你怎么傻傻的。”谢雨晴强忍着，拿出纸巾给他擦了眼泪，双手捂着他的脸，深吸一口气，抿嘴笑道：“好了，我要去机场了，你回去睡觉吧。”
<br/>　　“我、我送送你吧。”
<br/>　　叶少阳飞快进屋去穿衣服，行李昨晚就收拾差不多了，再收拾了一下，立刻就出门了。谢雨晴还在，一个人趴在栏杆上望着远方。
<br/>　　两人一起下楼，叫车去机场。
<br/>　　“能不能不走？”叶少阳可怜兮兮地问他。
<br/>　　“你给不了我要的东西，不过，你也已经给了很多了，这些经历，我会一辈子都记得。你要好好的，我偶尔会跟你联系的，我相信你，少阳，你一定可以打败无极鬼王……而我，跟你终究不是一个世界的人，我跟静茹一样，我们都是平凡人，谢谢你，给我一段这么美好的经历。”
<br/>　　叶少阳听着鼻子又泛酸了。
<br/>　　他想说点什么，但是车来了，谢雨晴上了车。叶少阳也跟着上去。
<br/>　　一路上谢雨晴滔滔不绝，像个没事人似的说着自己去云南之后的打算，叶少阳知道她在强撑，几次想开口挽留她，但是忍住了。
<br/>　　是啊，自己什么都给不了她，她应该有全新的生活。到这时，叶少阳才明白过来，为什么她要跟自己单独旅行，也只是为了留下更多美好的记忆吧。
<br/>　　“对了，你那个灭灵钉还在我这里……”谢雨晴伸手去包里拿，叶少阳按住她，道：“你留着防身吧。”
<br/>　　“好吧。也许有一天，你会再看到它的。”
<br/>　　到了机场，两人一起取票，却是一南一北，两个相反的方向。
<br/>　　过了安检之后，叶少阳先陪谢雨晴到她的登机口，谢雨晴不再说话，叶少阳满心惆怅，不知道说什么好，就去倒了杯水给她。
