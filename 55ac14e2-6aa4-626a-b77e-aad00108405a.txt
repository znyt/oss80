　　这不是普通的猿猴，而是黄金巨猿！
<br/>　　此黄金巨猿战魂，和苏莫当初在苍穹世界蛮荒山脉之中，救千寻月之时，所见到的那头黄金巨猿一模一样。
<br/>　　当然，那头是真正的黄金巨猿，身高达数百丈，体积远不是这战魂可比。

<br/>　　黄金巨猿战魂，雄壮的体魄如同一座小山，气息狂暴，霸气无边。
<br/>　　“圣猿拳！”
<br/>　　滕金山出手了，居高临下，再次一拳击出。
<br/>　　与此同时，他背后的黄金巨猿战魂，同样挥拳出击，姿势与滕金山一般无二。
<br/>　　无形的战魂之力加持在滕金山的身上，让他气势暴涨，实力暴增。
<br/>　　轰！
<br/>　　一拳击出，狂暴的拳劲再次化为了一只巨大的拳头，碾碎虚空，镇压而下。
<br/>　　这一拳更加的强大，巨大如上的拳头之上，紫金光芒闪耀，拳头之上的纹路清晰可见，分明就是黄金巨猿的拳头。
<br/>　　轰隆隆！
<br/>　　拳头未至，强大的拳力威压便碾压而下，让得下方无尽的海水炸开，形成排上倒海的浪涛，向四方狂涌。
<br/>　　轰隆隆！
<br/>　　巨大的拳头掀起滔天狂风，仿佛天神之拳，在苏莫的眼中急速放大。
<br/>　　“不过如此！”见识对方出的两招，苏莫心中对于对方的实力，已经有了大概的了解。
<br/>　　对方虽然强大，但与他差距太大了。
<br/>　　此人的实力，比之巫玛都要弱很多，更别提与他相比了。
<br/>　　当然，巫玛是武帝境六重修为。
<br/>　　苏莫再次举起了手中的长剑，血脉之力沸腾，强大的剑威冲天而起。
<br/>　　“战！”
<br/>　　一声轻喝，长剑一挥，犀利的三色剑气闪烁着琉璃色的白光，逆斩而上，迎击镇压而来的巨拳。
<br/>　　这一剑，苏莫不仅催动了血脉力量，更是催动了体内的四种剑之战魂。
<br/>　　耀眼的剑气，锋锐无匹，斩裂空间，瞬间便斩在了巨拳之上。
<br/>　　嗤！
<br/>　　轰隆隆！
<br/>　　而后，之所有人震惊的目光中，剑气直接便撕开了巨大的拳头，让得拳头炸裂，天地为止剧震。
<br/>　　而剑气所向披靡，速度丝毫不减，瞬息千里，直击天穹之上的滕金山。
<br/>　　“什么？”滕金山见此，顿时面色一变，苏莫的强大出乎了他的预料。
<br/>　　眼见剑气临身，滕金山双臂交叉，向身前一挡，赫然是要硬抗苏莫的剑气。
<br/>　　嗡~~
<br/>　　滕金山将金刚之体的血脉力量催动到了极致，全身紫光耀眼，闪烁着金属般的光泽。
<br/>　　下一息，耀眼的剑气狠狠的斩在了滕金山的身上。
<br/>　　铛！
<br/>　　一声刺耳的金铁交鸣之声响起，剑气仿佛是斩在了一件宝器之上，顿时崩碎开来。
<br/>　　而滕金山的身躯，虽然挡住了剑气的锋芒，虽也为伟岸的力量斩飞了出去，倒飞数百里。
<br/>　　“好恐怖的防御！”苏莫见此眸光一亮，刚才那一剑虽然再斩灭了对方的拳劲之后，威力稍稍减弱，但依旧相当不凡。
<br/>　　别说是武帝境五重武者，就算是武帝境六重武者，都能一斩两半。
<br/>　　但这么强的剑气，居然破不开对方的防御，这真是相当惊人。
<br/>　　嗖！
<br/>　　被斩飞的滕金山又飞了回来，没受丝毫的伤势。
<br/>　　“苏莫，就算你的战力比我强，你也不可能打败我！”滕金山居高临下的望着苏莫，眸中带着强大的自信。
<br/>　　他心中虽然自信自己的防御，但亦是暗暗震惊苏莫的实力，这种战力让他都感觉恐怖。
<br/>　　“你确定？”苏莫淡淡的问道。
<br/>　　“当然！”滕金山自信道。
<br/>　　“好吧！”
<br/>　　苏莫点了点头，略一沉吟，道：“接下来一剑，你若是能挡住，我饶你不死！”
<br/>　　“什么？”滕金山闻听此言，顿时脸色难看了起来。
<br/>　　这说的是什么话？
<br/>　　你若是能挡住，我饶你不死？
<br/>　　我都挡住了，你想杀我也不可能啊！还饶我不死？
<br/>　　很多观战的人，本来正神情紧张，凝视着两人的战斗，听到苏莫的话语，都忍不住笑了起来！
<br/>　　均是被苏莫这矛盾的话语，给逗笑了！
<br/>　　但是，他们不知道的是，这对于苏莫来说，却并没有丝毫矛盾，即便对方挡住了他接下来的一剑，他想杀对方也是轻而易举。
<br/>　　少倾之后，苏莫身形缓缓拔高，冲天而起，向滕金山飞去。
<br/>　　少倾，他身形站定，与对方相隔百余里而立。
<br/>　　苏莫之所以站的距对方这么近，并不是要增加攻击的威力，而是要让对方无法躲避，只能硬挡他的攻击。

<br/>　　“准备好了吗？”苏莫淡笑着望着对方。

<br/>　　苏莫微微点了点头，手中长剑高高的举了起来，浑厚无比的玄力如同奔腾的长河，疯狂的涌入长剑之中。
<br/>　　十几座岛屿上，所有人都期待了起来，眼眸一眨不眨。
<br/>　　分胜负的时候到了，若是苏莫能破开滕金山的金刚之体，那基本上就能胜了。
<br/>　　相反，若是无法破开，估计就必败无疑了！
<br/>　　试问，连防御都破不开，还如何能胜？
<br/>　　木璃导师望着面色淡然的苏莫，心中期待的同时，亦是暗暗猜测，苏莫的底线在哪里？

<br/>　　但即便只是八种剑之战魂，依旧让得他身上的剑威强大到极致，仿若人形宝剑，锋芒无可匹敌。
<br/>　　“什么？”
<br/>　　这一刻，所有感受到苏莫身上剑威之人，无不面色剧变！
