　　苏莫的武魂晋升到天级九阶之后，将所在的山脉吞噬干净之后，便立刻离开了山脉中。
<br/>　　他没有再继续吞噬，而是向途山镇而去。
<br/>　　他准备回苍穹神宫看看，一则是将风少羽、君无惜等人送到苍穹神宫，二则是集苍穹神宫全宗之力，帮忙猎杀妖兽，获取兽魂，这样能快上不少。
<br/>　　而且，回苍穹神宫也浪费不了多长时间，他赶路的时间可以用来参悟奥义。
<br/>　　苏莫从神武学府出来，便一直是顺着去途山镇的方向，此刻距离途山镇已经只有两日的路程了。
<br/>　　盘膝坐在一柄本命灵剑之上，苏莫一边御剑赶路，一边参悟吞噬奥义。
<br/>　　他一刻的时间都不敢耽搁，争取在和冷江一战之前，突破到武帝之境。
<br/>　　两日时间匆匆而过，中午时分，苏莫到达了途山镇。
<br/>　　飞抵苍穹神宫的府邸上空，望着下方的府邸，苏莫面上露出了一丝笑容。
<br/>　　短短不到两个月的时间，苍穹神宫的府邸，居然比他离开的时候，扩大了三倍不止，井然成了一座小城。
<br/>　　而且，他离开之时，府邸非常的简陋，简直就像个贫民窟。
<br/>　　但现在，白玉小道，亭台水榭，恢弘宫殿，全部都建造了起来。
<br/>　　“不错！”苏莫对此非常的满意，随即他身形下落，直接落在了府邸中一座广场上。
<br/>　　“宫主回来了！”
<br/>　　“宫主回来了！”
<br/>　　“见过宫主！”
<br/>　　霎时之间，苍穹神宫沸腾了，一些看到苏莫的苍穹神宫弟子，纷纷疾呼出声。
<br/>　　瞬息之间，便有数百名苍穹神宫的弟子或长老，向他飞了过来，随即纷纷行礼。
<br/>　　“传我令，让所有武王境以上的人，全部来此地。”苏莫伫立在广场上，朗声吩咐道。
<br/>　　武王境以下的武者，只能猎杀低级凶兽，没什么用处，他要派武王境以上的人出去。
<br/>　　“是！”众人得令，不过，还不待众人去传令，苍穹神宫的众强者，便纷纷急速而至。
<br/>　　战乱、夕儿等九宫之主，帝释、海明江等老辈武者，也尽皆到来。
<br/>　　仅仅只是片刻之间，广场上便挤的水泄不通。
<br/>　　苏莫无奈，只好下令所有武王境以下的人全部离开，各自回去修炼。
<br/>　　面对苏莫的命令，众人不敢反抗，便纷纷离开了广场。
<br/>　　“宫主！”
<br/>　　“宫主！”
<br/>　　一众苍穹神宫的强者，纷纷向苏莫见礼。
<br/>　　苏莫微微颌首，眸光扫视众人，心中暗暗点头。
<br/>　　此刻留在广场上的人，足有两万余人，换言之修为超过武王的人，有两万余人。
<br/>　　而且，很多人的修为，比之在苍穹世界之时，提高了很多。
<br/>　　比如阴天仇，苏莫记得在离开苍穹世界之前，对方才武王境六重境界，现在已经武王境七重了。
<br/>　　还比如洛千帆，那时才半步武王境的修为，如今已经是武王境一重修为了。
<br/>　　这外界天地灵气浓郁，远超苍穹世界十倍，无论是修炼环境，还是资源等等，都比苍穹世界好太多了。
<br/>　　众人的实力，能快速提升，倒也在情理之中。
<br/>　　“诸位，这次回来，是有一个任务要交给你们！”苏莫环视众人，朗声说道。
<br/>　　“宫主何需这么客气，有什么事尽管吩咐！”
<br/>　　“对啊！请宫主吩咐！”
<br/>　　“宫主尽管说！”
<br/>　　众人闻言，不少人立刻开口，声音嘈杂一片。
<br/>　　宫主有事需要帮助，他们自然不会推脱。
<br/>　　苏莫点头，微微抬了抬手，示意众人安静，待现场安静下来之后，他朗声道：“所有人都离开途山镇，前去猎杀凶兽，抽取兽魂。”
<br/>　　“抽取兽魂？”众人闻言一怔，不过并没有太过意外，因为很多人都知道苏莫可以吞噬兽魂。
<br/>　　“抽取六级以上的凶兽兽魂，越多越好，十天之内，回来交给我！”苏莫朗声道。
<br/>　　还有半个月的时间，他武魂能否晋升到天级九阶，全看众人了。
<br/>　　当然，这段时间他也不会闲着，毕竟他直接吞噬的速度，要比其他人加起来都快。
<br/>　　“是！”众人应声。
<br/>　　这时，人群中的夕儿心中念头一转，笑道：“苏莫哥哥，现在我们苍穹神宫已经掌控了整个途山镇，以前的霸主途家，以及不少小家族，都已经臣服，我们也可以驱使他们。”
<br/>　　夕儿心中明白，苏莫突然回来让众人收集兽魂，必然是急着提升武魂的等级，所以她便想到了调集所有人力。
<br/>　　“是吗？那真是太好了！”苏莫闻言心中一喜，人越多自然能得到的兽魂便越多。
<br/>　　“夕儿，那这件事就交个你了！”苏莫对夕儿说道。
<br/>　　“苏莫哥哥放心，一定圆满完成任务！”夕儿娇俏的说道。
<br/>　　随后，苍穹神宫的一众强者，纷纷离开，前去猎杀凶兽。
<br/>　　当然，为免无人镇守宗门，苏莫让彩芸和青雅两人留了下来。
<br/>　　两人都是武皇境九重巅峰修为，实力接近武帝境强者，足以坐镇苍穹神宫了。
<br/>　　随后，苏莫又将风少羽和君无惜四人，从玉镯空间中放了出来，让彩芸和青雅安排一番，他便也离开了苍穹神宫。
<br/>　　苏莫离开苍穹神宫之后，飞行了整整十几个的时间，远离了途山镇亿万里之后，终于找到了一座大型的山脉。
<br/>　　那些距途山镇较近的山脉，自然有苍穹神宫之人去猎杀，所以他只能去较远的地方了。
<br/>　　进入山脉中，苏莫没有立刻吞噬妖兽，而是落在了一座小山上，盘膝而坐，双眸紧闭。
<br/>　　因为他的吞噬奥义，即将达到一阶了。
<br/>　　静静的盘膝在地，他微微催动吞噬武魂，感受那吞噬的真谛。
<br/>　　随着不断的感悟，苏莫对于吞噬的理解，越发的深刻。
<br/>　　又过了半日之后，苏莫突然睁开了眼眸，一股吞噬奥义从他身上绽放而出，席卷方圆十里。
<br/>　　“一阶了！”苏莫面上露出了一丝笑意，他的吞噬奥义达到了一个新的境界，威力大增。
<br/>　　不用阎魔鬼圣提醒，他也知道自己吞噬奥义达到了一阶。
<br/>　　一月之期到来之时，他突破到武帝境的希望很大。
<br/>　　只要突破到了武帝境，冷江应该不足为惧了！
<br/>　　或许，他的确该像阎魔鬼圣说的那般，高调一些，展现超强的天赋，让得学府为之重视，各大势力争相拉拢。
<br/>　　如此的话，说不定以后，就没人敢找麻烦了！
<br/>　　打败冷江，就是他在神武学府之中，快速崛起的开始！
<br/>　　随后，苏莫不敢耽搁时间，立刻开始在山脉中吞噬了起来。
