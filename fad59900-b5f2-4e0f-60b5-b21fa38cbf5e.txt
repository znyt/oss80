        傅夜沉气愤得质问了自己的母亲，为什么不相信自己的儿子。
<br/>
<br/>        “妈，我离开的这三天，我就已经在电话里告知过你。筱颖家出了点事，阿景没空陪筱颖回去，所以我帮阿景去处理一下筱颖的家事。您为什么不相信我？您不相信我，要验身，可以请医生来验我便是，为什么要去验筱颖？如果让阿景知道这事，你让他俩夫妻以后还怎么过日子？”傅夜沉铿锵有力地质问。
<br/>
<br/>        他真的没想到，自己的亲生母亲，会做出如此荒唐的事情。
<br/>
<br/>        而殊不知，这种荒唐事，是苏漫雪唆使冷夫人去做的。
<br/>
<br/>        苏漫雪得知傅夜沉为了童筱颖，冲着冷夫人发火了。立即跑了过来，在最恰好的时机出现，替冷夫人说话。
<br/>
<br/>        “夜沉，你要骂，就骂我好了！跟妈妈无关！是我让妈妈这么做的！妈妈也是心疼我！”苏漫雪叫冷夫人这声“妈妈”倒是叫得十分顺口。
<br/>
<br/>        傅夜沉见状，直接拽住苏漫雪的手，将她拉出了冷夫人的住处。
<br/>
<br/>        “从今以后，你回梅园住，不用再住偏院了！还有，如果你再敢给我惹是生非！别怪我对你不客气！”傅夜沉将苏漫雪甩到长廊上，大声呵斥道。
<br/>
<br/>        他凶完她，便径直离开了。
<br/>
<br/>        苏漫雪靠着长廊的边墙，缓缓地蹲下身去。
<br/>
<br/>        傅夜沉凭什么这么对待她啊？
<br/>
<br/>        就算她是假冒的傅家大少奶奶，但是她现在已经被傅老太爷和冷夫人认可了，假的也成了真的了，不是吗？
<br/>
<br/>        傅夜沉疾步回到了偏院，见主屋的门是敞开的，便直接走了进去。
<br/>
<br/>        童筱颖身上捂着被褥，蜷缩着身子坐在了床角，脸颊上满是泪痕。
<br/>
<br/>        其实，从流言蜚语里听到，她童筱颖还是如假包换的处子之身时，他心里竟然曾有过那么一丝欣喜。
<br/>
<br/>        但当他看到她满面泪痕，目光呆滞地坐在床角的时候，他的心口又开始犯痛了。
<br/>
<br/>        他的心，每次都是为了她而痛。
<br/>
<br/>        “筱颖……”傅夜沉在童筱颖的床边坐下，抬起手来想要给她拭去泪痕，可他这抬起来的手，又在半空中停了下来。
<br/>
<br/>        他想安慰她，却不知道自己该跟她说什么。
<br/>
<br/>        其实，此时此刻，童筱颖最需要的人是傅昼景。
<br/>
<br/>        傅夜沉已经命令大庄园里所有人都闭嘴，不许把这件事情传到傅昼景的耳朵里，以免给筱颖和阿景之间造成困扰。
<br/>
<br/>        “你已经耽误了三天学习时间，所以，不要再浪费时间在发呆上了。你不是想重新参加高考，考上一所好的大学吗？阿景他很优秀，你也得让自己变得更优秀起来，才能配得上阿景！”傅夜沉疾言厉色道。
<br/>
<br/>        童筱颖这才从悲伤忧郁中清醒过来。
<br/>
<br/>        “什么事都没发生过，所以，阿景也不会知道。你放心。”傅夜沉缓了口气，语重心长地说。
<br/>
<br/>        童筱颖这才自己抹去眼泪，眨了眨有些哭乏了的双眼，对傅夜沉说道：“那大哥，你先走吧！我想一个人安静地看书。”
<br/>
<br/>        她果然是因为在乎阿景，才忧心忡忡。
<br/>
<br/>        傅夜沉见她恢复了精神，心里也放了下来，起身离开了。
<br/>
<br/>        大哥说得对，她一定要考上一所好的大学，这样才能跟上昼景的步伐。
<br/>
<br/>        从她决定要重新参加高考的时候起，就是为了要让自己能配得上傅昼景。
<br/>
<br/>        童筱颖振作起来后，让苏漫雪都畏惧了。
<br/>
<br/>        因为，童筱颖在学习上的冲劲，慢慢地让苏漫雪自惭形秽。
<br/>
<br/>        与此同时，在这期间，冷夫人觉得自己有愧于童筱颖，便对童筱颖比以前更好了。
<br/>
<br/>        童筱颖心里也知道，冷夫人要面子，不肯跟她当面说对不起，所以，她也不强人所难，只要大家和睦相处，井水不犯河水，她也不会把彼此的关系闹僵。
<br/>
<br/>        一个月的时间里，傅昼景在季思妍的辅助下，完成了远东城军机部项目的初稿设计图，自己也赚了二十万的外快。
<br/>
<br/>        傅昼景本想分一半给季思妍，但是季思妍一分不要，还说她的就是他，无需分得如此清楚。
<br/>
<br/>        有了季思妍这样的贤内助，傅昼景确实在工作上要轻松许多。
<br/>
<br/>        不管是自己的设计，还是在自己的公司管理上，季思妍都给了他很大的帮助。
<br/>
<br/>        更重要的一点是，季思妍还给他介绍了许多客户，让他自己的新公司在今后的十年内都不缺业务做。
<br/>
<br/>        一个月前，从季思妍拿出一个亿跟他提出合作开始，第二天，他便行动了起来。
<br/>
<br/>        他收购了一家甲级资质的老建筑公司，现在交由季思妍打理。
<br/>
<br/>        如今，他打算在幕后秘密操作“远东城军机部”这个项目的全过程。
<br/>
<br/>        傅昼景的野心在季思妍的催化下，已经日渐膨胀。
<br/>
<br/>        他要把这个项目的代建方、审计方、监理方、施工方这四方队伍，全部掌控在他自己的手中。
<br/>
<br/>        他已经迈出了第一步，接下来要继续往下走第二步。
<br/>
<br/>        因为，他已经无法回头了。
<br/>
<br/>        白天，傅昼景坐在地利置业的总经理办公室里工作，晚上则到季思妍的公寓里去工作。
<br/>
<br/>        这一个月来，傅昼景每天晚上都和季思妍在一起，不管是一起吃饭，还是一起上床，他们之间的话题，除了“远东城军机部”的这个项目，还是“远东城军机部”这个项目。
<br/>
<br/>        每次不管多晚，傅昼景都会回自己的公寓去睡，从来不在季思妍那里过夜。
<br/>
<br/>        这晚，傅昼景一如既往地穿上衣服，拉起车钥匙离开了季思妍的公寓里。
<br/>
<br/>        季思妍拿起一根香烟，正闷闷不乐地抽着的时候，哥哥季运恒打了一通电话过来。
<br/>
<br/>        “小妹，傅昼景那边的进展如何？”季运恒直接开门见上地问。
<br/>
<br/>        一开始，“远东城军机部”的这个项目，其实是季运恒说起，并让季思妍去跟傅昼景谈合作的。
<br/>
<br/>        美人计、苦情计、挑拨离间计，只要是能用上的计策，季运恒让季思妍全用在傅昼景身上。
<br/>
<br/>        季思妍吐了一口嘴里的香烟，看着这白色烟雾，在眼前缭绕，她红唇微扬：“设计图是我和阿景的团队一起设计的，到时候，这个项目的造价标底肯定也是由傅氏集团地利置业的预算部出。我和阿景新收购的建筑公司，也够资历给这个项目投标。现在，我们在秘密计划围标的事情。这个项目，迟早是我们的囊中之物。”
<br/>
<br/>        季运恒听完季思妍的讲述后，得意地在电话那头笑了起来，并夸赞道：“不愧是我的妹妹，聪明能干！”
<br/>
<br/>        “哥，阿景可不是省油的灯！他不太好操控！虽然，他现在表面上是在跟我合作。但我知道，他暗地里也在防着我！”季思妍深沉又无奈地倾诉。
<br/>
<br/>        季运恒却不以为然地给季思妍出馊主意：“当初，我是推荐你去爬他大哥傅夜沉的床。你说他大哥不太好操控，还不如爬他傅昼景的床，让他成为我们的傀儡！现在你又说傅昼景不太好操控，他防着你，那是因为你跟他还不是一体的。你光爬他的床有什么用？你还要成为他的户口本上的合法妻子才行！小妹，你应该不会想一辈子只当傅昼景的情人吧！”
<br/>
<br/>        “我明白的，我会找合适的机会，解决掉那个叫‘筱颖’的女人！”季思妍一边说着，一边将烟蒂按灭在了床头柜的烟灰缸中。
<br/>
<br/>        同时，童筱颖也在各科家庭教师的补习辅导下，也已经做到了查漏补缺。
<br/>
<br/>        参加高考的这两天，傅老太爷可是派了专车接送童筱颖和苏漫雪，完全就是把她两当成是自己的两个亲孙女对待。
