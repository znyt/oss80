　　天劫来了，天劫就这样来了。
<br/>　　来得毫无征兆，来得出人意料，但却又在情理之中。
<br/>　　盘古顶天千万年，正是一身神力消耗到极限的时候，就算盘古修炼力之大道法则，一力破万法，一力破苍穹。
<br/>　　但，要让他时时刻刻保持着这样一个巨大力量输出的状态，一顶就是成千上万年，就算力量再大也会耗尽，更别说，盘古顶天之前还与三千魔神大战，斩尽三千魔神，还开出如此巨大的一个世界，拥有的力量已经达到骇人听闻的地步了。
<br/>　　到如今才神力枯竭，这已经不是第三步修士能做到的了，这已经是第四步不朽强者的手段了。盘古以半步不朽做到如此地步，现在出现力竭现象都已经不正常了。所以，此刻正是他最虚弱的时刻。
<br/>　　而，天劫，天道降下的大劫，不就是喜欢干乘虚而入的事情吗？盘古如今已是最虚弱的时候，他现在不来，又要到何时才来？
<br/>　　往前，盘古还不是最虚弱的状态，虽然度过天劫证道成功的机会不大，但不怕万一，就怕一万，机会终究是有的，天道也不敢以身试险，盘古的力道他可是一直很清楚，之前的战斗天道一直躲在虚空观察。他天道虽然堪比天道境修士，但终究挡不住盘古一斧子。
<br/>　　往后，天地已经越来越稳固了，要是盘古回过气来，修为定会大涨，自己更不是其对手，盘古度过天劫的机会就更大了。
<br/>　　所以，只有此刻出手，才是最佳时机！
<br/>　　只不过，这次的天劫，有些不一样。并不是想象中的斗转星移，也不是传说中的天降神雷化作神兵、神兽等对渡劫者进行人身或神魂、道心攻击，更不是天翻地覆，世界毁灭。而是一个人。
<br/>　　一个人，天劫是一个人？不错，就是一个人，不，也不对，现在人道未出，哪来的人？而是一个人形的神秘强者。而且，这个神秘强者盘古还认识，正是盘古还没有出世之前拜的神秘老师。
<br/>　　只见神秘人手持造化玉牒，趁着盘古力竭之际，从虚空突然闪现，径直朝着盘古的心脏处划来，速度之快，匪夷所思。
<br/>　　“鸿钧？！”盘古大怒，眼看自己就快要证道成功了，天劫居然在这个时候出现，而且还是以如此方式出现，最重要的是天劫居然以自己最熟悉的人来乘虚而入，偷袭他。
<br/>　　盘古大手一招，盘古开天斧就出现在盘古手中，盘古顺手霸道一斧，犀利的斧芒如同一道利剑斩在造化玉牒之上，造化玉牒顿时放出无量造化之光，想要抵挡巨斧的锋芒。
<br/>　　混沌第一攻击至宝对上混沌第一天道至宝，结果是，天道至宝造化玉牒破碎一角，神秘人直接被斧刃锋芒破灭，只留下一道元神附在剩下的造化玉牒上，没入虚空，消失不见。
<br/>　　原地一个巨汉头顶苍天，脚踏大地，手持开天斧，此刻，他仿佛明白了什么：
<br/>　　“鸿钧？天道？天选者？原来如此，这一切原来都是混沌天道的计谋。在本尊还没有出世之前，天道就在算计吾，天选者鸿钧就是你的棋子吗？你是在谋划我的天地，想吞噬吾之世界天道吗？哈哈哈！”
<br/>　　“先有鸿钧后有天？笑话！”
<br/>　　盘古在大笑，嘲笑自己的无知？还是嘲笑自己的单纯？还是嘲笑过去的种种？笑中还含着泪，热泪滚滚落下，但他还在大笑，哈哈哈！笑声传遍整个新生洪荒世界，传入混沌深处：
<br/>　　好大一会儿，笑声才停止，此刻，盘古猛然暴喝：“混沌天道！安敢欺吾！”
<br/>　　“盘古开天九式，本尊只创出八式，原先本尊估计这第八式是没有机会施展了，罢了，既然如此，那就——
<br/>　　八斧——斩——天——道！”
<br/>　　猛然一声暴喝，天地顿时出现一道覆盖整个天地巨大斧芒，这斧芒璀璨无比，浩大广阔无边，若是这个世界还存在生灵，那么，他们只要是看见一眼这斧芒，无需斧芒近身，就会立刻化作飞灰湮灭。
<br/>　　惊天、霸道、浩大、壮观……这些语言都无法形容这一斧的璀璨，但在这璀璨的背后，却是深入骨髓的冰冷杀机，这杀机令人灵魂颤栗。冰冷、幽暗、破灭、衰败……
<br/>　　巨大的惊天斧芒霸道的在虚空之中划破一道深深的裂痕，惊天斧芒冲入虚空，没入混沌之中，顿时天地寂静。而混沌却形成无量风暴。
<br/>　　不一会儿，天空仿佛出现漏洞一般，一道又一道金黄色光芒从漏洞中射出，瞬间，整个天地都被金光笼罩。金光之上，带着锐利的锋芒，射向四面八方，将大地打得残破不堪。
<br/>　　好大一会儿，天空才恢复寂静，但此时的天地，已经变得漏洞百出，残破不堪，地水火风重新涌动，但同时，天地也在扩大。
<br/>　　盘古望着这残破的天地，目光呆滞，不知在想些什么。此刻他已经度过天、地、人三劫，划破天道，道行又有了提升，不过，却并没有证得大道圣人不朽境界，因为他将自己证道的道基——洪荒世界给毁了，但就算如此，盘古真身却已度过灾劫，达到真正的第四步——不朽境。
<br/>　　……
<br/>　　而此刻，洪荒天地的一个神秘空间之中，无尽的规则与法则延伸至每一个角落，就像一张巨大的蜘蛛网。
<br/>　　这个神秘的空间有一个特殊的名字，它叫做天道空间，它是整个世界的核心！
<br/>　　原本这里应该是一条条如细线般的规则，密密麻麻遍布空间，一根根如锁链般的法则，一层又一层横穿整个空间的样子，但，如今却是细线般的规则与锁链般的法则断裂了无数。
<br/>　　空间中除了破碎的规则与法则外，还有漫天的紫气，无边的紫光，带着无尽道韵，无边无涯，好似道的化身。
<br/>　　它们是被盘古所斩的部分混沌天道！
<br/>　　而那法则与规则最中央，是一颗如同太阳般散发着灼热光芒的天道源球，球面上无尽光耀，它们在不断地吸收着空间之中的无尽紫气，壮大自身。
<br/>　　仔细看，就会发现那光球实际上是有五十条井井有序的紫色光线组成，一条条紫色光线交织在一起，汇成一个光芒四射的源球，这是天道本源！
<br/>　　其中有一条紫色光线最为特殊，因为它最为粗大，它身上散发的紫色光芒最为耀眼，而且，它吸收紫气的速度最快。最为特殊的是这条紫色光线竟然会动，不错，它会动。其他的紫色光线都是静止不动。
<br/>　　粗大光线时而若有若无，上蹿下跳，最为活泼。而，其余的紫色光线却冰冷无情。突然，活泼的紫色光线一下消失了，等它出现时，却已在光球的之外了。
<br/>　　此刻，它更仿佛脱了缰的野马，出了笼的小鸟，到处乱跑乱飞。它吸收紫气的速度更快了。不知过了多久，它不再吸收吞噬紫气。之后，它再次消失，却已出现在天道空间之外。“嗖”的一声，瞬间遁入虚空，消失不见。这一去，它就一去不复返！
<br/>　　原先五十道紫色光线，现在只剩下了四十九道，终究是残缺了一道。直到有一天，这些冰冷无情的紫色光线吞噬完天道空间中所有紫气后，变得更加强大了，同时也变成了真正的无情，或许直到有一天它再次诞生意识时，它才会变得不再无情，到时候，是好？还是坏？
<br/>　　而消失的那道光芒四射的紫色光线，却成为了别人的希望。
<br/>　　它们原先是兄弟，而后来……
<br/>　　它们都是天道！无情乎？有情乎？
<br/>　　天道五十，天衍四十九，遁去其一。
<br/>　　……
