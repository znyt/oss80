　　贞观十七年，也就是李承乾来到大唐的第十八个年头，上元大朝会如期举行。
<br/>　　百官朝拜，山呼万岁过后，老头子凝重的看着下面黑压压的一片人头，最后将目光定位在一个佝偻的身影上面：“魏征，你的身体好些了？”
<br/>　　“回陛下，臣……好些了！”魏征哆哆嗦嗦的站出来，似乎风一吹就倒摔倒的样子。
<br/>　　这老头儿已经病了好长时间了，往日里根本就不理朝政，可是今日却意外的出现，这让李世民着实有些摸不着头脑。
<br/>　　不过，世事无绝对，也许这老魏身体真的有些见好呢，或许也说不准吧。
<br/>　　听着老魏的解释，李二又见目光移向远处，看向大殿的外面，颇有一种物是人非，沧海桑田之感。
<br/>　　十七年前，自己初临帝位，整个唐帝国外有突厥犯边，内有蝗灾大水，可以说内忧外患，开个大朝会还要缩在东宫那个小地方开。
<br/>　　而现如今的大唐国富民强，已经是这片大陆上的霸主，一声号令天下莫敢不从的存在，开一次大朝会很多官员甚至需要提前两个月从自己的封地出发，才能按期赶到长安。
<br/>　　不一样，真的不样，或许也只有这样才配得上‘天可汗’的称号吧！
<br/>　　李世民的双眼微微眯着，享受着下方那无数忐忑的而又……。
<br/>　　又什么呢？李二觉得有些说不上来，不知怎么，他总是觉得这一次的大朝会显得有些诡异。
<br/>　　而就在这个时候，刚刚走出来的魏征颤抖着突然跪了下来，以额触地，口中言道：“陛下，老臣有一事，不吐不快。”
<br/>　　“讲来！”李世民深深看了一眼已经缩成一团的魏征，面色严肃了起来。
<br/>　　“陛下，自武德九年您登基以来，历时十年又七载，灭突厥、平辽东、镇吐蕃、破西域，功比三皇五帝。然，凡事不可尽，陛下虽‘武’‘功’盖世，但也要……。”
<br/>　　“魏征，你什么意思！给朕说清楚！”魏征的话才刚刚说了一半，老李同志已经听不下去了，面沉似水的他声音中带着无比的寒意。
<br/>　　不过，伏在地上的老魏却并没有任何犹豫，老李的话音刚落，但听到他的声音缓缓传来：“臣，想请陛下禅位于太子殿下！”
<br/>　　“嘭”的一声，只见李二狠狠一脚踢在桌上，人猛的站了起来，手指下面跪伏于地的魏征，须发皆张怒声吼道：“大胆魏征，胡言乱语，来人呐，将此逆贼拿下，推出去，斩！”
<br/>　　静，整个大殿上静的可怕，老李如此发怒，整个大殿上竟无一人说话，甚至连一个上来劝阻的人都没有，而且外面那些禁军也没有任何一人移动一下脚步。
<br/>　　面对如此情况，李世民明显意识到了什么，将目光投向站在一边的李承乾。
<br/>　　而就在此时，只听“轰隆”一声，整个朝堂上下内外，无数人突然跪倒尘埃，齐声言道：“请陛下禅位于太子殿下！”
<br/>　　卧槽，这是特么要造反啊！
<br/>　　李世民被眼前的一幕气的眼睛都红了，死死盯着站在一边，一脸无辜的李承乾：“太子，这便是你想要的？”
<br/>　　“没，没啊！不，不是我！”李承乾被老头子一问，瞬间回过神来，把头摇的飞快，心中却在暗暗叫苦。
<br/>　　老魏这到底是要闹哪样啊，老头子皇帝当的好好的，你就让他当呗，左右也不过就是个名头而已，像以前那样自己躲在幕后不好么？为毛非要让老头子退位，自己上前台呢？这和以前有什么区别！除了把老头子气的够呛之外，根本没有任何实质上的改变。
<br/>　　而且，就算是想要让老头子退位，你们和老子商量一下不行么？何必弄出这么大场面至于么！这让十分好百子的老头子还怎么下台，搞不好要出人命的好吧！
<br/>　　正想着，下面的魏征又开口了：“陛下，可否听老臣一言。”
<br/>　　“讲！”李二扭过头，瞪着下面一群人沉声说道。
<br/>　　“陛下，此事太子殿下的确不知道，是老臣这段时间借着同僚来探病的机会与诸位同僚沟通过的。”魏征先是替愣在一边的李承乾解释了一句，然后又缓缓说道：“陛下，这是老臣最后一次谏言，所谓人之将死，其言也善，还望陛下莫要辜负老臣。”
<br/>　　李二没有说话，就那么看着魏征，只是老魏根本就没有抬头，依旧用一种平静的声音说道：“陛下，我们都已经老了，已经不跟不上大唐的发展步伐，您多久没有看过奏折老臣不知，但老臣却知道，现在的下面报上来的一些东西都是老臣听所未听，闻所未闻的。”
<br/>　　“陛下，大唐已经是年轻人的大唐了，年轻人在成长，大唐也在长成，如果我们再控制着朝政，再在这个位置上继续待下去，只会压制他们，也会施慢大唐前进的脚步。”
<br/>　　“陛下……”
<br/>　　“够了！朕不要再听你的这些唠叨！”听着魏征上气不接下气，越来越弱的声音，李二也不知道自己此时的心情到底是什么样的。
<br/>　　痛心？是的，很痛，这个他称为人镜的臣子，竟然是第一个想要让他禅位的人，不得不说这真是一件最值得讽刺的事情。
<br/>　　恨？是的，也有恨，这大唐的江山是他一手打下的，现在让他供手让人，这让控制欲极强的老李如何受得了。
<br/>　　后悔？这个道没有，大唐皇帝做事从不后悔，哪怕是走到了今天这一步他也不后悔，只是有些遗憾罢了。
<br/>　　甚至在很多年以前，老李就已经预料到会有这样的一天，只是没想到这一天会来的这么快。
<br/>　　可是，这些感怀似乎都没有什么用处，理智告诉他，如果今天他坚决不让位，其实也没什么，边上那个臭小子到现在还是懵懵懂懂的，显然这件事情并不是出自他的手笔。
<br/>　　这也就是说，下面那些朝臣只不过是在试探，或者说在表达站队的方式，他们并不敢真的用武力逼着自己退位。
<br/>　　只是，事情到了这个地步，老李心中有一种继续在皇位上坐下去也没啥意思的感觉，手底下一个人都没有，这个位置不过就是别人施舍的罢了。
<br/>(本章完)
