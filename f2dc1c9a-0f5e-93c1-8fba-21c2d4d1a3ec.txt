        傅夜沉轻轻地挪开童筱颖环在自己脖子上的手，然后抓着她的手腕轻轻地放在了毛毯上。
<br/>
<br/>        他对她非常轻柔，不想扰了她的美梦。
<br/>
<br/>        “筱颖，晚安。”
<br/>
<br/>        我爱你……
<br/>
<br/>        傅夜沉刚想俯身亲吻童筱颖的眉心时，在半空中，还是忍住了。
<br/>
<br/>        他与她的身份悬殊，不能再对她有其它的非分之想。
<br/>
<br/>        这样的话，他已经不知道自己在心底告诫了自己多少遍。
<br/>
<br/>        双拳紧握，傅夜沉还是起身离开了。
<br/>
<br/>        回到自己的房间后，傅夜沉扯开领带，连卧室里的灯都懒得去开，直接脱下西装外套，随手将外套和领带扔在了门旁的长凳上，然后一边解着衬衣的纽扣，一边朝浴室走去。
<br/>
<br/>        洗完澡从浴室里出来后，傅夜沉正准备往大床那边走时，脚步蓦地停下了下来。
<br/>
<br/>        他拿着毛巾搓拭着头发的单手顿了顿，像是看到了什么令他讨厌的东西一样，剑眉微微蹙了起来。
<br/>
<br/>        看到自己的大床上躺着一个人，傅夜沉一脸不悦地紧锁眉头，目光冷冽地瞪着躺在自己大床上的人。
<br/>
<br/>        这人，还是个女人，与筱颖有着一模一样睡颜的女人。
<br/>
<br/>        几乎不用想，这房子里除了苏漫雪，还有谁有这个胆子敢睡他傅夜沉的大床？！
<br/>
<br/>        傅夜沉看清这女人是苏漫雪后，脸色瞬间冷到了极致。
<br/>
<br/>        大床上的苏漫雪像是故意在装睡般，慵懒地翻了个身。
<br/>
<br/>        松垮的浴袍里，她一丝不挂。
<br/>
<br/>        长腿肆意分开，好不矜持地摆出暧昧的姿势，两个大得像装了水的雪白“气球”，在空气里随着翻身的动作波动着。
<br/>
<br/>        苏漫雪还误以为傅夜沉看到了，哪知傅夜沉早已转过身去，打电话把刘婶叫了过来。
<br/>
<br/>        当苏漫雪悄悄睁开眼睛，眯成缝看到的人，便是刘婶一脸未睡醒，阴郁地瞪着大大的眼珠子，像是要把她吃掉。
<br/>
<br/>        刘婶没有丝毫犹豫，恼火地扯起床上装睡的苏漫雪，直接拽着往门外拖。
<br/>
<br/>        苏漫雪瞬间清醒，睁着大眼睛，故作受了惊吓般“啊啊”地大叫了起来。
<br/>
<br/>        刘婶刚将苏漫雪拽到门口，傅夜沉看都不看苏漫雪一眼，淡漠地对刘婶说道：“刘婶，帮我把床单和被褥都换了！”
<br/>
<br/>        他这是嫌她脏？！
<br/>
<br/>        苏漫雪顿时气得大呼小叫起来，把所有的怨气都发泄在了刘婶身上：“刘婶，你这是做什么啊？”
<br/>
<br/>        隔壁房间的房门“咔嚓”一声开了，瞬间打破了走廊上这三个人的聒噪。
<br/>
<br/>        童筱颖打着哈欠从屋内走出来，她其实是被自己定的手机闹钟给唤醒的，虽然不知道是谁把她弄回房间的，但是她得下楼去厨房里管自己炖的汤。
<br/>
<br/>        “你们这是……”
<br/>
<br/>        童筱颖睡眼惺忪地看着站在走廊上像是起了争执的刘婶和苏漫雪，一脸迷茫地问。
<br/>
<br/>        苏漫雪刚想把火气转移到童筱颖身上，嘴巴才张，一个字还未说，就被傅夜沉给抢先了一步。
<br/>
<br/>        “还不进来睡觉！”傅夜沉站在房门内，冷冷地呵斥。
<br/>
<br/>        童筱颖并未看到傅夜沉的人，但听他那声音，像是极其的不耐烦。
<br/>
<br/>        苏漫雪刚刚还是一脸狰狞，瞬间眉开眼笑，妖娆地撩了撩耳后的长发，睨了童筱颖一眼后，毫不客气地对刘婶命令道：“刘婶，我和夜沉刚刚把床单弄脏了。你去给我们拿一套干净的过来换上。”
<br/>
<br/>        “是！”刘婶颔首，没好气地应了声。
<br/>
<br/>        这前因后果，童筱颖并未弄清楚，但苏漫雪说什么床单脏了，童筱颖也大概听明白了是什么意思，只好不再多问，自顾自地下楼去了。
<br/>
<br/>        苏漫雪得意洋洋地回到傅夜沉的屋内，谁知傅夜沉突然抓起床单将她卷成了木乃伊，然后连人带床单一起推到屋内的一间独立的小隔间内。
<br/>
<br/>        “既然你喜欢没事找事，那么从今晚起，你就睡在这隔间里。”傅夜沉黑眸阴鹜地瞪着苏漫雪，随即毫不留情地将隔间的门给关上并反锁了。
<br/>
<br/>        苏漫雪愣了半晌才缓过神来，扭着身子，从床单里钻出来，匍匐在门板前，抬起双手，使劲地拍着门板：“夜沉，你这是什意思？快放我出去啊！”
<br/>
<br/>        房间的隔音效果很好，尽管苏漫雪很吵，但外边的人听不到。
<br/>
<br/>        刘婶给傅夜沉重新换好了床单和被褥，并未多嘴问半句，毕恭毕敬地欠身退下去了。
<br/>
<br/>        苏漫雪待着隔间里恼火不已，刚刚傅夜沉又是在做戏给童筱颖看。
<br/>
<br/>        他就那么怕童筱颖知道他的心意吗？
<br/>
<br/>        既然，他那么怕，那么她就去告诉童筱颖，说他傅夜沉对她有非分之想！
<br/>
<br/>        苏漫雪刚这么一想，转念间，又觉得不妥。
<br/>
<br/>        好不容易让童筱颖误以为傅夜沉爱的人是她苏漫雪，她若是再去跟童筱颖坦白这里那里，岂不是搬起石头来砸自己的脚？！
<br/>
<br/>        傅夜沉现在是骑虎难下，她苏漫雪都已经以“准大少奶奶”的身份见过傅家的长辈了，而且爷爷和未来婆婆都很喜欢她，那么她又在担心什么呢？
<br/>
<br/>        与其跟傅夜沉拼个你死我活，倒不如撮合童筱颖和傅昼景，让童筱颖早点给傅昼景生个孩子，彻底断了傅夜沉的念头！
<br/>
<br/>        苏漫雪这么一想，也就不再闹腾了。
<br/>
<br/>        翌日。
<br/>
<br/>        童筱颖把自己做好的“佛跳墙”端上了桌。
<br/>
<br/>        傅夜沉每天是这个家里起得最早的那个，他换了衣服从楼上下来，脚下步伐格外轻盈。
<br/>
<br/>        童筱颖正在专心致志地摆餐盘，所以，傅夜沉已经走到了她身后，她都未发现。
<br/>
<br/>        “这是什么？”
<br/>
<br/>        身后突然传来傅夜沉的声音，将童筱颖吓了一跳。
<br/>
<br/>        “大哥！”童筱颖手里拿着餐盘，挡着嘴巴，接着问候了一声，“早上好。”
<br/>
<br/>        “早上好，你做的是什么，给我盛一碗。”傅夜沉很自然地拉开椅子，在餐桌前坐了下来。
<br/>
<br/>        童筱颖连忙给傅夜沉盛了一碗汤，端到了傅夜沉的面前，回答道：“是连医生想吃的佛跳墙。”
<br/>
<br/>        “你会做这个？”傅夜沉抬眸很意外地瞥了童筱颖一眼。
<br/>
<br/>        童筱颖在傅夜沉的对面坐下，微笑道：“小时候见外婆做过，不过外婆给我做的是野食版的‘佛跳墙’，也就是一些野菜和野味一锅炖。我外婆做的那种，比我做的这个味道更香！”
<br/>
<br/>        “你小时候是你外婆带大的？”傅夜沉随口问道。
<br/>
<br/>        一提起外婆，童筱颖就有说不完的美好回忆，忍不住向傅夜沉滔滔不绝起来：“小时候，我跟着外婆生活在大山脚下，有农田，有果树，有菜园，我还养过小猫小狗和两只小羊。每次上山，外婆除了带着我一起识别药草以外，还会背上弓箭，经常就地猎取食材，天上飞的斑鸠，溪水里游的鱼，洼泥里藏的虾蟹，地上跑的野兔，草堆里长的野菜，腐叶下埋的菌菇，只要是能吃的，外婆都会带回去给我做一顿丰盛的饭菜。我烧柴看火，外婆下厨，香喷喷的饭菜出锅后，我和外婆就围着黑乎乎的灶台吃饭。那饭菜的味道是这世上独一无二的美味……”
<br/>
<br/>        那是她小时候最幸福难忘的时光，可是，童筱颖总觉得自己的记忆里好像还少了谁。
<br/>
<br/>        有人曾经在她最美好的时光里出现过，但那个人，她好像忘却了。
<br/>
<br/>        然而她说着说着，声音就没了。因为，她发现，大哥只是自顾自地吃早餐，似乎压根就没心思听她唠叨过去的事情。
<br/>
<br/>        突然间，周遭安静了。
<br/>
<br/>        “怎么不说呢？”傅夜沉抬眸看了童筱颖一眼。
<br/>
<br/>        童筱颖怔了怔，尴尬地咧嘴一笑：“我、我说完了。”
<br/>
<br/>        “很有意思。”傅夜沉淡淡地应了声。
