这就对了。”<br/>　　<br/>　　    就下来轮到穆广击球，他很老练的站在高尔夫球面前，轻轻挪挪脚步，然后猛地挥杆一击，高尔夫球划出一道漂亮的弧线，便落到了离球洞不足一尺的地方。<br/>　　<br/>　　    关莲扁了一下嘴，说：“你自己打得这么好，教我的时候却打得离球洞那么远，真是的。”<br/>　　<br/>　　    穆广笑了，说：“你别这个样子了，你才第一次打高尔夫，打成这个样子已经很不错了。你知道我是打过多少次才练出来的吗？”<br/>　　<br/>　　    钱总在一旁笑着说：“是啊，关小姐，你打的已经很不错了。你不能跟比，他打这个已经算是老手了。”<br/>　　<br/>　　    轮到钱总击球，他也是老手，技术娴熟，也姿态优雅把球击了出去。穆广看到钱总球的落点不如自己的好，笑着说：“老钱啊，你的球技还需要练练啊。”<br/>　　<br/>　　    钱总说：“，你不要一次比我大的好就这么骄傲，敢不敢跟我赌一把啊？”<br/>　　<br/>　　    穆广笑着说：“算了吧，就你这技术还想跟我赌？我都觉得胜之不武。”<br/>　　<br/>　　    钱总笑笑，说：“，你你这可是有点瞧不起人啊，怕是你不敢跟我赌吧？”<br/>　　<br/>　　    穆广说：“赌就赌，我怕你啊？”<br/>　　<br/>　　    三人就走下了了下一击球处，穆广再次手把手教了关莲击球，然后才把自己的球一杆进洞，击完球，穆广笑着看着钱总，说：“钱总看你的了。”<br/>　　<br/>　　    钱总笑笑说：“看就看，你以为我不行啊。说着钱总也很有架势的走到球前，挥杆击球。不过架势虽然很好，准头却偏了很多，短短的距离竟然击了三次才进洞。”<br/>　　<br/>　　    打完之后，钱总干笑了一下，说：“不行了，不行了，最近老是疏于练习，准头真是不行了。”<br/>　　<br/>　　    关莲在一旁笑着说：“钱总你这是故意让着吧？”<br/>　　<br/>　　    钱总说：“关小姐，你怎么这么说，我可是跟在赌着呢？你以为我不想赢啊，才不是呢。”<br/>　　<br/>　　    穆广笑笑说：“老钱啊，你这么就不地道了，愿赌服输，不要自己一输了，就找这样或者那样的理由。”<br/>　　<br/>　　    钱总说：“好好，打的就是好，我认输，行了吧。不过以前我的技术不是这儿差的。哦，我知道了，我知道原因在哪里了。”<br/>　　<br/>　　    穆广笑了，说：“在哪里啊？说出来听一听。”<br/>　　<br/>　　    钱总说：“原因还是我疏于练习了，去了海川之后，海川没有高尔夫球场，我找不到练习的地方，球技就生疏了。”<br/>　　<br/>　　    穆广笑着说：“你真会找理由，这么说我也没机会练习啊，还不是比你大的好？”<br/>　　<br/>　　    钱总说：“你跟我不一样，你可能天生运动技能就好一点，而我呢，必须要练习才行。不过话说到这里，，你觉没觉得海川没有高尔夫球场，这是一个很大的缺项。很多像我一样的老板，都很喜欢高尔夫球运动，有些时候他们考察投资地点的时候，没有高尔夫球场就会影响他们在这个地方的投资欲望。”<br/>　　<br/>　　    穆广看了看钱总，说：“老钱你这是什么意思啊？”<br/>　　<br/>　　    钱总说：“我在想，能不能在海川建一个高尔夫球场？”<br/>　　<br/>　　    穆广迟疑了一下，说：“这个嘛，好像不太行吧，国家有这方面的政策，严禁兴建高尔夫球场。”<br/>　　<br/>　　    钱总笑了笑说：“政策？我记得似乎有一句话叫上有政策，下有对策，是吧？”<br/>　　<br/>　　    穆广说：“可是，这顶风而上总不太好。”<br/>　　<br/>　　    钱总说：“我怎么感觉你的胆子变得小了起来，你在县里的时候不是这个样子的啊？”<br/>　　<br/>　　    穆广说：“我的胆子倒是没变小，只是这里有政策红线。”<br/>　　<br/>　　    钱总说：“中国的国情你还不明白吗？政策禁止了很多东西，可是很多东西不还是大行其道吗？你就说这个高尔夫球场吧，这些年全国各地不是建了很多吗？他们能建，为什么海川就不能建？关键是你没找对方法。”<br/>　　<br/>　　    穆广说：“老钱，这个情况我也知道，但是他们那是采用曲线救国的方式，他们报批的都不是高尔夫球场，而是什么体育公园、休闲俱乐部什么的。”<br/>　　<br/>　　    钱总说：“看来也知道这其中的猫腻啊，那我们为什么不也建体育公园或者休闲俱乐部之类的呢？”<br/>　　最快更新，无弹窗阅读请。<br/><br/><br/>
