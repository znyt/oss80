　　他可不是什么正人君子，随心所欲才是他的本性，有便宜不占更是虚伪，何况是自己的女人。
<br/>　　毕竟是斗皇强者，云韵御空飞行速度很快，仅仅片刻就落在一处山巅，瞪了一眼慕清霄，视线落在百米外的山峰处。
<br/>　　慕清霄恋恋不舍的离开柔软幽香的怀抱，顺着云韵的视线，也是落在远处的山峰上。
<br/>　　山峰上树木并不是很茂盛，半山腰上有一座洞穴，洞穴石壁上挂满了藤蔓，加上稀稀疏疏的树木，周围也堆积着一些碎石，显得极其隐蔽。
<br/>　　慕清霄神识悄然无息的覆盖出去，探查片刻后，道：“四周戒备有些森严，倒是难不住我，开始行动吧，你去吸引紫晶翼狮王，牵制住就行了，不可交战。”
<br/>　　闻言，芳心一暖，云韵知道他是在关心自己，道：“你自己也小心，记住我先前说的话。”
<br/>　　下一秒，云韵背后能量羽翼一振，离地而起，丝毫不隐藏身形，眨眼间就来到洞口，手持青色奇异长剑，二话不说就大开杀戒。
<br/>　　可见，这些天魔兽地毯式的搜寻，使她心中颇有怨气。
<br/>　　除了紫晶翼狮王，周围魔兽岂是云韵的对手，只见她化作一道青色光影冲入周围的森林，顿时一阵阵凄厉的兽吼声在山脉中回荡起来。
<br/>　　“人类女人，你竟然还敢出现，今日必取的性命，纳命来！”
<br/>　　云韵虐杀魔兽的时候，洞府深处，紫晶翼狮王暴怒的咆哮声，猛的传出洞外。
<br/>　　伴随着狂暴的咆哮声，一道紫色光影自洞内疾射而出，直接奔向青色光影，随后两道光影交战在一起，周围地势都因恐怖的力量改变了起来。
<br/>　　双方实力本就在伯仲之间，一时半会，云韵不会有危险，慕清霄隐匿气息，化作一抹剑光，以迅雷不及掩耳之势冲入半山腰的洞穴中。
<br/>　　山洞里并不昏暗，反而光亮十足，在周围的墙壁上，镶嵌着一排紫色水晶，这些晶块是山洞里的自然产物，若是拿出去，能够够卖出极高的价格。
<br/>　　只是，对于慕清霄来说，最多也就是一些装饰品，根本没有任何价值。
<br/>　　由于是紫晶翼狮王的住处，其中自然不可能有其他魔兽，哪怕是有，慕清霄也毫无顾忌，连脚步声都没有压制。
<br/>　　仅仅半分钟，他就来到洞口，周围的温度在不知不觉中提高了不少。
<br/>　　放眼望去，只见山洞内部的中央位置，是由紫晶石堆积而成的平台，其上摆放着一颗脑袋大小的紫色圆球。
<br/>　　而温度的提高，也是来源于紫色圆球。
<br/>　　慕清霄缓缓走到圆球旁，直接用蛮力，将紫色圆球自石台上扯了出来。
<br/>　　不重，可其中却蕴含着一股颇为庞大的能量，若是猜的不错，这圆球估计是‘伴生紫晶源。’
<br/>　　伴生紫晶源因为在紫晶翼狮王的肚内储存的时间比较长，所以其中蕴含的精纯能量颇为庞大，当小兽成长到四阶魔兽后，只要将它吞噬，就能直接成为五阶魔兽。
<br/>　　慕清霄嘴角牵起一丝笑容，在原著中，伴生紫晶源被萧炎拿了去，却没想到，现在落入了自己手中。
<br/>　　左思右想后，自储存空间中取出容器，将伴生紫晶源封放入其中收了起来。
<br/>　　他已经是斗皇强者，紫晶源对他来说没什么效果，至于其中的兽火，他也看不上，但无用归无用，却也不能白白便宜了别人。
<br/>　　至于给萧炎，纯属是扯淡，两人虽以师兄弟相称，却无感情瓜葛，若非药尘手中有自己想要的东西，他绝对不会与萧炎接触。
<br/>　　失去伴生紫晶源，对于萧炎会不会有影响，慕清霄从来就没想过，有影响如何，没有影响又如何？异火他都吞噬了，何况拿走这玩野。
<br/>　　他不是圣人，没有当别人保姆的义务，哪怕偶尔帮助一些也是看心情，在没有完全成为真正的强者前，在不触碰底线的情况下，顾好自己和老婆就行了。
<br/>　　唯有自身实力强大才有话语权，特别是在这种弱肉强食的世界中，否则，活着连条狗都不如。
<br/>　　至于冷血，慕清霄不屑一笑，谁想当圣人就由谁去当吧，反正他是不干。
<br/>　　伴生紫晶源不仅对魔兽，对实力不高的修炼者，也有着显著的效果，倒不如留着给自己的女人用。
<br/>　　将东西收好，慕清霄就开始寻找紫灵晶，很快通过另一条通道，来到最深处的洞穴中。
<br/>　　山洞不是很大，且几乎被紫晶石占满，光线明亮堪比洞外，慕清霄的视线跃过紫晶石，直接落在中央位置，此处有一只幼小的紫晶翼狮王匍匐在其上。
<br/>　　此时，云韵给他的感应水晶也派上了用场，慕清霄也不隐匿身形，直接向中央位置走去。
<br/>　　小紫晶翼狮王睁开双眼，瞳孔中呆泄一秒后，猛地站起身来，狰狞的嘴中传出一道低沉的咆哮声，二话不说就冲慕清霄扑了过来。
<br/>　　慕清霄径直走向中央，看都不看它一眼，一鞭腿甩出，宛若秋风扫落叶。
<br/>　　“砰！”
<br/>　　小紫晶翼狮王犹如野狗，连反应的机会都没有，瞬间被恐怖的力量砸飞出去，下一秒，猛的撞在镶满紫晶石的石壁上，坚硬的紫晶石壁面都被砸出蜘蛛网一样密密麻麻的裂痕。
<br/>　　“蓬！”
<br/>　　待小紫晶翼狮王掉落在地上，呜咽了几声，已经是出气多进气少。
<br/>　　见慕清霄投来目光，瞳孔中尽是恐惧，连呜咽声也消失，躺在地上瑟瑟发抖，生怕慕清霄当场就宰了它。
<br/>　　不过，慕清霄可没有嗜杀的兴趣，也并不想杀了它，毕竟他是有底线的，他们本来就理亏在先，抢劫别人的家当，若是真将孩子都杀了，就有些太过分了。

<br/>　　来到小紫晶翼狮王睡觉的石台，慕清霄眼里何等惊人，一眼就看出了石台上的猫腻，灵气一卷，石台中央就打开一个暗格，其中正放着一颗无规则的尖锐紫色灵石。
