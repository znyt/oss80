　　“额...”萧逸忽然醒了过来，身体有些僵硬和疼痛。
<br/>　　一睁开眼，看到大长老、二长老、三长老和四六长老。
<br/>　　“这是哪里？”萧逸迷糊地问道。
<br/>　　大长老微笑着说道，“萧家。”
<br/>　　三长老满脸笑容，“逸儿你终于醒了。”
<br/>　　萧逸满脸惊疑，“我们怎么回萧家了？慕容家的人呢？”
<br/>　　大长老道，“慕容山以及一众慕容家长老已死，在你昏迷的这段时间里，我们把整个慕容家都灭了，萧家危机已除，你可以放心。”
<br/>　　“都解决了吗？”萧逸笑笑，松了口气，问道，“我昏迷多久了？”
<br/>　　“整整七天七夜。”一旁的二长老轻笑道，“你小子，当时抱你回来的时候，你冷得跟块冰坨似的。吓得我们几个轮流用真气给你温养身体，可算把你盼醒了。”
<br/>　　大长老沉声问道，“我们与慕容家的战斗结束后，立刻去找你。却看到慕容墨的尸体，你则重伤昏迷，到底怎么回事？”
<br/>　　“咳咳。”萧逸虚弱地咳嗽了两声，身体的伤势仍旧没有复原。
<br/>　　三长老关心道，“逸儿你没事吧？”
<br/>　　大长老也是一惊，连忙要给萧逸把脉。
<br/>　　萧逸虚弱地摇摇头，道，“我没事，体内的经脉受创伤，但已无大碍休养一番即可。”
<br/>　　“那就好。”大长老道，“你自己也是炼药师，你自己的伤势如何你比任何人都清楚，倒不用我们去为你担忧。”
<br/>　　“哼。”三长老面露愤怒之色，“该死的慕容家，要不是这群小人，逸儿你岂会伤得这么严重。可惜，没能将他们全都杀掉。”
<br/>　　萧逸疑惑道，“大长老不是说把整个慕容家都灭了吗？”
<br/>　　大长老解释道，“我们当初回到萧家后，个个都带伤，稍微休养了一番才去的慕容家。慕容家的人狡猾得很，有几个子弟事先逃掉了。”
<br/>　　三长老愤愤地说道，“当我们去到慕容家时，慕容千军、慕容娇儿和慕容轩已经不在了。”
<br/>　　“不用管他们了。”萧逸摇摇头，“他们几人成不了气候，对萧家没威胁。”
<br/>　　“萧逸。”大长老忽然叫了一声，而后沉默起来。
<br/>　　“嗯？”萧逸皱起了眉头，他稍稍恢复了精神，敏锐地捕捉到三位长老脸上浓浓的忧愁之色。
<br/>　　如今萧家危机已除，三位长老却仍旧脸色凝重，事情必然不简单。
<br/>　　“怎么了？”萧逸沉声问道。
<br/>　　三位长老对视了一眼，而后互相点点头。
<br/>　　大长老出言道，“有两个坏消息要告诉你。”
<br/>　　萧逸点点头，意识绝非等闲之事，打起了精神，认真道，“大长老请说。”
<br/>　　“第一个坏消息。”大长老道，“这几天，我们留了几个慕容家执事的性命，并审问了一番。毕竟慕容墨这些年到底去了哪里？又为何一回紫云城就迫不及待地对付我们萧家，这些事我们需要清楚。”
<br/>　　萧逸认真地听着，其实他之前对此事也心存很多疑惑。但那时时间紧迫，便没有多想。
<br/>　　大长老认真地看着萧逸，语气很是凝重，“慕容墨此番回来，是要称霸紫云城。然而，他称霸的原因，又和紫云城内的一切都无关。”
<br/>　　“紫云铁矿。”不待萧逸发问，大长老率先说道，“他的目的，仅仅是获得陨星山脉中的所有紫云铁矿。而且，之前慕容山带人袭击萧家府邸时，他也并不是在闭关，而是不方便出手。”
<br/>　　“若不是我们杀了他们慕容家的长老，恐怕那老家伙会一直都不出现的。然而，他一旦出现了，更不惜一切追杀我们，为的是灭口。”
<br/>　　萧逸眉头皱得更紧，问道，“到底怎么回事？”
<br/>　　大长老整理了一下脑海中的思绪，没有回答萧逸，而是直视萧逸，问道，“你可知道北山慕容家？”
<br/>　　“北山慕容家？”萧逸面露疑惑，摇了摇头。
<br/>　　大长老点点头，道，“你没有到过紫云城外的地方历练，不知道也是正常。北山慕容家，在整个北山郡，几乎没人不知晓，可谓声名赫赫。”
<br/>　　“那你可知盛宝商行？”大长老继续问道。
<br/>　　“知道。”萧逸点点头，“这乃是我们北山郡最大的商行。”
<br/>　　盛宝商行，售卖武者修炼的各种物品，天材地宝、奇珍异宝、丹药、材料、武器等等，应有尽有。
<br/>　　紫云城中便有一间，萧逸曾去那里买过炼丹材料。
<br/>　　大长老点点头，道，“不错，北山郡一百五十城，每一城皆有盛宝商行的分行，它可谓我们北山郡最强大的商行。”
<br/>　　萧家所在的国家，名为炎武王国，是一个地域极为辽阔的王国。
<br/>　　炎武王国，共三十六郡，城池无数。
<br/>　　北山郡便是其中一郡，拥有一百五十城。
<br/>　　紫云城，便是北山郡中的一城。
<br/>　　紫云城已经很大了，但也不过是北山郡的沧海一粟。而北山郡又只是炎武王国的一郡。
<br/>　　可想而知，北山郡非常辽阔；而炎武王国，又庞大到何种程度。
<br/>　　言归正传。
<br/>　　大长老继续道，“我们北山郡最大的商行，盛宝商行，便是属于北山慕容家。”
<br/>　　萧逸点点头，这倒没什么好意外的，世界那么大，总有些强大的家族，其底蕴和势力都大得让人发指。
<br/>　　但忽然，萧逸想到了一个可怕的念头。
<br/>　　惊疑道，“大长老，你可别告诉我，慕容家和这个北山慕容家有关系。”
<br/>　　大长老点点头，道，“是有关系，但关系不大。慕容家，是他们很多年前的一个分支。不过，已经隔了许多年，彼此早就没有任何联系了。”
<br/>　　“那慕容墨，当年悄然在紫云城消失，便是回到了北山慕容家，还当上了外门执事。”
<br/>　　外门执事，处于家族并不太重要的地位。家族通常会以这种手段，吸引外族的武者为家族效命。
<br/>　　外门执事的地位，比不上执事，甚至比不上一个家族的出色子弟，仅仅比普通子弟的地位高一些。
<br/>　　大长老继续道，“慕容墨此番回来，就是想得到所有紫云铁矿，以上贡北山慕容家，希望提升自己的地位。”
<br/>　　“毕竟紫云铁是需求极大的材料，北山慕容家又是个商业家族，他们很需要这些商物。”
<br/>　　“然而，北山慕容家是商业家族，尤重声誉和名声。慕容墨这个外门执事做出这等为了提升自己地位，而不择手段，为了一己私利杀害其他家族，甚至为害一方之事。”
<br/>　　“这种事传了出去，坏了北山慕容家的名声，导致商业受损，慕容墨会受到家族的严惩。”
<br/>　　“这就是他之前不轻易出手，而一出手就要灭口的原因。他想神不知鬼不觉地得到所有矿脉。”
<br/>　　“我懂了。”萧逸听罢大长老的话，瞬间明白过来。
<br/>　　当然，他也明白三位长老如今脸色凝重的原因了，道，“你们是怕北山慕容家会找我们的麻烦。”
<br/>　　虽然北山慕容家尤重名声，而慕容墨又只是个无关紧要的外门执事。
<br/>　　但，慕容墨始终是他们的人。所谓树要皮人要脸，自家势力的人被外面的人杀了，自家势力不可能无动于衷，这是打脸的行为。
<br/>　　自从听到北山慕容家五个字后，萧逸虽脸色凝重，但仍旧沉着应对；大长老则在理性分析，二三长老沉思着。
<br/>　　只有四长老和六长老，满脸的担忧和踌躇。
<br/>　　此刻听到萧逸的话，更是差点没吓得跳起来。
<br/>　　“天阿。”四长老惊声道，“北山慕容家要是来找我们麻烦，我们萧家会被灭得渣都不剩吧。”
<br/>　　“不会。”萧逸和大长老同时摇摇头。
<br/>　　“聪明。”大长老满脸赞赏之色地看着萧逸，笑道，“你来说。”
<br/>　　萧逸点点头，道，“在北山慕容家那等庞然大物眼中，我们萧家不过是弱小的蝼蚁，一只手指便能捏死，随便派个家族强者都能灭了我们。”
<br/>　　“但，这次毕竟错在慕容墨，是慕容墨挑起的纷争。若是他们因此而灭我们萧家，只会影响他们的声誉。”
<br/>　　“虽然，被影响的声誉可能只有一丁点儿，甚至微不足道。但，对于他们来说，也比我们整个萧家重要百倍。”
<br/>　　萧逸面露自嘲之意，道，“简单来说，他们不把我们放在眼中。”
<br/>　　“我们整个萧家的人命，抵不上他们北山慕容家的一丁点儿声誉，他们不值得这样做。”
<br/>　　虽然很讽刺，但这是事实。
<br/>　　对于这个弱肉强食的残酷世界来说，人命在某些大势力眼中，确实一文不值。
<br/>　　萧家，在紫云城算三大家族之一，但放在整个北山郡，只是不入流的小家族。
<br/>　　而北山慕容家，是北山郡最强大的家族之一，就算在整个炎武王国，也是声明不小。
<br/>　　两者根本不是一个级别的。
<br/>　　大长老笑道，“萧逸，有些时候我真想不明白，你的心智为何会如此老道，竟丝毫不逊色于我这个活了大半辈子老人的人生阅历。”
<br/>　　“但。”大长老忽然脸色一变，道，“你看漏了一点。”
<br/>　　大长老沉声道，“北山慕容家，虽然不会灭我们，但肯定会找我们的麻烦。而且，那慕容墨如果我没有猜错的话，肯定是不知出于什么原因，死在你手上的。”
<br/>　　“北山慕容家，必然会杀你。一命抵一命，方不会损他们家族的脸面。”
<br/>　　“什么？”三长老顿时大急，道，“逸儿不过是个后天境，说他能杀了慕容墨，根本没人信吧。”
<br/>　　大长老摇摇头，道，“信也好，不信也罢。萧逸作为萧家第一天才，如今的少家主，未来的萧家家主。北山慕容家肯定会把气撒到他头上，要他的命。”
<br/>　　“所以，萧逸你必须马上离开紫云城。”大长老说道。
<br/>　　三长老叹息了一声，“以北山慕容家的势力，若真的要杀一个人，天大地大，逸儿能逃到哪里去？”
<br/>　　大长老当即笑笑，对萧逸道，“去裂天剑派，我们北山郡的第一门派。”
