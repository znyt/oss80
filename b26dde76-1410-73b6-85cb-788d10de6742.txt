　　脸上带着古怪的表情，刘哔从空间戒指中取出一颗通体灰色珠子，抛向半空。
<br/>　　瞬间，灰芒大盛，珠子在半空中碎裂开，一股惨烈无比的尸气从中散发了出来。
<br/>　　这是他在第一个位面的僵尸村里获得的战利品，在此之前，一直都不知道有什么作用。
<br/>　　而就在刚才，保安小哥掏出佛珠来的时候，他才感应到了空间戒中这颗珠子的异动。
<br/>　　为了搞明白它的作用，刘哔往里注入了一些尸气，随后出现的就是这般场景。
<br/>　　佛珠已经压下，但灰色珠子依旧没有什么动静，刘哔眉头皱起，猛地抬起手中的火焰棍棒。
<br/>　　“给我停住！”
<br/>　　火焰棍棒在空中变大数倍，架在了佛珠和灰色珠子中间，金灰两色交替，照亮了刘哔扭曲的面容。
<br/>　　两人之间的实力相差不是很大，这串佛珠，似乎也不是凡品，互相对攻之中，谁都没能占到一丝便宜。
<br/>　　“嗷！”
<br/>　　突兀，一个震天的吼叫声响起，灰色珠子中，尸气已经凝练到了极致，随后一个庞大的黑影冲了出来，拦腰将挡在正前方的刘哔撞飞。
<br/>　　速度太快了，刘哔根本无法反应，只来得及看到一个模糊的影子，紧接着一股大力袭来。
<br/>　　“噗！”
<br/>　　吐出一口黑色的血液，刘哔撞穿了墙壁，从二十层高的大楼上掉了下去。
<br/>　　庞大的力道，让他的意识都有些模糊，腰部传来了一股微微的痛感，刘哔向下看去，腰部被撞出了一个缺口，一些发黑的内脏，顺着伤口掉了出来。
<br/>　　察觉到自己现在的状态不对，刘哔调集体内散乱的能量，身后张开了一对黑色的羽翼。
<br/>　　在空中不稳的摇晃了半天，总算是勉强稳住了身形，刘哔咬牙，顺着大楼飞了上去。
<br/>　　上方好像正在战斗，不时有剧烈的轰击声传来，那个灰色珠子中出来的东西，应该是正在和保安进行交战。
<br/>　　话说这到底是个什么玩意，他还没搞清楚状况，就突然受到攻击。
<br/>　　如果不是自己有一对翅膀，从二十楼的高度掉下去，还受了这么严重的伤势，说不准都要丢掉性命。
<br/>　　……
<br/>　　“铿！”
<br/>　　手中的佛珠砸在敌人身上，只是瞬间就被弹飞了出去，在那头生物面前，这种程度的攻击，甚至只是在那如同艺术品一般剔透的骨头上留下了一道白痕。
<br/>　　他都不敢想象，有一天自己会面对这样的怪物。
<br/>　　亡灵生物他知道，尸兄也勉强和这种生物沾点关系，但是……TMD骨龙是什么鬼？
<br/>　　正在对他发动攻击的，赫然是一条骨龙，这种不可思议的生物，身躯几乎有三米多长，浑身都是由玉石一般的白骨拼凑而成，眼眶中，还带着两团绿油油的火焰。
<br/>　　没人会小看这种生物，尤其是这条骨龙还会吐息的情况下。
<br/>　　“哗啦啦~”
<br/>　　随着骨龙的动作，周围的墙壁如同豆腐渣一般塌陷，这样战斗下去，他迟早被活埋在这栋大楼里。
<br/>　　但现在根本不是想这些的时候，因为……他根本无法在骨龙手中支撑下去！
<br/>　　窗外，飞上来一个身影，惊鸿一瞥之间，他看清了刘哔的面孔。
<br/>　　这个家伙，正面承受骨龙的冲撞，还从二十搂上掉了下去，都已经这样了，居然还能活？！
<br/>　　当注意到刘哔身后的一对黑色翅膀时，他也就有些理解了。
<br/>　　“砰！”
<br/>　　在骨龙的攻击下，保安小哥也只有躲闪的份，但是在大楼这种环境下，他的行动受到了很大的限制。
<br/>　　虽然这些墙壁对骨龙来说如同豆腐一般，一触即碎，但对他来讲，却成了不可逾越的阻拦。
<br/>　　这些建筑，会拖慢他的速度，但骨龙明显不会给他这样的机会。
<br/>　　躲闪了十余秒的时间，保安小哥终究出现了失误，骨龙硕大的利爪，堪堪蹭到了他的后背。
<br/>　　瞬间，整个后背都变得血肉模糊，被这样一阻挡，后面紧跟着的吐息攻击，已经是躲不过去了。
<br/>　　没有任何反抗的余地，幽黑的吐息过后，保安已经彻底消失在了原地，在这样的攻击下，甚至连尸骨都没能留下。
<br/>　　刘哔不免觉得有些可惜，早知道的话，不将这头骨龙放出来了，如果只有保安，他完全可以应对。
<br/>　　这可是一个和自己层次差不多的能量体，不说积分，就算能留下尸骨，恐怕他都会有很大的收获。
<br/>　　可现在，捡漏的思想被彻底打消了。
<br/>　　“咯吱~”
<br/>　　解决完对手之后，骨龙转过头来，两团碧绿的鬼火紧紧盯着刘哔，附近空气中的温度，似乎都降低了下来。
<br/>　　这一眼，让他遍体生寒，现在已经不是能不能捡漏的问题了，现在最大的问题，是怎么从这家伙手上逃出去。
<br/>　　即使是他放出来的，但这头骨龙的态度可不怎么友好，就在刚才，还差点杀掉了他。
<br/>　　幸亏装偏了，要是头颅，还不得交代在这里。
<br/>　　“吼！”
<br/>　　出乎意料的，骨龙收起了翅膀，随意的趴在地上，眼睛里的两颗鬼火，竟逐渐开始暗淡。
<br/>　　精通所有种族的语言，这个刘哔一直续费的项目，此时却派上了极大的用场。
<br/>　　“你身上没有主人的味道。”骨龙的声音无比虚弱，眼中的鬼火抖动的更加厉害了：“但是你怎么会有主人的骸骨珠？”
<br/>　　硬生生止下了想要逃走的步伐，刘哔转过身来，和骨龙对视着，头皮有些发麻。
<br/>　　“我不知道，这颗珠子，是我家传的，我一直都有带在身上。”
<br/>　　被那两团鬼火盯得心中发虚，碧绿的光芒，如同实质一般照在刘哔身上，像是要看穿他的灵魂一般。
<br/>　　但他还是硬着头皮，找了个借口。
<br/>　　无法想象，被骨龙称之为主人的存在，得有多么强大，但至少沾点因果，不是坏事。
<br/>　　万一从中得到什么好处了呢，人总是要登高望远的嘛。
<br/>　　“世界要毁灭了吗？”骨龙似乎并没有在意刘哔的话语，遥望着远方，低喃着：“这是过了多久啊，竟然都没有了神灵法则的约束。”
<br/>　　骨龙的话语引起了刘哔心中的兴趣，犹豫了一下，他开口问道：“法则？神灵？您能告诉我一点，和这些有关的事情吗？”
<br/>　　“唔……”
<br/>　　从沉思中被打断，但骨龙没有任何气恼，反而回答了刘哔的问题：“法则……你们天庭，大概都叫这种东西为天条吧。”
