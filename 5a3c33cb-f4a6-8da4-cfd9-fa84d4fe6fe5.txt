　　四宝怔了一下，这才抬头看了叶少阳那边一眼，似乎觉得这是一件很没理由的事，说道：“这有什么好担心的？少阳既然不让我们帮忙，肯定不会输。”
<br/>　　“可是他只是一名真人啊。”
<br/>　　“谁说的？”四宝用筷子点着叶少阳，“你看着像吗？”
<br/>　　的确……不像。
<br/>　　妙心还想问问他，四宝又继续吃上了。他不关心战局，之前看到叶少阳败退，上去帮忙，也是以为叶少阳是中了什么手段发挥不出实力，后来叶少阳赶他们到一边去，四宝看他没事，就彻底放心了。
<br/>　　他之前就关注过，叶少阳的对手，是个人类。既然是人类，按照他的逻辑，那就不可能是叶少阳的对手。叶少阳在一百年后的人间法术界，已然无敌，在这个时代，自然也是。
<br/>　　战局瞬变，起初陈晓宇还能应付，很快就手忙脚乱起来，好几次差点就要顶不住，再又一次被击退之后，陈晓宇一咬牙，从腰间接下了一把拂尘，在面前刷出了不定符。
<br/>　　众人一片哗然。
<br/>　　本来两个人都是空手斗法，陈晓宇率先拿出法器，说明已经被叶少逼的没办法了。这是在场的大部分人都没有预料到的结果……这个叶少阳，怎么好像脱胎换骨一般？
<br/>　　更让他们吃惊的是，在陈晓宇亮出拂尘之后，叶少阳依然没有取出任何法器，两手空空地冲了过，单手结印，凌空画符，一股水波形态的灵力，朝陈晓宇布置出的结界冲过去。
<br/>　　“疯了！”正在观战的众人都感到十分不解，这种水形态的法术，以柔克刚，只要时间足够，能渗透灵力超过自身少许的结界，这个是没错，但是现在的局面是，叶少阳完全占据了上风，陈晓宇正是因为防御的节奏被打乱，这才用拂尘画出不定符，想要挡上一会，自己调整一下气息和节奏，之前积累下来的劣势就可以一扫而光，重新作战。
<br/>　　叶少阳这个应该强攻，争取快速打碎结界，这个时候却用水系的法术，简直就是帮对方争取时间……
<br/>　　“还是太嫩了。”云春生摇了摇头。在他看来，叶少阳之所以这么做，是想要赢的漂亮一点，年轻人，总是这样年轻气盛，说的不好听点是得意忘形，为了面子问题，居然将好不容易收获的优势都放弃了。
<br/>　　陈晓宇看到叶少阳施展的法术后，心中也是暗喜，正想说点什么来激怒他，突然，叶少阳手中那一道水汽，一分为二，一明一暗，上下旋转，撞击在了自己布置出的结界上。
<br/>　　“太乙生水！”
<br/>　　伏明子惊叫出声，随即问云春生，“师伯，北宗有这法术？”
<br/>　　这是茅山南宗的内门法术，一向被认为是南宗的祖师自创的，如今却被叶少阳施展出来，让这师徒二人实在有点震惊。
<br/>　　“这……他既然施展出来，那自然是北宗的法术了。”云春生用了逆向推理，很轻松地得出这个结论。
<br/>　　太乙生水的两道水波，瞬间震碎了陈晓宇布置的结界，叶少阳贴身而上，陈晓宇急忙双手捏决，推了出去。
<br/>　　叶少阳一只手捏诀，拍了过去。
<br/>　　“啪！”
<br/>　　两股灵力相撞，产生出一股强大的冲击，叶少阳身上的衣服晃了晃，人影纹丝不动，陈晓宇却是倒飞出去，摔在地上，立刻又爬起来，但是一只脚在他后背上重重地来了一下，将他踏在地上，惊慌之中转头，看到叶少阳微笑的一张脸。
<br/>　　“叶少阳，你……”
<br/>　　一脚踹在他脸上，将他踢翻过去，陈晓宇还想挣扎，叶少阳一只脚踏在了他脖子上，将他半张脸踩进了泥土里。
<br/>　　结束了。
<br/>　　陈晓宇完败，并且被叶少阳踩在脚下。
<br/>　　这个结果，让在场的所有人都出乎意料，一时间，所有人都像被施了定身术一样，怔怔地看着叶少阳和他脚下的陈晓宇。
<br/>　　梧桐再次掀起了盖头，偷偷看见这一幕，惊的捂住了嘴。
<br/>　　只有四宝和吴嘉伟，还在继续吃着刚上桌的阳春面……
<br/>　　“我不会对你磕头认错的，就算你杀了我！”陈晓宇羞愤至极，无奈被踩在脚下，一动不能动，张红着一张脸，死死地瞪着叶少阳。
<br/>　　他万万没有想到会是这么一个结局。万万没想到，自己居然不是叶少阳的对手。
<br/>　　叶少阳抬起了脚，送开他，目光从呆滞的众人脸上扫过，说道：“法术公会？不过如此！”
<br/>　　全场哗然。
<br/>　　“叶少阳！”张晓寒大步走来，冷冷道，“不过是赢了我一个师弟，有什么资格叫嚣？”
<br/>　　叶少阳平视着他，说道：“正有此意。”
<br/>　　全场再次哗然。
<br/>　　“叶少阳你疯了！”妙心起身叫道。
<br/>　　“好啊，来吧！”张晓寒往前迈了半步，做出了战斗的姿态。
<br/>　　“等会，之前的约定，还没有兑现吧？”叶少阳转头看了一眼，陈晓宇刚从地上爬起来，浑身都是土，披头散发的，看上去极为狼狈，听见叶少阳这句话，立刻就愣住了。
<br/>　　之前的约定……
<br/>　　自己那么信誓旦旦地提起约定，让在场的人都作个见证，本来是怕叶少阳耍赖，谁想到竟然坑了自己……陈晓宇满脸涨红，嘴唇哆嗦着，一个字说不出来。
<br/>　　叶少阳目光找到鬼姨，问道：“你是管事的吧，之前的约定，你也听到了，你们法术公会的弟子，不会说话不算吧？”
<br/>　　鬼姨脸色极为难看，却是一个字也说不出，毁约？这是万万不能的，传出去，法术公会出尔反尔，实在丢人，但是让陈晓宇给他磕头认错吧，陈晓宇的前途就毁了，将来肯定没有脸在法术界混下去，而且当着这么多客人的面，法术公会的面子实在也没地方放。
<br/>　　因此就算是鬼姨，一时间也是说不出话来。
<br/>　　说到底，还是谁都没有预料到会是这个结果。
<br/>　　“鬼姨，陈晓宇输了，自然是要履行约定……”一个弟子走到鬼姨身边，大声说道，“不过，眼下吉时就要到了，咱们还是大事为重，先让两位新人拜天地，之后再让陈师兄出来谢罪，比较妥当……”
<br/>(本章完)
