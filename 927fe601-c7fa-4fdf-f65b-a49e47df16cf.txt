　　武器装备之类东西基本上都是从大唐现役部队淘汰下来的东西，或者说是从边军那里淘汰的东西，自从十六卫开始换装新式步枪，以前的那些火铳什么的全都转手给了边军和地方。
<br/>　　而地方上的那些远近战武器便再一次淘汰下来，成了堆在一起的废铜烂铁。
<br/>　　现在，有人要买，称心自然不会再藏着掖着，响应李承乾的号召，大手一挥：卖！
<br/>　　所以当拜占庭人找上门的时候，称心同样没有任何犹豫，把王子木叫到身边，给他写了一张字条，连人都没见直接就打发了出去。
<br/>　　一千把钢弩，一千柄横刀，连同配套的弩矢，不算运费一共卖了五千贯，也就是五百两黄金。
<br/>　　拜占庭人显的很是高兴，将东西打好包装之后，用几辆大车将东西装了，‘雇’了一些脚夫便将东西运往咸阳车站的临时仓库，等待下一班通往天竺道的火车。
<br/>　　……
<br/>　　平安无事的度过了一个安静的夜晚之后，第二天下午，王子木接到了‘自己人’的消息，那个渊男健果然派人来了，送来了一张字条，上面写了一个时间和一个地点。
<br/>　　李毅恒拿着字条反反复复看了数次，又交给黄浦看了半天，最终两人决定按照字条上写的时间去赴约，看看对方到底是怎么安排自己这些人混进皇宫的。
<br/>　　而就在李毅恒与王子木拿着字条讨论是否要去赴约的同时，那个神秘的渊男健却出现在情报科杨天的办公室。
<br/>　　“怎么样？他们相信你了没有？”杨天命人给‘渊男健’泡上一杯热茶，正色问道。
<br/>　　“他们有选择么？”‘渊男健’摇着手中的折扇，文质彬彬的说道。
<br/>　　“老吴，你还真把自己当成渊男健了？”杨天看了一眼‘渊男健’，没好气的说道。
<br/>　　“要装就要装的像一些，否则万一露出麻脚怎么办？”‘渊男健’呵呵一笑，‘唰’的一声合上折扇，摆了个poss，假模假式的端起茶盏，慢慢的吸溜了一口。
<br/>　　杨天无可奈何的看着‘渊男健’，叹了口气说道：“老吴，咱好好说话不行么？你再这德性，可别怪我跟嫂子告你状。”
<br/>　　听到告状，‘渊男健’一下子坐直了身子，打着哈哈说道：“别，别啊，小杨，咱们再怎么说也是兄弟一场，你的本事还有一半是我传给你的，对吧？”
<br/>　　“对，所以你能有点前辈的样子么？你这样让我很难做。”杨天不无威胁的说道。
<br/>　　“好好好，算我怕了你了。”‘渊男健’终于卸下自己的伪装，露出一本正经的神情说道：“死了四、五个人，足以取得那个什么鬼面的信任，让他相信我并不是情报科的暗线，而他们的力量似乎也不是很强，从其据点的防御水平来看，他们找来的人多数都是些散兵游勇，或者一些对大唐政策不满囚犯，这样的情况下他们这一伙儿人除了阴谋诡计根本玩儿不出什么太大的花样。”
<br/>　　“万一他们有别的打算呢？这一次他们可以弄走了一千把强弩，还有一千柄横刀，能在长安城有这么大的势力，你觉得他们还是小势力？”杨天皱了皱眉头反问道。
<br/>　　老吴淡淡一笑：“掩人耳目而已，如果他们在长安真的能弄出一千人，情报科也就没有存在的必要了。”
<br/>　　杨天苦笑一声叹道：“呵呵，这么多年你一直在外面，不知道大唐的变化到底有多大，现在已经不是当年了，长安城里数百万人，情报科就算是再强大，也不可能把这么多人全都清查一遍。”
<br/>　　“就算是这样又能如何？该露出的尾巴他们已经全都露出来了，我明确的告诉过他们要刺杀陛下，这已经把他们逼到了绝路上，所以这次他们一定会来，而且还是孤注一掷。”
<br/>　　“你有把握就好，这股势力一直藏的很深，我们也没有想过他们竟然和世家还有接触，所以这一次必须毕其功于一役，彻底将他们铲除，否则将来必成大患。”杨天斩钉截铁的说道。
<br/>　　“放心吧，一定没问题。”老吴深吸一口气，抓起桌上已经冷下来的茶水一口灌进嘴里：“两个主要头目就在称心的眼皮子底下，王家也被李悦监控起来了，宫里还有黑子在等着他们，就算是他们生了翅膀也很难再跑掉。”
<br/>　　杨天沉默了一会儿，似乎在考虑老吴的判断是否准确，半晌之后才点点头，接着问道：“那另一伙人呢？就是那个蒙着脸的家伙，你知道他的身份么？”
<br/>　　“听口音应该是西域那边的人，虽然这家伙在尽力模仿咱们的官话，但多少还是带了一些西域口音。”老吴想了想，十分肯定的说道。
<br/>　　“知道大概是什么人就好，他们那一伙人那天离开之后就进了林子，后来干脆就躲进了山里，我们的人跟了一段之后就被他们给甩开了。”杨天的语气中带着懊恼，似乎在为这一次的失利而后悔。
<br/>　　“加强戒备吧，重点监控一下进城的西域人，还有那些胡商。”老吴在知道那蒙脸的汉子脱离了监视之后，脸色微微一变，有些担心的说道：“在没有找到此人之前最好不要让陛下离宫，以防他们狗急跳墙。”
<br/>　　“也只能这样了。”杨天叹了口气：“不过，我担心的是陛下不听劝阻，你与陛下接触的时间比我要久，相信你应该知道他的性格。”
<br/>　　老吴的沉声说道：“那就加派人手，总之陛下一定不能出问题，否则大唐必将迎来一场大乱。”
<br/>　　“嗯，我尽最大的能力，如果在长安杯开始之前还没有那伙儿人的消息，我杨天就是死，也不会让陛下出宫半步。”杨天似乎也发了狠，咬着后槽牙说道。
<br/>　　“你有打算就好，我先走了，那伙人我会让我的人注意一些，你这边也要再努力一些。”老吴见事情说的差不多了，从椅子上站起，负手离开了杨天的办公室，从情报科离开，消失在外面的大街之上。
