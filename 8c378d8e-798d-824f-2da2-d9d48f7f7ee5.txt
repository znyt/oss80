　　第503章
<br/>　　聂隐娘窥见程大雷，便是无名火冒，恨不得立刻一刀割了他的喉咙，了却这件事情。然而，今日被困在此境地，一根树杈撑不住二个人，任何的轻举妄动，树杈折断，二人都得丧命。
<br/>　　程大雷盯着聂隐娘寒霜一张脸，忽地悠悠叹了口气，道：“好姑娘，你我往日无怨，近日无仇，何苦缠着我不放，非要杀我才好？”
<br/>　　聂隐娘冷哼一声，道：“我杀你不为私仇，为的是天下无辜百姓，杀你这一恶人，便可救无数苍生。”
<br/>　　程大雷撇撇嘴，道：“这恶人二字，不知从何说起？”
<br/>　　聂隐娘抿起嘴，冷笑道：“你当自己还是什么好人么？自从你出世以来，做过多少恶事，大闹长安，血洗草原，并州杀人盈野，一件件一桩桩多少无辜之人因你而死。程大雷，你坏事做尽，今日我不杀你，改日天也要杀你。”
<br/>　　“啊呀呀。”程大雷长叹一声：“这实在天大的误会，想程某出世以来，虽不敢说件件事情都做得对，却也不敢违天地正道。姑娘如此想我，实在令程某无地可容。”
<br/>　　聂隐娘斜眼睨着程大雷，一脸不屑。
<br/>　　“程某也是自小读圣贤书，懂得仁义二字，姑娘若是不信，某可一一说与你听。”程大雷道：“青牛山一战，挡的是戎族十万兵，救我帝国多少黎民百姓，这堪称一个勇字。大闹长安，剑斩杨龙停，是为我死去的弟兄讨个公道，这堪称一个义字。千里走草原，刺杀草原王，一剑换帝国数载无刀兵之祸，堪称一个忠字。”
<br/>　　聂隐娘怔了怔，她也打听过程大雷做的事，程大雷此刻倒也不算撒谎，一时竟有些糊涂起来。
<br/>　　“如此有情有义，有勇有谋，天下又有几人，姑娘何以觉得我是个恶人，苦苦相逼一定要杀我！”程大雷如蒙奇冤，仰天长叹。
<br/>　　聂隐娘倒是一时不知该说什么好，被程大雷逼问得张口结舌，只是道：“是我师父说的。”
<br/>　　“师父？”程大雷心中一动，便该知道，如聂隐娘这般脑袋拎不清的，背后一定有个更加拎不清的教导出来。
<br/>　　他挥挥手，道：“罢罢罢，天下凡能成大事之人，谁不是誉满天下，谤满天下。程某人自行自己的事，天下人如何想我，却又与我何干。”
<br/>　　聂隐娘一时无语，当真无言以对。事情真真假假，她倒也分辨不清。她从小在山中长大，由白衣尼教导武艺，与山间的猛兽白猿为友，对于人情世故一窍不通。白衣尼说某个人是恶人，她便持剑去杀，至于师父说得是对是错，她委实没有仔细想过。
<br/>　　“呔，你来看！”
<br/>　　程大雷忽然一声喝，手指着大树之下。聂隐娘这时候才回过神来，只见怪蛇和大虫的战斗已快分出胜负。
<br/>　　二兽都是山中的异物，皆有泼天本事，斗下来必然是两败俱伤的局面。这时候，怪蛇缠住了大虫，用力绞杀，猛虎衔住了怪蛇的脑袋，用力撕咬，将怪蛇硕大一颗脑袋咬得稀巴烂。
<br/>　　聂隐娘微微闭目，知道二兽的战斗已快结束，如此看下去，倒是那怪蛇撑不下去。
<br/>　　“姑娘，你怎样看？”
<br/>　　“怕那大蛇已撑不久，不过大虫骨骼也有被这大蛇绞断。”
<br/>　　说出这话，聂隐娘心中一慌，自己怎会回答得如此自然。是下意识将程大雷当做了自己人，而程大雷此刻距离自己，是不是颇近了些。
<br/>　　“喔，既然如此，你便给我下去吧。”
<br/>　　程大雷道了一声，突然一脚踹在聂隐娘身上。以有心算无心，聂隐娘身子直直的从树上掉下来，七八米的高度，瞬间跌得腰酸背痛，全身骨头像是都断了一般。
<br/>　　程大雷在空中一折，轻飘飘落地。他心中想得明白，传说中，那聂隐娘不就是被男人忽悠，背叛了师门么。旁人做得到的事情，自己为何做不到，以这聂隐娘的本事，如果能收为手下，将会是一个不错的助力。
<br/>　　不过眼前的情况，还是要先制服她要紧，程大雷可是不敢冒险。
<br/>　　一脚将聂隐娘从树上踹下来，聂隐娘抬起头，恶狠狠的盯着程大雷，果然，这恶人绝不可信。
<br/>　　程大雷哈哈一笑，上前揪起聂隐娘的肩膀，猛地甩在树上，直接将她砸晕了过去。
<br/>　　此刻那大虫和怪蛇的战斗已结束，大虫将怪蛇脑袋咬烂，从它的束缚中窜了出来。
<br/>　　还没来得及喘口气，仰头就是一块磨盘大小的石头。程大雷哈哈怪叫，石头照着脑袋便砸了过来。
<br/>　　一头斑斓猛虎，被砸得脑浆迸裂，歪死在大树旁。
<br/>　　程大雷又提着剑走到怪蛇边，见它已死透了，倒是不用自己补刀。
<br/>　　然后他走到聂隐娘身边，先摘下她两个肩膀，又用剑砍了一截树藤，手脚绑在一起。
<br/>　　忙碌完这些，程大雷才算是松了口气，一口气泄了，疲惫感瞬间涌了上来，全身骨头像是都散架一般。
<br/>　　今天这一阵忙碌，可是将程大雷累得不轻，他歪在石头边歇了一阵。走到那大虫的尸体前，割开皮肉，让虎血流出来，趴在老虎身上灌了几口虎血。
<br/>　　这时节，聂隐娘悠悠醒转，就见程大雷浑身浴血，趴在老虎身上，脸上头上都是鲜血。若此刻说程大雷是个人，倒不如说他是从山上走出来的猛兽。
<br/>　　只灌了几口，程大雷便觉得有些腹胀，此刻他体力恢复了大半。仰头看见聂隐娘正盯着自己，他哈哈一笑：“醒了，要不要一块来点？”
<br/>　　聂隐娘打了个寒颤，心底冒出一股寒意。
<br/>　　“不吃不吃吧，不吃咱们就早点出去。”
<br/>　　程大雷还担心这山林中再冒出什么怪兽，他上身的小衣被汗水和血水浸湿，程大雷穿着不舒服，顺手脱了丢在地上。
<br/>　　他光着膀子，鲜血模糊一张脸，踉跄来到聂隐娘身边，将她扛在肩头。聂隐娘但凡有半点力气，就算不杀程大雷，也得自杀。可她被程大雷摘了膀子，委实动弹不得，只能任人宰割。
<br/>　　程大雷看着聂隐娘，深一步浅一步向山外走去。
<br/>(本章完)
