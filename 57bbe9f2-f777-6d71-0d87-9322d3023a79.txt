　　五通神一听，心中泛起了嘀咕，忍不住说道：“萧郎君也要管这人间之事吗？”
<br/>　　萧逸云道：“不敢，你们之间的恩恩怨怨，本神无意过问，只是想请神座行个方便，放过我这两个兄弟。”
<br/>　　五通神笑了笑，道：“小神再不济，在人间也是一尊山神，这两位烧我山神庙，又打伤我，如此冒犯，恕我直言，这口气实在认不得。”
<br/>　　萧逸云面不改色，语气也有点硬了起来，说道：“我刚说了，我无意过问你们之间的恩怨，只是恳请神座看我面子，放过这二人。”
<br/>　　五通神沉吟了一下，道：“既然萧郎君开口，那小神也只有照办，请便。”
<br/>　　“谢了。”萧逸云拱了拱手，对叶少阳和毛小方招手，叶少阳立刻从地上抱起云儿，转身离去。
<br/>　　五通神突然一个闪身，挡住三人去路。
<br/>　　萧逸云微怒，说道：“神座反悔了？”
<br/>　　五通神道：“既然答应，怎可反悔。只是……我只答应让他们两个走，那姑娘是我……我的信徒，若是被他们带走，小神面子往哪儿搁？”
<br/>　　萧逸云看了叶少阳一眼，叶少阳不容置疑地说道：“必须带走。”
<br/>　　萧逸云对五通神道：“一位信徒而已，也请神座放过。”
<br/>　　五通神哼了一声，道：“小神刚也说过，这是面子的事，恕不能照办。”
<br/>　　萧逸云刚要开口，突然想到什么，对叶少阳使了个眼色，叶少阳会意，抱着云儿奔跑起来，毛小方一愣，看了萧逸云一眼，也跟着跑。
<br/>　　五通神也是愣了一下，想要追赶，萧逸云突然打出三道玉符，在身前形成一道结界，挡住了五通神的去路。
<br/>　　“神座，还请行个方便。”
<br/>　　“你……你欺人太甚！”五通神大怒，看着远去的叶少阳，盛怒不已，欺人太甚几个字脱口而出，但立刻就后悔了。
<br/>　　萧逸云面色一沉，道：“五通神，注意言辞。”
<br/>　　“是，小神失言了。”五通神在人间再怎么牛逼，没有法师敢动，但毕竟也只是个山神。萧逸云是天子门生，所谓不看僧面看佛面，就算是阴司诸位大佬也要给几分面子，得罪他没什么好果子吃，最重要是木已成舟，现在再跟萧逸云闹翻，就很不值得了，自己总不能强行突破封印去把云儿抢回来。
<br/>　　“今日小神就给萧郎君一个面子，那位女信徒，便让那二人带走，此事终了，只是若那二人再来与我纠缠不休，请恕小神无礼了……”
<br/>　　“这是自然，我说过我不参与你们的恩怨，多谢。”
<br/>　　萧逸云转身离去。
<br/>　　五通神恨恨地望着几人的背影，最终也只能拂袖而去。
<br/>　　叶少阳抱着云儿跑了一阵，回头看见萧逸云走过来，五通神却不见了，心下彻底放松，把云儿放了下来，冲她笑道：“没事了。”
<br/>　　云儿神情呆滞，一时间有点不敢相信五通神就这么放过他们，怔怔无语地望着叶少阳。
<br/>　　眼看萧逸云走过来，叶少阳说道：“你怎么来了？”
<br/>　　“我来告诉你一件事情，结果正好看到你们斗法，也是巧了。”
<br/>　　叶少阳道：“你来得还真是及时。对了，找我什么事。”
<br/>　　“你们茅山掌教云秋生羽化了，刚去了阴司报道。”
<br/>　　云秋生……叶少阳一下子愣住了。云秋生是自己的太师祖，自己当然没见过，也谈不上有什么感情，听到这消息不过是有些吃惊，于是问道：“怎么死的？”
<br/>　　“正常羽化。”
<br/>　　“哦。”既然是善终，自己也没什么可说的，叶少阳问道：“阴司怎么安排的？”
<br/>　　“阴司自然是想留下，但云秋生想要去轮回，也只要由他了。”
<br/>　　叶少阳点点头，茅山派几百年来，除了陶弘景，其余的掌教一向都是死了之后就去轮回，几乎没什么留在阴司做官的，因此青云子死了之后才那么坚持去轮回，要不是自己苦苦恳求，也不会留在阴司。
<br/>　　萧逸云接着说道：“云秋生羽化之后，伏明子继任掌教。”
<br/>　　叶少阳当然知道这一结果，道：“然后呢？”
<br/>　　“没什么然后，就是想起你是茅山弟子，过来告诉你一声。”
<br/>　　叶少阳耸肩道：“我不是这个时代的人，我又不能去吊唁，我师父可能都还没生出来呢，茅山上也没人认识我。”
<br/>　　想了一下，叶少阳道：“五通神走了，没跟你为难？”
<br/>　　萧逸云冷哼一声，都不想回答这个问题。五通神如果敢跟他动手，除非真是脑子进水了。一眼扫到云儿，对叶少阳说道：“五通神说了，放过这姑娘，将来不会再来找她。”
<br/>　　云儿一听，立刻激动地哭起来，突然又有些不放心，说道：“他若不守信用呢？”
<br/>　　“那不可能。”萧逸云斩钉截铁，“他既然当面答应我，就不可能再反悔。”
<br/>　　“为什么？”事关自己将来的安危，云儿当然想问个清楚。
<br/>　　萧逸云笑起来。
<br/>　　叶少阳说道：“你就放心吧，五通神不敢跟他耍什么手段，他没这个胆子。”
<br/>　　云儿怔怔地看着萧逸云，一时间五味杂陈，想说声谢谢都说不出来。
<br/>　　萧逸云对叶少阳说道：“五通神说了，也不会再找你们麻烦，你还是要屠神吗？”
<br/>　　叶少阳沉默了一瞬间，点点头。
<br/>　　萧逸云耸了耸肩，“随便你，不过你如果坚持对付他，下一次，我真的不好再保你了，也保不住，毕竟这事关人家性命，人家总不能站着让你砍，而且我也不能一直待在阳间。”
<br/>　　叶少阳道：“我知道，忙你的去吧，我自有分寸。”
<br/>　　萧逸云因为身份所限，不太好知道太多关于屠神的事，反正这件事上自己也帮不上忙，跟他们说了几句话，也就走了。
<br/>　　五通神不会再来追，叶少阳也不着急赶路，问毛小方受伤重不重。
<br/>　　“我没什么，倒是你怎么样？”
<br/>　　“也还好，没大事。”
<br/>　　想起刚才的经历，叶少阳忍不住骂了一句，“妈个鸡的，虽然成功了，不过实在有点憋屈！这蛤蟆精，如果我有十成法力，哪有这么费事！”
