<br/>　　战斗已经结束了。

<br/>　　“你不要忘了，我也是茅山弟子，茅山的任何法器吗，我都能用。”
<br/>　　战斗已经结束了，虽然不知道过程怎么样，但是，道风打赢了。
<br/>　　芮冷玉松了一口气，冲道风喊道：“杀了他！”

<br/>　　“师兄……”叶少阳突然低声喊了一声，用哀伤的眼神望着他，“不管怎么样，你总是我师兄啊。”
<br/>　　道风犹豫起来。
<br/>　　虽然明知道他不是真正的叶少阳，但毕竟长的一模一样，这种视觉上的冲击，仍然很难克服，就算是道风，面对这样的局面，不免也犹豫了起来。
<br/>　　“师兄，不要杀了我，我……”
<br/>　　“噗嗤！”
<br/>　　七星龙泉剑插进了他的喉咙，一股热血喷射而出。
<br/>　　叶少阳睁大了眼睛，不敢相信地望着道风。
<br/>　　“你不是叶少阳，为了他，我更要杀了你。”
<br/>　　从剑锋刺进去的地方，叶少阳的肉体开始一点点分解，化成粉末状，消融在空气中。
<br/>　　他的身体就这样一点点腐蚀掉，最后是脑袋，一双满是血丝的眼睛还死死地瞪着道风，眼神中满是绝望和不甘，最后，连这双眼睛也消失了……
<br/>　　没有精魄。
<br/>　　他的精魄，八成是又回到了自己那个世界去，毕竟他是属于那个世界的。
<br/>　　几秒钟后，道风手中的七星龙泉剑也开始碎裂，像叶少阳的身体一样，粉化消失……
<br/>　　它也不是这个世界的东西，主人不在了，它也就消失了。
<br/>　　道风丝毫没有惋惜，反倒松了一口气，如果七星龙泉剑留在这世界，反倒会出问题，到时候两把同样的剑，虽然多一把可以用，但这感觉太过怪异，还是这样好，干干净净，这个恶灵空间的叶少阳一死，把什么都带走了，就像没发生过任何事情。

<br/>　　“是我自己。”
<br/>　　“什么！”
<br/>　　“我是身外化身。”
<br/>　　芮冷玉呆住，身外化身……
<br/>　　“怎可能，若是身外化身，怎会如此饱满充实，还要这么强大的法力，不可能的。”芮冷玉伸手捏了捏他的脸。

<br/>　　“龙晶！”

<br/>　　洞口水帘外面，还有一个道风，正盘膝坐在地上，全身上下，头顶，双肩，双膝，还有两个手掌上，都有一朵鲜花落在上面，七朵花，一共其中颜色，释放着七种颜色的暗光，连在一起，形成了强大的封印。
<br/>　　道风盘膝不动，身上不断渗出黑色的邪气，苦苦抵抗着。
