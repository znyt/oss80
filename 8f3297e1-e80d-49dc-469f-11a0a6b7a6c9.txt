　　日上三竿，沉睡中的佳人们也纷纷起床。
<br/>　　坐在客厅沙发上，慕清霄拿着面包塞入口中，道：“紫嫣，有件事情，需要跟你商量一下。”
<br/>　　“说吧，什么事情？”
<br/>　　欧阳紫嫣美眸中闪过一抹诧异，略有疑惑的问道。
<br/>　　“现在，别墅区用来给普通人居住，实在太浪费了......”
<br/>　　说着，慕清霄道：“我想在周围种植一些灵药灵果，但需要很大的空间，所以需要将金帝别苑全都圈下来。”
<br/>　　欧阳紫嫣经商已久，自然清楚所谓的圈下来是什么意思。
<br/>　　“事情很急吗？”
<br/>　　“倒是不急，但越快落实越好，毕竟将周围的别墅全部推倒重建，需要很大的工程。”
<br/>　　慕清霄已经思考过了，既然要改造，那就要改造的彻底点，以后金帝别苑就完全是自己的地盘。
<br/>　　可能在不久的将来，其中会种植培养许多高贵的灵药，杜绝普通人进入是最好的。
<br/>　　微微点头，欧阳紫嫣道：“那些住户，你打算怎么办？”
<br/>　　“你派人去协商吧，价格尽量让住户满意，我最近可能需要出去一趟，很快就回来。”
<br/>　　“行，你先忙自己的事情吧，别墅区的事情，我待会就让人开始处理。”
<br/>　　将杯子轻轻放在桌上，欧阳紫嫣满口答应，现在钱对她们来说，顶多就是一串数字，自然分得出轻重。
<br/>　　几句话就将此事定了下来，吃过早餐，语嫣和小依则前往学校，欧阳紫嫣几人也前往集团。
<br/>　　雪琪和阿碧则守在家中，除了打扫家务，其余的时间都用来修行，彼此磨练。
<br/>　　站在院落中，慕清霄喃喃自语，道：“也是时候回去看看了，系统，回火影位面吧。”
<br/>　　“叮，返回火影位面，宿主需支付10000掠夺积分。”
<br/>　　“支付！”
<br/>　　“掠夺积分扣除成功，即将返回火影位面，宿主信息数据化......”
<br/>　　宿主：慕清霄（龙骑士）
<br/>　　血脉：未知
<br/>　　资质：未知
<br/>　　剑意：无上剑意
<br/>　　武器：魂剑
<br/>　　战技：无
<br/>　　修为：化神九重中期！
<br/>　　心法：《御女心经》
<br/>　　伴侣：王语嫣，阿碧，阿朱，波雅.汉库克，欧阳紫嫣，日向雏田，小依，陆雪琪，彩鳞。
<br/>　　掠夺积分：220000
<br/>　　储存空间：7200
<br/>　　系统提示声刚落，慕清霄眼前一黑，再次睁开双眼，已经出现在一片诺大的森林中。
<br/>　　在茂密的枝叶遮掩下，丝丝阳光只能穿过缝隙洒落而下，所以森林中的光线非常暗淡。
<br/>　　当然，对于慕清霄来说，并无大碍，视线在四周环视一圈，道：“灵气依旧是如此的稀薄，空气倒是干净。”
<br/>　　“这么长时间没回来，那妮子估计恨死我了吧，也不知道现在是怎样的局势。”
<br/>　　说着，慕清霄就从储存空间中，取出一套绣着红云的黑色风衣，另外还有一件头戴系着风铃的斗笠。
<br/>　　旋即，身躯整个都笼罩在黑袍中，根本看不清脸部，显得神秘异常。
<br/>　　“也不知道，晓组织是否还存在，罢了，存在与否，与我有何关系，先回木叶看看吧，叮叮铃铃......”
<br/>　　“站住！”
<br/>　　就在他准备离开之际，森林中陡然响起一道厉喝声，随后就有数十道身影，快速围了过来。
<br/>　　慕清霄抬起头来，朝森林四周的高大树木看去，只见每颗大树粗壮的枝干上，都站着一道身影，手中拿着苦无。
<br/>　　包围过来的总共有二十人左右，从他们的护额上来看，赫然都是沙忍者村的忍者。
<br/>　　其中，有一名男子吸引住了慕清霄的视线。
<br/>　　男子身形颇为高大，身穿黑色连身装，脸上涂绘着紫色的花纹，但此刻的脸色，却是显得极为难看。
<br/>　　盯着慕清霄许久，眼中尽是不可思议：“晓组织，应该早就覆灭了才对。”
<br/>　　对于男子的着装打扮，慕清霄一眼就认出了他来，正是沙忍者村的傀儡师，勘九郎。
<br/>　　听到他的话语，慕清霄道：“你说晓覆灭了？斑也已经死了吗？”
<br/>　　勘九郎目光有些怪异，显然对慕清霄的话语，感到莫名其妙。
<br/>　　“斑早就已经是过去式，你是晓组织的成员，应该比我更清楚吧，不老实在暗中待着，又想出来扰乱秩序了吗？”
<br/>　　虽然勘九郎的语气慎重，却没有害怕的意思，慕清霄叹息一声，看来真的已经过去很长时间了。
<br/>　　“距离第四次忍界大战，过去多久了。”
<br/>　　这下不仅是勘九郎，就连周围的忍者都露出怪异的神色。
<br/>　　此时，勘九郎不断思索后者的身份，如果没必要动手，最好是不动手，毕竟晓组织在他印象中，都是异类。
<br/>　　“距离第四次忍界大战，已经过去十年了。”
<br/>　　话语刚落，森林中的变得一片死寂，针落可闻，气氛变得有些压抑。
<br/>　　“叮铃铃......”
<br/>　　风铃随风飘动，响起一阵清脆的铃音，谁也不知道，绣着红云的黑色风衣下，男子是怎样的表情。
<br/>　　见他无视自己等人，其中一名忍者顿时有些不爽，刚想要出手，却被勘九郎快速拦了下来，道：“不要乱来。”
<br/>　　望着慕清霄离去的背影，勘九郎道：“给木叶传讯，让他们最近警惕一点。”
<br/>　　“是，勘九郎大人。”
<br/>　　...
<br/>　　...
<br/>　　离开砂忍者村的范围后，慕清霄站在树尖上，望着木叶方向，思绪万千。
<br/>　　“居然已经过去十年了，先不管了，回木叶看看吧。”说罢，周围空间一阵扭曲，身影就自原地消失不见。
<br/>　　几个时辰后，五大国之一火之国境内，坐落着一座村庄。
<br/>　　木叶忍者村外的一座山峰上，空间一阵扭曲，一道身影凭空而现，站在山巅，数千米外，一座偌大的村庄尽收眼底，其中尽显繁荣。
<br/>　　在村庄的尽头，黄土高坡上，刻着七张脸谱，慕清霄眼眸中古井无波。
<br/>　　“已经是第七代了，当年的毛头小子，都已经成为火影了吗？真是时过境迁，沧海桑田啊。”
<br/>　　想到当初，自己夺走了万花筒，也不知道佐助的命运会如何，至于雏田，慕清霄根本就没有担心过。
<br/>　　“先回村子里看看吧。”
<br/>　　说着，慕清霄身形一闪，便出现在森林中的小道上，迈着缓慢的步伐，向着木叶忍者村走去。
