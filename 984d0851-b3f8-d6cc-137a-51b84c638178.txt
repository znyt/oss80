　　祝明飞顿时急了，
<br/>　　“这怎么行？叶兄弟，你的伤势很严重，不能再和人交手，否则会很危险的！我绝对不允许！”
<br/>　　叶尘嘴角微微一扬，
<br/>　　“谁说我要和他们交手了？我只是负责将人引开而已，他们那群废物还伤不了我！”
<br/>　　祝明飞依旧摇头，
<br/>　　“不行不行！叶兄弟，以你如今的状态，恐怕根本无法做到，还是让我来吧！”
<br/>　　叶尘淡淡一笑，
<br/>　　“怎么？你小瞧我？”
<br/>　　祝明飞还欲解释，叶尘又摆了摆手道：
<br/>　　“这样吧！祝兄，我们来比试一下，咱们以一百个数为限，就在这间屋子里，若是你能碰到我，我便不和你争，如何？”
<br/>　　祝明飞听到这话，顿时眉头一挑，
<br/>　　“你当真？”
<br/>　　叶尘的身法，祝明飞之前是见识过，在他生平所遇之人之中，绝对无出其右。
<br/>　　不过，如今叶尘身受重伤，而且又是在如此狭小的房间之中，他怎么也不相信，自己连碰到对方都做不到？
<br/>　　好歹他也是潜龙榜首，实力就算比五大宗门的宗主，也只是稍逊一筹而已，绝对是蓬莱仙界的顶级强者！
<br/>　　叶尘微微一笑，
<br/>　　“当然当真！不过避免引起外人注意，我们只比身法，不得损坏房间里的任何物品，如何？”
<br/>　　祝明飞略微思量了片刻，直接点了点头，道：
<br/>　　“比就比！若是我赢了，你可不许反悔！”
<br/>　　叶尘哈哈一笑，
<br/>　　“我叶狂仙一言九鼎，从不食言！”
<br/>　　“好！”
<br/>　　祝明飞也顿时来了脾气，脚下猛然一步跨出，便直奔叶尘扑了过去。
<br/>　　唰！
<br/>　　祝明飞虽然不以身法擅长，可好歹也是元婴境强者，速度自然不差，几乎如同瞬移般冲到叶尘跟前，然后一把便抓住了叶尘的肩头，
<br/>　　“你输……”
<br/>　　祝明飞才刚刚开口，却戛然而止，脸色的笑容也瞬间凝固。
<br/>　　因为他无比惊愕的发现，自己这一掌抓下去，竟然扑了个空！
<br/>　　而此时身后，陡然响起了叶尘的声音，
<br/>　　“祝兄，你这是瞧不起我吗？还是拿出你的全部实力来吧！”
<br/>　　祝明飞猛然回头，却见叶尘，不知何时，竟然已经坐在他身后的椅子上，一副老神在在的模样。
<br/>　　“怎么会……”
<br/>　　祝明飞顿时大骇，
<br/>　　他刚刚，竟然丝毫没有捕捉到，叶尘的行动轨迹！
<br/>　　叶尘就像是鬼魅一般，先是凭空消失，然后又凭空出现在他的身后！
<br/>　　若是在叶尘没有受伤之时，或许他还可以接受，可是他亲眼看到叶尘昨天受伤有多重，这才短短一天时间，居然已经恢复了如此速度，简直令人匪夷所思。
<br/>　　而就在祝明飞惊愕不已之时，另外一边，负责数数的祝小白，已经数到了“十”
<br/>　　祝明飞顿时心头一禀，也不敢再继续耽搁时间，当即再次扑了过去，
<br/>　　“我还就不信了！”
<br/>　　祝明飞不敢再有所留手，这一步跨出，脚下的石地板，都出现了一个清晰的脚印。
<br/>　　呼！
<br/>　　连同整个屋子都狂风大作！
<br/>　　然而，
<br/>　　祝明飞闪电般冲到叶尘跟前，却又再次扑空！
<br/>　　“什么！”
<br/>　　祝明飞这一次，打起一百分的精神，并且神念死死锁定叶尘的身影，努力的想要捕捉他的行动轨迹。
<br/>　　可是却依旧一无所获！
<br/>　　叶尘似乎根本就没有动，就那么凭空消失了！
<br/>　　而等他再次感应到叶尘的气息之时，叶尘已经不知道用了何种手段，再次到了他的身后。
<br/>　　这一惊，当真是非同小可！
<br/>　　“难道他已经掌握了传说中的法则之力……”
<br/>　　祝明飞惊骇不已，心头思绪纷纷的同时，却也没有轻易认输，在他重新锁定叶尘的气息之后，便当即一转身，直接再次朝叶尘抓去。
<br/>　　这一次，
<br/>　　就连叶尘也颇感意外，没想到祝明飞的反应竟然如此之快！
<br/>　　在他动用空间法则之力的情况下，居然还是险些让对方碰到自己的衣角。
<br/>　　好在叶尘反应也不慢，在最后关头，身体先是暴退一步，然后再次动用空间法则之力，险而又险的躲开了对方这一抓。
<br/>　　“61……79……88……100！”
<br/>　　一转眼的功夫，
<br/>　　一百个数已经到了，祝明飞使出了浑身解数，却愣是连叶尘的衣角都没有碰到。
<br/>　　而叶尘依旧双手负在身后，回到了最初的位置，似乎从来都不曾移动过一般，似笑非笑的望着祝明飞，
<br/>　　“如何？”
<br/>　　祝明飞顿时颓然长叹一声，继而冲着叶尘郑重行了一礼，道：
<br/>　　“叶兄弟修为惊人，我服了！”
<br/>　　“枉我一向自认为是蓬莱第一天才，今日方才知道，人外有人，天外有天！”
<br/>　　“我的天赋和叶兄弟一比，真是不敢再称天才二字了！”
<br/>　　叶尘哈哈一笑，
<br/>　　“祝兄不必妄自菲薄，其实我的天赋，也未必强过你，只是这其中缘由复杂，一时半会很难解释清楚，你若是都算不上天才，那整个蓬莱仙界可还有天才吗？”
<br/>　　叶尘这话却是肺腑之言，
<br/>　　祝明飞的天赋，即便是在修真界中，也属于一流，虽然还无法和那些惊才艳艳绝世天骄相比，但在地球这样的低级星域，绝对是凤毛麟角的存在。
<br/>　　只可惜，他所遇到的是，有着八百年记忆的叶尘，自然没有可比性。
<br/>　　“好了！时间紧迫，我们来商量一下行动计划，马上动身！”
<br/>　　……
<br/>　　太初城中央，传送高台之上，
<br/>　　十余道强大的气息，冲天而起，周围太初城的市民们，一个个全都躲得远远的，根本不敢靠近。
<br/>　　“南宫长老，听你们罗霄宗主所说，那个世俗界的小子，以一人之力灭了整个药王宗，还杀了三十余名元婴境，此事是到底是真是假？”
<br/>　　其中一名老者忽然开口道。
<br/>　　其余众人听到这话，顿时纷纷望向另外一名老者，应该就是太乙门的那位南宫长老了。
<br/>　　“这个……”
<br/>　　南宫长老的脸上顿时露出一抹尴尬之色，
<br/>　　五大宗门宗主，被斩杀其三，药王宗更是惨遭灭门！
<br/>　　任谁也无法相信，这竟然是一名世俗界的少年所为。

<br/>　　毕竟在他们眼中，世俗之人，犹如蝼蚁一般，而且区区一名少年，怎么会有如此实力？
<br/>　　就在众人议论纷纷之时，忽然一道淡漠的声音，陡然在太初城上空响起，
<br/>　　“是真是假，你们试试不就知道了？”
