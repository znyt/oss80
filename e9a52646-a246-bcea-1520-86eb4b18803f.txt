陆子羡看了看她，然后目光落在她手里的化验单上，“你是不是中了花毒？”
<br/>
<br/>        夏夕绾迅速将化验单折叠了放在自己的包里，她没有回答这个问题，而是转移了话题，“陆院士，谢谢你又一次救了我，你已经救我好几次了，等有时间，我跟陆先生请你吃饭吧，表达我的谢意。”
<br/>
<br/>        她说她跟陆寒霆一起请他吃饭，大家都是成年男女，陆子羡揭开了她脸上的面纱，她现在给予了回应，陆子羡懂了。
<br/>
<br/>        “我还有事，先走了。”夏夕绾抬脚离开。
<br/>
<br/>        “夕绾！”陆子羡伸手扣住了她纤细的皓腕。
<br/>
<br/>        夏夕绾停下了脚步，她用力的抽回了自己的皓腕，“陆院士，我是你嫂子，你是我小叔子，以后叔嫂之间还是保持一点距离，陆先生会误会的，别人看见了也会非议。”
<br/>
<br/>        陆子羡看着她澄亮的翦瞳，抿了一下薄唇，然后道，“夕绾，跟我回帝都城吧，你中了花毒，跟我回帝都城，我可以救你。”
<br/>
<br/>        夏夕绾抬眸看向陆子羡。
<br/>
<br/>        “夕绾，难道你想死吗？”
<br/>
<br/>        夏夕绾摇头，“不，我想活着，我还有很多事情要做。”
<br/>
<br/>        “那你就跟我回帝都城，这是你活下来的唯一办法。”
<br/>
<br/>        夏夕绾垂下了纤长的羽捷，没有说话。
<br/>
<br/>        “夕绾，你舍不得这里？这座海城没有你的亲人了，它留给你的记忆大概只有儿时的背叛和抛弃，你为什么还要留在这里，帝都城是医学中心，那里繁花似锦，我有一种感觉，你天生就来自那里，你是属于那里的，跟我离开海城去帝都城。”
<br/>
<br/>        夏夕绾看着脚下斑驳的路，半响轻声道，“可是…这里有陆先生啊，这里是我和他相遇的地方，有我和他所有的记忆，我舍不得离开这里，放不下他，纵然这座城市有我最痛苦的记忆，但是因为陆先生，我心里还是深爱着这座城。”
<br/>
<br/>        因为一个人，爱上一座城，陆子羡看着夏夕绾，“那你的身体，怎么办？”
<br/>
<br/>        “去帝都的事情我再考虑一下吧，不过陆院士，不要将时间浪费在我的身上，我不值得。”说完，夏夕绾转身离开。
<br/>
<br/>        陆子羡一个人站在原地，看着夏夕绾的俏影消失在自己的视线里，然后他拿出手机，给陆寒霆发了一条短信。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾不想死，她要好好的活着，所以帝都一行必须纳入计划里了，不过再离开之前，她要将夏家的事情全部处理掉。
<br/>
<br/>        十年前爷爷摔下楼的真相她要亲手揭开，还有夏妍妍最后的结局，也应该由她来谱写！
<br/>
<br/>        夏夕绾打车去了夏家，想见夏老爷子，按照时间来推算的话，爷爷应该就这几天会苏醒了。
<br/>
<br/>        但是当夏夕绾进入房间时，床上已经没人了，夏老爷子不见了！
<br/>
<br/>        夏夕绾迅速问向女佣，“我爷爷呢？”
<br/>
<br/>        这时李玉兰扭着腰肢走了过来，“夏夕绾，不用找了，我们已经将老爷子转移到别的地方去了，以后我们不会让你再见老爷子的。”
<br/>
<br/>        夏夕绾的眸光倏然一冷。
