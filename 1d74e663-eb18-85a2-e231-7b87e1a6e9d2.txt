　　“还有一件事，影长老哪里去了？”
<br/>　　星月奴目光一动，道：“上师既然知道，绕什么弯子？没错，影师兄去找那叶少阳了。”
<br/>　　“要杀他？”
<br/>　　“能抓就抓，抓不住，那就只有杀了他，以绝后患。”
<br/>　　鸠摩罗什微微皱眉，道：“不该如此。”
<br/>　　星月奴笑道：“上师是想替那叶少阳求情？”
<br/>　　“非也，我是替影长老不值，此战若丧命，千年道行，毁于一旦。”
<br/>　　星月奴愣了一下，哈哈大笑，“影长老千变万化，修为极深，会打不赢他一个乳臭未干的小法师？”
<br/>　　鸠摩罗什没言语，隔了一会说道：“星长老，阴司正四处捉拿你，往后行动，当小心谨慎。”
<br/>　　星月奴冷笑：“我而今已证大道，阴司又怎么了，无非是一群冢中枯骨！”
<br/>　　“话不可说尽，阴司之中，卧虎藏龙，强者众多，再有，你而今气候已成，不可再造杀业，于道心无益。”
<br/>　　星月奴面色一沉，道：“本宫自有分寸，无需上师提醒！”语气很不耐烦。
<br/>　　鸠摩罗什身边沙弥气不过，说道：“星长老，我师父好心为你护法，如今功成，你是觉得自己了不得了吗，说话这般没大没小！”
<br/>　　“你算个什么东西！”星月奴骤然出手，青州鼎挟着万钧之势，朝他砸过来。
<br/>　　鸠摩罗什突然分开双手，一只手平推出去，咣当一声拍打在青州鼎上，发出铜钟一般的响声，鸠摩罗什身形微微晃了晃，竟然卸掉了青州鼎上的万钧力量。
<br/>　　星月奴再度作法，这才发现青州鼎被鸠摩罗什紧紧托住，无论自己如何用力，竟然纹丝不动，当下骇然失色。
<br/>　　鸠摩罗什道：“星长老，何故对我弟子出手。”
<br/>　　星月奴暗暗吸了口气，笑道：“不敢，只是刚证道，想要试一试手段而已，还请上师高抬贵手，将鼎还我。”
<br/>　　鸠摩罗什松开手，让星月奴把青州鼎收了回去。
<br/>　　星月奴吃了瘪，心中有些感慨。
<br/>　　鸠摩罗什道：“证道，只是第一步，星长老，你要走的路还很长。无上甚深微妙法，百千万劫难遭遇……星长老，慎之。”
<br/>　　言毕，转身厉害，带着沙弥从之前打开的缺口飞了出去。
<br/>　　一个弟子走到星月奴身边，道：“师尊，这老和尚真是多事啊！”
<br/>　　星月奴斥道：“休得多言，他是大佛，连大帝都要给他几分面子的。”
<br/>　　众弟子不敢再多说，请示下一步怎么办。
<br/>　　星月奴抬头看了看头顶的缺口，让他们赶紧补上，自己要先在这里养息一段时间，巩固境界。
<br/>　　“另外，去两个人到人间，看影长老那边打完了没有，你们提醒着点，不要杀了，将他擒住，先送到这里来，我要亲自问他一些话。”
<br/>　　她还是想要拿到山海印。
<br/>　　至于影魅能否擒住叶少阳，那是不用担心的，星月奴只担心影魅没轻没重，直接把叶少阳元神也打散了。
<br/>　　“哈哈，人间法师，纵然修炼到顶峰，依旧是人间法师……”
<br/>　　影魅的声音在耳边响起，充满着蔑视和炫耀。
<br/>　　如今的叶少阳，整个人都陷进了虚空之中，虽然死命作法抵抗，但在影魅看来，也只是垂死挣扎罢了。
<br/>　　“好了，结束了。”
<br/>　　他从下面裹住叶少阳的身体，将体内修为一瞬间爆发出来，用力拉扯叶少阳，试图一鼓作气将他拉下去，结束这场挺无聊的战斗。
<br/>　　叶少阳突然睁开眼睛，说道：“结束了吗？”
<br/>　　“就快了。”影魅轻笑，“你要求饶吗，现在倒是来不及了。”
<br/>　　叶少阳表情平静，道：“影魅，你经历过绝望没有？”
<br/>　　“绝望？你而今正在经历的这种吗？”
<br/>　　“那就对不住了……九曜顺行，元始徘徊，华精茔明，元灵散开；流盼无穷，降我光辉，诛尽妖邪，九星齐明！！”
<br/>　　念完最后一个字，他的人也完全陷入了虚空之中。
<br/>　　影魅起初大惊，随后一想，也是觉得搞笑，这时候念咒，就算他身上藏着几张暗金神符，又能把自己怎么样？以他的修为，对付十几张暗金神符的威力还没问题的。
<br/>　　但是数秒钟后，天花板突然亮了起来，影魅抬头看去，还真是暗金神符，不光天花板上，左右墙壁上也有，但也不过是寥寥几张，心中也没当回事，作法聚集了一道结界去抵挡，一边抓住叶少阳的脖子，把他往无间地狱里塞。
<br/>　　叶少阳朝地面上伸出了手，吴嘉伟一个箭步上前，抓住了叶少阳的手，用尽全力网上提，虽然拉不动，但好歹也阻挡了一下他下限的趋势。
<br/>　　“又来一个送死的！”
<br/>　　影魅右手不断生长，巨大的黑影攀附上了吴嘉伟的双腿，试图把他也一并拉下来。
<br/>　　九星神符放射出的星光，大部分并没有击中影魅，而是落在了附近的地面上，然后……奇迹就发生了：
<br/>　　星光并没有消失，而是……折射了出去，又弹到了天花板和四周的墙壁上，墙上一块以一块的灵符相继明亮起来，接着，便是万一束接着一束的星光飞出，打到对面墙壁，立刻激活了新的灵符，落在地上，却又被反射起来。
<br/>　　所有神符的灵力，没有一道消失，反而越来越多，如暴雨一般，密密麻麻，直上直下地打下来。
<br/>　　“啊……”
<br/>　　墙角传来几声惨叫，几道黑影在雨点般密集的星光之中被打得上下翻飞，这些都是影魅的弟子，早就潜伏在四个墙角，神不知鬼不觉地守在这里，是为了埋伏前来救援的人的，现在却被打成了筛子。
<br/>　　“怎么会这样！”
<br/>　　影魅大惊，朝地面上看去，这才发现，在一地的土灰之下，竟然露出了明亮的一块块，将星光映照下来，不断反射。
<br/>　　“镜子！！”
<br/>　　地上铺成一层镜子！
<br/>　　影魅一瞬间明白了。
<br/>　　这道理很简单，镜面，能够反射一切虚拟的光线，包括法术、邪术等等一切力量制造出来的虚幻的精神力，九星神符的光落在上面，自然能够反射出去，而墙上又贴满了九星神符。
