　　几个倭人互相看了看，彼此间都很尴尬，可是没有办法，身高是硬伤。
<br/>　　那倭人青年揉了揉鼻子，收回刀：“你最好不要骗我们，否则你会死的很惨，我们可以有一百种方法让你活的生不如死。”
<br/>　　“我明白。你们现在已经知道我的家在哪里了，我怎么敢把你们的事情向上面汇报。”唐义苦着脸说道。
<br/>　　“最好是这样！”青年狠狠的盯着唐义，半晌看向几个护卫：“你们觉得他说的是真是假？要不要相信他。”
<br/>　　青年最后一句是用倭语说的，唐义听不懂，但看他的表情大致上也能猜到些什么，人也紧张了起来。
<br/>　　“殿下，我觉得可以相信，而且不如我们尝试着收买他一下，也许会有意外的收获。”其中一个护卫比较机灵，眼珠一转想到一个比较可行的办法。
<br/>　　“你觉得可行么？”青年将目光转向那护卫。
<br/>　　“殿下，他并不知道我们有多少人，所以应该不会去告发我们，既然这样的话，我们就有了操作的余地。”护卫十分确定的点点头，自作聪明的说道：“人没有不爱钱的，只要我们给他钱，生存与发财之间相信他一定会知道如何选择。”
<br/>　　“这个办法不错，收买了这个人之后至少也算是有一个内线，下一次抓人也会有目标。”另一个护卫像是得到了某种启发，在一边投了赞成票。
<br/>　　倭人青年低头琢磨了一下，也觉得这个办法不错，看了一眼唐义，然后在身上掏了半天，拿出几张纸，选了一会儿挑了一张放到唐义面前：“知道这是什么么？”
<br/>　　“存单！”唐义点点头。
<br/>　　那个倭人举在他面前的是一张百贯的存单，不记名存单，到了这一步他大概可以确定自己的小命应该是可以保住了。
<br/>　　事实证明，他的猜测并没有错，只见那倭人青年点点头，脸上露出一副和蔼的表情：“只要你听话，按我们说的做，这张存单就是你的。”
<br/>　　“真的么？”唐义眼前一亮。
<br/>　　“本王子还没有骗过人，不过如果你不答应，那么你的忠诚就出了问题，我就会立刻杀了你。”
<br/>　　“我答应！”唐义回答的飞快。
<br/>　　眼下这个情况傻子才会不答应，唐义不是倔驴，在这样的情况下他知道自己只有配合才有活路，至于说答应下来之后，那些倭人有什么条件，回头再说好了。
<br/>　　几个倭人见唐义答应了，纷纷露出笑意，为了表示诚意还特地将他放开。
<br/>　　不过倭人也不是彻底相信了唐义，在将他放开之后又问了他一些在算是机密的问题，一件一件都记了下来算是把柄，然后才提出了他们的条件——大唐最先进装备的制造工艺！
<br/>　　唐义真的拿这些倭人有些无语，看着那个自称王子的倭人青年，琢磨了半天才说道：“你们有蒸汽机么？”
<br/>　　“没有！”倭人青年摇摇头，这东西他听说过，也见过，但是倭国没有。
<br/>　　“那你们有剪板机么？”唐义继续问。
<br/>　　“那是什么？”倭人青年一头雾水。
<br/>　　“十万斤压力机呢？”
<br/>　　倭人青年摇摇头，表示不懂。
<br/>　　“你们啥都不知道？那你们要装备的制造工艺干什么？没有那些机器，你们连最基本的弹体，弹头都做不出来。”唐义发自内心的问道。
<br/>　　几个倭人面面相觑，与唐义的一番对话下来，让他们觉得自己是个傻子，根本理解不了这个扫地的家伙到底在说什么。
<br/>　　最后还是那个倭人青年比较靠谱，瞪了唐义一眼：“你不用管我们要那个工艺干什么，只要你打听出来就好，记住，你只有三天时间，三天时间一过，你跟我们合作的事情就会被掀出来，到时候你就死定了。”
<br/>　　“是，你放心，三天之内一定会给你答复的。”唐义信誓旦旦的保证着。
<br/>　　他眼下需要的是保命，倭人需要的是消息，同时也不想暴露自己，双方很快达成一致。
<br/>　　……
<br/>　　一夜过去，第二天天一亮，唐义如往常一样离开家，带着自己的饭盒向城外的科学院走去，几个倭人隐藏起来暗中观察着他的动向，没发现他去县衙报官，这才长长的松了口气。
<br/>　　但倭人青年最后还是没有掉以轻心，在离开咸阳城后看着唐义进入科学院里面之后，才在隐于外面的荒野之中，等待着夜晚的降临。
<br/>　　而他们不知道的是，那唐义在进入了科学院之后，脚步停都没停便去了科学院的保卫部，将昨天晚上发生在自己身上的事情全都说了一遍，重点还强调了一下几个倭人的衣着。
<br/>　　倭人青年并不知道，他们在唐义的眼中其实就是一群傻X。
<br/>　　要知道，想打大唐科学院主意的人可并不是只有倭人，其他一些国家甚至大唐国内一些自己人也都在打科学院的主意。
<br/>　　在这样的情况下，大唐科学院又怎么可能没有任何一点应对措施，如果科学院里出去的人随随便便就可以被人威胁而变节，科学院估计早就完蛋了。
<br/>　　……
<br/>　　程处弼，程咬金家的老三，大唐科学院的安全保卫工作就是由他来控制。
<br/>　　这程家老三在听到唐义所说的事情之后，并没有急着出去抓人，反而是沉默了一会儿然后才缓缓说道：“唐义，你觉得他们晚上还会去找你么？”
<br/>　　“应该会来找我，不过时间并不确定，他们现在已经知道了我的住处，随时都可能会来。”唐义寻思了一会儿说道。
<br/>　　“我打算让你将计就计，引出后面更大的目标，只是这样做你也会有一定的危险，你好好考虑一下要不要接受，如果你不想冒险，我们也可以想别的办法。”
<br/>　　“我觉得暂时没有问题，至少在他们拿到想要的东西之前，他们不会对我怎么样。”唐义低头想了想，然后肯定的说道：“那就按照程将军的意思办吧，我没有意见。”
<br/>　　“好，既然这样，晚上你就找一些能说的说一些，让住不要透露太多东西，要让他们觉得你还有用，只有这样你才能安全。”程处弼似乎还有些不放心，再次叮嘱了一遍，才放心的让唐义离开。
