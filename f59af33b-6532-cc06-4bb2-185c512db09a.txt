        童筱颖深吸了口气，只身朝傅夜沉的越野车走了过去，然后拉开了副驾驶座的车门坐了进去。
<br/>
<br/>        傅夜沉的脸色很不好，有点惨白不说，两只眼睛眼窝很深，还有两圈黑眼圈，就连下巴也长出了些青色的胡渣。
<br/>
<br/>        “大哥，我……”
<br/>
<br/>        “我累了，想睡觉。”傅夜沉看都没看童筱颖一眼，就直接打断了她的话。
<br/>
<br/>        他将驾驶座的座椅放倒，怀抱双臂，闭上了眼睛就躺了下去。
<br/>
<br/>        车内的气氛格外凝重，透着几分寒意。
<br/>
<br/>        “大哥，你带身份证了吗？我们去开房吧！”童筱颖侧着身子，双眸直勾勾地盯着傅夜沉。
<br/>
<br/>        他一定一宿没睡，在这车上睡，肯定睡得不舒服。正好，这里就是酒店，上去开间房睡，总比在这车上睡要好。
<br/>
<br/>        “想跟我开房？好！成全你！”傅夜沉倏然睁开了黑眸，坐起身来后，直接打开了驾驶座的车门。
<br/>
<br/>        他从车头绕过，来至副驾驶座，打开车门后，霸道地将童筱颖从车上拉了下来。
<br/>
<br/>        童筱颖任由傅夜沉拽着手腕，重新回到酒店里。
<br/>
<br/>        他在前台做了登记交了定金拿了房卡，紧紧地抓着她的手腕，坐电梯去了顶楼总统套房。
<br/>
<br/>        一路上，童筱颖沉默不语，就连手腕被他硬生生地抓痛了，她也没作声。
<br/>
<br/>        傅夜沉俊脸阴沉，拿房卡刷开门后，拉着童筱颖刚走到玄关处，房门自动关上时，他便开始立即进入主题，加快了步伐，直奔主卧，并将童筱颖一把甩到了床上。
<br/>
<br/>        童筱颖讷讷地看着傅夜沉，他三下五除二地脱了上衣，露出了精壮的上半身，皮带刚巧卡在他的人鱼线上，性感不失健硕，身段完美得能令女人窒息。
<br/>
<br/>        当他的身子压下来的时候，男性滚烫的身躯像火一样燃烧着她。
<br/>
<br/>        他不再像以前那么温柔，大手直接扼制她的脸颊，将她的下巴微微抬起，低头攫住她的甜美，撕扯着她身上的裙子，粗暴地将她吻了个彻底。
<br/>
<br/>        洞房花烛夜那晚，她喝了酒，昏昏欲睡，他没有趁人之危。
<br/>
<br/>        现在，她是清醒的，不是吗？
<br/>
<br/>        他结实的手臂揽住她的腰，另一只手重重地按住了她微颤的肩头，令她肩骨生出一丝痛意。
<br/>
<br/>        她的连衣裙被他半褪在腰间，身上那些新旧吻痕更是令他触目惊心。
<br/>
<br/>        呵！他到底爱上了一个什么样的女人？
<br/>
<br/>        昨天早上，他天还未亮就早早起床，去了厨房，想跟厨师现学现卖，给她做爱心早餐。
<br/>
<br/>        结果，当他端着自己做的生滚粥和包点回房间的时候，却被女佣告知，她已经出门了，还不让司机送她。
<br/>
<br/>        她出门，手机和钱包全都没带。
<br/>
<br/>        他在家等了她一整天，中途接到了刘婶的电话。
<br/>
<br/>        刘婶得知他和她已经回临海城的傅家大庄园，当着爷爷和母亲的面前拜堂成亲了，刘婶一时高兴说漏了嘴：“大少奶奶想通了就好，不枉费我一番苦心地告诉大少奶奶，大少爷您这三年来是如何思念大少奶奶的。”
<br/>
<br/>        对，刘婶把所有的事情，都告知了她，所以，这才是她突然对他改变了态度的真正原因吗？
<br/>
<br/>        因为感动？
<br/>
<br/>        感念他对她的相思之苦？
<br/>
<br/>        所以，她才跟他说，想要试着当好他的太太！
<br/>
<br/>        傅夜沉心如刀割地痛着，终于按捺不住，给弟弟傅昼景打了通电话。
<br/>
<br/>        所以，那通电话，就是他等到的结果。
<br/>
<br/>        三更半夜的时候，阿景再次给他发了信息，附上了他俩的“床照”，还告知了他地址。
<br/>
<br/>        就连开房的门牌号，都跟他说得一清二楚。
<br/>
<br/>        真讽刺！
<br/>
<br/>        他抱着她睡的时候，她总是在睡梦中皱着眉头，但是到了阿景的怀里，她却是笑着的。
<br/>
<br/>        “童筱颖，你给我滚！从今天开始，我傅夜沉不再需要你了！”他爱累了，爱到身心都很累。所以，他说这话时，有气无力，只能怒吼了出来。
<br/>
<br/>        童筱颖睁大了眼睛，看着傅夜沉额冒青筋，一句话也说不上来。
<br/>
<br/>        傅昼景说得对，她一点都不了解大哥，而且大哥其实打心底里就没有信任过她。
<br/>
<br/>        或许，他对她的爱，只是因为爱而得不到才变得如此执着罢了。
<br/>
<br/>        他拿什么去相信，她会在这么短的时间内去爱上他？
<br/>
<br/>        昨晚，傅昼景发给他的信息里，还对他说了其他话。
<br/>
<br/>        正是因为那些话，才让傅夜沉彻底醒悟过来。
<br/>
<br/>        是他对自己太自信了……
<br/>
<br/>        不，是他自己太傻了……
<br/>
<br/>        傻到以为，这段时间她所有的主动，是因为爱上了他。
<br/>
<br/>        其实，不然。
<br/>
<br/>        她对他，不是爱。
<br/>
<br/>        完全不是爱……
