　　第341章
<br/>　　如今程大雷拥有的山寨已经达到三座：二级本落玉寨，三级本蛤蟆寨，四级本琴川关。
<br/>　　一杆大旗已经矗立在琴川关头，依然是一只面目丑陋的蛤蟆，贪婪的窥视着天上的月亮。
<br/>　　这幅图是李行哉亲手所绘，毕竟是帝国王子，程大雷也沾他点气运。
<br/>　　事实上，程大雷实际控制的山寨更多，以云中龙为首的十八家首领都听程大雷指挥。只不过程大雷目前只有指挥权，没有拥有权，这些山寨还不属于他。
<br/>　　当然，目前三家基地已经够了，程大雷暂时没有再开分基地的打算。落玉寨可以用来爆骑兵，蛤蟆寨可以爆水兵，琴川关各样建筑齐全，住的也舒服，程大雷就用来当大本营了。
<br/>　　他打开了系统界面，查看自己的信息。
<br/>　　姓名：程大雷（名扬四方的绝世山贼）
<br/>　　技能：雄辩、阿飞快剑、身无彩凤双飞翼、心有灵犀一点通、刑天三斧
<br/>　　隐藏属性：无
<br/>　　何谓绝世？顶级万里无一，绝世屈指可数，放眼整个天下，程大雷都是有数的高手。
<br/>　　随着等级提升，程大雷的气力，速度，反应又到一个新的层次。各种战斗经验、对敌招式涌入程大雷的身体，不仅是大脑记住了，更多是肌肉记忆，临阵对敌时，靠的就是下意识的身体本能。
<br/>　　以前程大雷使用身无彩凤双飞翼还有体力限制，飞不了太远。但现在的他已经可以随意使用这种超负荷技能。
<br/>　　当然，失去隐藏属性还是蛮遗憾的，只能以后找机会再说。
<br/>　　大魔王三斧升级成为刑天三斧，不应只是名字改了，威力也提升了很多。
<br/>　　刑天是怎样的主儿，那是脑袋砍掉，还能以乳为目，以脐为口，抄家伙和天帝干的存在。这三板斧敢挂他的logo，想也知道不可能太弱。
<br/>　　手扶过鬼面斧的漆黑斧面，似乎有某种冰凉的东西传进程大雷身体，让他觉得疼痛，却又隐隐有些兴奋。
<br/>　　程大雷忽然心中一动，拎着大斧跃到院子中央，手握大斧，在空地上亮了个架门。
<br/>　　阳光将他的影子打在地上，一股愤怒从心中升起，程大雷挥斧与影而战，出手并不快，招式也称不上复杂，可每一斧砍出，却裹夹着澎湃力量。
<br/>　　他打得越来越快，身周影子也越来越多，这些影子宛若成了实体，程大雷陷入四面包围中。愤怒和仇恨在胸腔内积蓄，不吐不快，他有一种冲动，似乎想要毁了目睹的一切。
<br/>　　影子打在墙上，程大雷挥斧砍了过去，恍惚中，对面有个黑影也持斧砍来。一样的招式，一样的速度，程大雷口中发出一声怪叫。
<br/>　　“死啊！”
<br/>　　轰隆！
<br/>　　围墙倒塌，向程大雷砸来，顿时将程大雷砸了个头破血流。
<br/>　　徐神机此刻正好从墙外经过，看到这一幕吓了一跳。
<br/>　　“大当家，你怎么了，没事砸墙玩？”
<br/>　　额头的伤口让程大雷清醒过来，他用毛巾捂着头，摆摆手道：“没什么，没什么，让我缓缓。”
<br/>　　程大雷的确有些后怕，刚才似乎体内装了一个别的灵魂，一个想要杀死一切，毁灭一切的凶魂。
<br/>　　传说有个地方叫鹰愁涧，因为水面太清，苍鹰飞过时以为自己的影子是敌人，于是愤怒冲下，然后被淹死了。
<br/>　　传说有个人叫李元霸，恨天无把，恨地无环，打遍天下无三合之将。最后举锤骂天，要与天一战。大锤扔出去，老天没搭理他，大锤掉下来把他自己砸死了。
<br/>　　刚才在程大雷身上发生的事情与此差不多，他差一点死在自己手中。
<br/>　　“大当家，你喝口水缓缓。”徐神机端着茶过来：“刚才你可是把我吓着了，你猜我刚才看到你以为看到了什么？”
<br/>　　“什么？”
<br/>　　“像一个刚从地狱爬出来，不想死的怪物。”徐神机将茶送到程大雷手中。
<br/>　　程大雷一怔，脊背被冷汗浸湿。怪物么，刚才的感觉真像体内藏了一头怪兽，想要控制他，占有它的力量，靠的不仅仅是身体的强壮，也不是大脑的智慧，而是心的强大。
<br/>　　程大雷抿了口热茶，轻轻舒了口气：“胡说八道什么，我是看这堵墙太碍眼了，准备砸掉他透透气。”
<br/>　　“你刚才可是差点把自己埋在墙下？”
<br/>　　程大雷瞥他一眼，道：“你跑来做什么，大白天的不做事，专门过来和我逗贫。”
<br/>　　“什么呀，我也是日理万机的好不好，是咱们这儿发生了件怪事，我专门来通知你的。”
<br/>　　“什么样的怪事啊？”
<br/>　　徐神机领着程大雷来到城门口，只见这里已经聚集着不少人。在众人中间围着一头黑牛。
<br/>　　就连程大雷看到，也不免轻咦一声。关键是这头牛太丑了，浑身漆黑，顶着两只怪角，模样看起来狰狞凶恶，又丑又凶。
<br/>　　更出奇的是，他背上有鞍桥、缰绳，上面还背着一副盔甲。
<br/>　　徐神机在一旁道：“大当家，这头黑牛凶得很，关鱼和秦蛮都使过了，上去就摔，大家都比着看谁能降服这头黑牛。所以我特意把大当家唤来……”
<br/>　　“让我也丢丢人？”
<br/>　　“哪里话，我不是为了让大当家人前显贵，傲里夺尊么。”
<br/>　　程大雷白了他一眼，目光搁在这头黑牛身上，忽然心中一动，隐隐想到什么。
<br/>　　系统大大……该不会真这么扯吧。说是奖励一顶级座骑，但没说座骑一定是马。
<br/>　　程大雷在青牛山骑过瘦马，在草原骑过驴，难道在这里就该骑牛了？
<br/>　　程大雷也不敢吐槽什么，系统没给他搞出乌龟、野猪什么的，程大雷已经应该庆幸了。如果自己真成了什么骑猪大将，以系统的尿性也不是做不出来。
<br/>　　又上去几人，想要降服这头黑牛，却都没黑牛撂倒了，狼藉一片。
<br/>　　“呔，都退下，别上去裹乱。”
<br/>　　程大雷喝了一声，大家目光都看向他，各自退后，将位置留给程大雷和这黑牛，想要看看程大当家如何降牛。
<br/>　　程大雷清了清嗓子，忽然冲那黑牛大喝一声。
<br/>　　“呔！兀那畜生，还敢撒野，见到本寨主，还不乖乖过来！”
<br/>　　
<br/>(本章完)
