　　第155章
<br/>　　似乎不怎么有人注意程大雷拉风的姿势，血并不能令穷凶极恶的人却步，他们一窝蜂的冲过来，口中大吼：“杀呀，杀！”
<br/>　　“特么的！”程大雷大骂：“挡我装X者死！”
<br/>　　剑出，直来直往，穿喉。
<br/>　　同样的位置，同样鲜红的血，同样死不瞑目的眼神。那空洞的眼神写满恐怖，似乎在问：世间可还有比这更快的剑！
<br/>　　若程大雷知道他心中的问题，一定会心怀善意的告诉对方：其实有的。
<br/>　　因为这只是第一式，而程大雷只是初学者。
<br/>　　剑出，则例不虚发，程大雷一连刺出六剑，剑剑封喉。
<br/>　　可没人知道程大雷的痛苦，他并没有驾驭此剑的能力，每刺出一剑都给身体带来巨大痛苦。如同在顶尖舞者鞋内撒了一把图钉，一支酣畅淋漓的舞蹈过后，舞者的双脚也同样鲜血淋漓。
<br/>　　最后一剑，程大雷含而未发，血顺着剑身流淌，程大雷抬眉问，眼神比剑意更寒。
<br/>　　“汝，谁可接我一剑！”
<br/>　　秦蛮那里的战斗也快接近结束，他同样观察着程大雷这边的情形，因程大雷的剑法震撼。
<br/>　　仗着体力之便，他一枪结果熊老大，立时来到程大雷身边。
<br/>　　熊老大的喽啰见老大死了，立刻做鸟兽散，秦蛮也不去追赶。
<br/>　　“大当家，你何时学会如此了得的剑法？”秦蛮。
<br/>　　程大雷抬眉：“难道你以为神仙传艺，只传了我斧法？”
<br/>　　“……”秦蛮。
<br/>　　“哎呦，大当家，你怎么了，大当家你快醒醒。”
<br/>　　程大雷，晕。
<br/>　　等他再醒来，却也不知昏睡过去多久，看周围情形已回到蛤蟆寨。还是感觉身体浑身疲软，看来这一剑消耗是极大的，以后能不动用的情况下，还是不要轻易动用。
<br/>　　秦蛮、徐神机、刘悲闻听程大雷醒来后，赶忙过来。
<br/>　　“我昏了多久？”程大雷问。
<br/>　　“足足一天一夜。”
<br/>　　“这么久！”程大雷吓了一跳，亏自己只使了六剑，若是七剑使尽，估计也报销了。
<br/>　　“大当家，这剑法是江湖武艺，在战场上没多大用处的。”秦蛮。
<br/>　　程大雷点点头，示意自己明白。战场杀敌，还是斧头，大枪，大刀这种势沉力猛的兵器来得痛快。
<br/>　　“我昏过去这段时间，发生了什么事？”
<br/>　　“幽州王把熊老大的尸体拉走了，听说幽州王对大当家赞不绝口，还差人赏赐许多东西。”徐神机
<br/>　　“送来了什么？”
<br/>　　“白银百两，玉器十件，绸缎五十匹，还有一些家具。”
<br/>　　“都是没用的东西。”
<br/>　　“杀了熊老大，咱们蛤蟆寨的名声可是传遍了幽州城，现在谁不知道咱们蛤蟆寨的名声。怎么看上去，大当家有些不开心？”
<br/>　　“狗咬狗，无论死的是那条狗，永远是逗狗的最高兴，我们有什么可开心的。”程大雷揉揉发疼的脑袋：“若不是现在拳头不大，牙齿不硬，我何必受人驱使……只愿熊老大的尸体，能堵几天杨龙停的嘴吧。”
<br/>　　时间一日一日过去，终于，这个漫漫长的冬天也接近尾声。天气转暖之后，建设山寨的工程进度加快，眼看就要完成全部工程。
<br/>　　冰雪初融，从山寨底部的悬崖上冲出一道瀑布，水顺着深挖后的河道流出去，以河流为分界线，左边是房屋，右边是农田。
<br/>　　包括：点将台，民居，医馆，点兵场，仓库，铁匠铺，木坊，制衣铺。
<br/>　　水顺着谷口流出，与绕着山谷而过的河流汇合，成为一道天然的护城河。在谷口外建造一座翁城，城墙上能容五人并排行走，城墙口搁着投石机，以及仿造的重弩，上面常备弩箭和投石。
<br/>　　如果熊老大活着走入蛤蟆寨，知道程大雷每天都在准备些什么，便该知道程大雷不屑与任何人做走狗。
<br/>　　这一日是安装城门的日子，算是工程的最后一道程序，极有象征意义。
<br/>　　在大家的号子声中，沉重的木门被推上城墙。
<br/>　　到此时，程大雷要的山寨全部完工，这座山寨才真正称得上一夫当关，万夫莫开。
<br/>　　轰隆！
<br/>　　天空中一道闷雷响过，有凉意落在脸上，众人伸出手去，发现落在掌心的是雨。
<br/>　　时隔一年，老天终于舍得给人间降下一场雨，天地寒气也为之一清。
<br/>　　程大雷抬起头看着头顶的彤云，口中呐呐道：“惊蛰了。”
<br/>　　惊蛰日，春雷动，万物复苏。
<br/>　　春天到了。
<br/>　　这场雨淅淅沥沥下了好久，积雪融化后，顺着悬崖流下来，谷内的河流竟有几分湍急的意思。
<br/>　　程大雷避雨窝在屋子里，看着窗外的雨景，有几分愁眉不展的意思。
<br/>　　“大当家，山寨都建好了，这里比以前安全多了，你怎么还不开心，要不奴家给你唱个小曲吧？”柳芷道。
<br/>　　“随便，随便。”
<br/>　　程大雷有些心不在焉，他不开心的原因是，山寨建设完成后，自己并没有收到系统提示。没有提示就代表自己还没有把山寨升到三级，因此也没有系统的附加属性。
<br/>　　究竟差了什么呢？
<br/>　　系统曾奖励程大雷一份三级山寨建设图纸，上面还有三级山寨的升级要求。现在程大雷正一样一样的比对着。
<br/>　　“点将台，练兵场……这些自己都有了，人口要求一千，如今也满足了。不，不，问题出在这里，还差一项……”
<br/>　　三级山寨，还需满足一个条件：顶级人物3。
<br/>　　秦蛮、赵子龙这是山寨唯一两名顶级人物，以一当百绝不在话下，本以为有这两人就够用了。谁料升级山寨，必须拥有三名顶级人物。
<br/>　　可这第三位，自己往那里去找呢？
<br/>　　“罗衣叠雪，宝髻堆云。樱桃口，杏脸桃腮；杨柳腰，兰心蕙性。歌喉宛转，声如枝上流莺；舞态蹁跹，影似花间凤转”
<br/>　　正此时，耳边响起软糯的甜声，扰了程大雷的思绪。他回过神见这声音是柳芷口中发出的。
<br/>　　“你怎么会这些？”程大雷吓了一跳。
<br/>　　“本来就会呀。”柳芷有几分得意，却又低下头来：“青楼学的。”
<br/>　　这场春雨下了三天，只等雨过天晴，已算是春到人间。
<br/>　　然后程大雷得到消息，杨龙停挥兵五万，兵发黑石城，喊出的口号是：别个不问，只诛薛四。
