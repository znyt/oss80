　　微微眯起了眼，一股浓郁的杀戮气息从刘哔身上显露出来。
<br/>　　这是他屠戮了无数生命，方才养成的滔天煞气，虽然境界上教皇可能略高一些，但两人之间的气势都无法压制对方。
<br/>　　如果是一个等阶不高的能量体误入进来，估计瞬间就会被两股气势影响，而引发自身一系列的问题。
<br/>　　高手对决，往往是从气势上开始的。
<br/>　　“教皇先生，是否对我提供的丹药有什么不满意的地方？”剑拔弩张之下，刘哔淡然开口，扰乱了两人之间压抑的气氛。
<br/>　　他不想动手，白日庭这些人，对他还有很大的作用。
<br/>　　“恩，很满意，我想，我们大概可以达成一种合作关系。”沉默了一会，教皇收回了外漏的气息，脸上露出了笑容。
<br/>　　方才出手，也只是一个试探，他没有太大的把握留下刘哔，自然也不会选择战斗。
<br/>　　当然，如果刘哔只是一个外强中干的纸老虎的话，他自然也不会客气。
<br/>　　毕竟这个丹药太过于重要，如果有可能的话，还是要掌握在自己手中。
<br/>　　被别人控制着，他不放心。
<br/>　　“那么教皇先生，是不是也要兑现自己先前的诺言了。”
<br/>　　见到教皇收手，刘哔自然也不会继续相逼，两人又像先前一般，有说有笑的畅聊起来了。
<br/>　　“当然。”教皇浅笑了一声，坐会先前的椅子上，开口向刘哔娓娓道来：“CCG几乎是和修家族一手缔造出来的，而独眼喰种的存在，也并不像外界传闻一样，是从芳村功善开始的。”
<br/>　　“早在一百多年前，东京就出现过一次独眼喰种，当时它的强大，举世震惊，人类根本无法阻挡，那是第一代独眼之王。”
<br/>　　“受不了独眼之王的侵略，终于有一个人站出，联合了当年最为强大的三个家族，组建起了CCG。”
<br/>　　“但所有人不知道的是，当年联合组建CCG的发起人和修，就是当年战乱的来源，独眼之王。”
<br/>　　叹了口气，教皇的语气有些无奈，回忆起CCG的历史，他也是有些唏嘘。
<br/>　　“谁也不知道独眼之王想要干什么，它亦正亦邪，不像喰种，更不像人类，而这，也给它提供了一层完美的伪装。”
<br/>　　“独眼之王一边猎杀喰种，一边和人类结合，剩下更为强大的子嗣，操控者战争的均衡，所有的人，都被和修一族玩弄于鼓掌之中，就连当时一起组建CCG的其余三个家族，都因为各种原因覆灭了。”
<br/>　　“后来，芳村功善的孩子，独眼枭出现了。”
<br/>　　“他们终于发现了不对，一个新生的独眼喰种，自然撼动不了他们家族的地位，但他们惧怕的是，这种制造独眼喰种的方式流传出去。”
<br/>　　“为了稳固自己的地位，他们开始动用更加血腥的手段，喂养再生能力强悍的喰种，摘取这些喰种的赫包，来制造更为强大的战士。”
<br/>　　“另一边，他们开始培养CCG的特种搜查官，来追杀那些和人类结合的喰种，譬如，我们。”
<br/>　　“这么多年过去了，谁都不知道和修家族发展到了什么地步，或者说，几乎所有的喰种和CCG都被这个家族掌控着。”
<br/>　　“而你想要了解的那些喰种势力，除了小丑组织之外，其余的并不入流，而这些组织，也大都在和修家族的掌控之下。”
<br/>　　“小丑组织的领导人，和修旧多宗太，已经被你杀了，现在的小丑组织，应该是由另一名SSS级喰种，萝玛来领导者。”
<br/>　　“但实际上，小丑组织对和修家族来说，只是宗太刷小孩子脾气，制造出来的一个玩物罢了，等待它的命运，要么是效忠，要么，抹杀。”
<br/>　　“你想要对付和修一族，就等同于和世界为敌，最重要的一点是，独眼之王，或许还没死。”
<br/>　　“现在，你还想，和这样的家族对抗吗？”
<br/>　　听完教皇的诉说，刘哔陷入了沉默，他知道和修一族很强，但从来没有想到，会强大到这种地步。
<br/>　　只是一个旧多宗太，他应对起来就十分吃力，更不要说面对不知道活了多久的初代独眼之王了。
<br/>　　这个世界，终究还有太多他所不知道的地方，有些事情，他也想的太过于简单了。
<br/>　　或者说，是实力快速的提升，导致他心态，现在有点膨胀了。
<br/>　　如果不是教皇的提醒，他在一无所知的情况下，天知道会出现什么样的危险。
<br/>　　这个家族，就如同教皇所说，根本不是他们能够招惹的存在。
<br/>　　至少，现在不行。
<br/>　　“谢谢，这些消息很有帮助，我觉得，教皇先生提出的合作意见不错，我们可以尝试一下。”
<br/>　　沉默了良久，刘哔抬起头来，嘴角露出一抹自信的笑容。
<br/>　　没有任何事情能够难倒他，对手越强，才更加有趣，不是么。
<br/>　　至少，短短几分钟的时间里，他已经制定好了一个粗略的计划。
<br/>　　“但我说的合作，只是想像你收购你给我的那种药丸，不知道，你说的合作，和我说的合作，是不是一个合作。”
<br/>　　教皇挑了挑眉，眼中露出颇感兴趣的色彩。
<br/>　　他已经将事情说的这样通彻了，可这家伙，还是一副不死心的样子。
<br/>　　究竟是有什么样的底牌，才让他有如此大的底气。
<br/>　　“我会按月给你们提供十颗丹药，来辅助你们白日庭成长，免费。”刘哔嘴角挂着笑容，提出了一个令教皇十分心动的条件。
<br/>　　十颗药丸，那代表的，就是十个拥有正常人类寿命的半人类，也就是十个接近或者超越SSS级的战士！
<br/>　　所有的半人类，天生都是战士，即使天赋一般，也足以匹敌SSS级。
<br/>　　但他知道，这口食，不可能是白吃的，他只要吃进去，就一定要从别的地方吐出来。
<br/>　　“我唯一的要求，就是等我发动对月山家战争的时候，你们会站在我的一边。”果不其然，刘哔接下来的话语，立即给他泼了一头凉水。
<br/>　　这个家伙，还是执着的想要对和修家族发动战争吗。
<br/>　　可是，他明明已经说明了和修家族的强大了，他不认为刘哔有机会胜出。
