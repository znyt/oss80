　　“嗯？怎么会有一个男人，你是谁？”
<br/>　　冰漓长老这时才注意到楚风眠，看着楚风眠站在寒月漓身边，不由的眉头一皱道。
<br/>　　“你是我们北邙圣域的弟子？”
<br/>　　冰漓长老看着楚风眠，有些不确认的开口道。
<br/>　　在这北邙圣域中，应该不会有其他人才对。
<br/>　　不过楚风眠的样子，她却是没有什么影响，极为陌生。
<br/>　　“师尊，他就是我的夫君，楚风眠。”
<br/>　　寒月漓出声，解释道。
<br/>　　“之前便是因为他被龙党的人，威胁去了生死台，我才会贸然离开，还请师尊原谅。”
<br/>　　“你的夫君？”
<br/>　　听到这句话，冰漓长老脸色一变，便是认真了许多。
<br/>　　一双眼睛，扫视着楚风眠，似乎是要将楚风眠上上下下，都给打量一遍。
<br/>　　寒月漓也不是没有在冰漓长老面前，提过楚风眠的事，但是那个时候在冰漓长老看来，这就是一个笑话。
<br/>　　只是在北域中相遇，来到北邙学院之中，地位天差地别，这关系，迟早会断的，所以也就没有放在心上。
<br/>　　如今楚风眠的突然出现，也是让冰漓长老吓了一跳，楚风眠能够来到这北邙圣域，也就是意味着，楚风眠的实力，已经是达到神力境极限的。
<br/>　　“你的实力，已经是达到了神力境极限？”
<br/>　　再三确认一下，冰漓长老的眼神中还是露出了几分惊讶。
<br/>　　冰漓长老也突然想了起来，在之前收徒大会的时候，她也曾经见到过楚风眠。
<br/>　　不过那个时候，楚风眠甚至就连神力境都没有踏入，只是一个御风境的小家伙，在她看来，只是一个运气不错的小子，走了好运才通过了北邙学院的考核，根本没有被她放在心上。
<br/>　　可如今，短短几个月的时间，楚风眠居然就拥有了如今的实力，还晋升成为了种子弟子，也是让她极为惊讶的。
<br/>　　这种速度，简直惊为天人，除非是得到了极大的奇遇，不然不可能进展如此之快。
<br/>　　“徒儿你说，刚刚他被龙党的人，带到生死台上去了？”
<br/>　　冰漓长老打量了楚风眠一眼，开口道。
<br/>　　“龙党的人，算不得什么，不过龙未央还有着几分实力，不好对付，这次月漓可以救你，不过你要小心，以龙党的性格，迟早还会找你麻烦的。”
<br/>　　冰漓长老忠告道。
<br/>　　在她看来，也应该是寒月漓出手，救下了，才能让楚风眠全身而退。
<br/>　　不然以龙党的作风，楚风眠肯定会被他们灭杀在生死台上的。
<br/>　　听到冰漓长老的话，寒月漓笑着开口道。
<br/>　　“师尊，不用担心，从今日起，已经没有龙党了。”
<br/>　　“没有龙党了？什么意思？你将龙党的人，全部灭杀了？”
<br/>　　冰漓长老听到寒月漓的话，一愣，随后开口道。
<br/>　　“不过无妨，龙未央的背后，并没有什么人，只是他跟龙族，有些关系，未来很有可能龙族的人，会来找你麻烦。”
<br/>　　冰漓长老还以为是寒月漓杀了龙未央，思考这。
<br/>　　“龙未央的确死了，不过杀了他的人，却不是徒儿，是他。”
<br/>　　寒月漓指着楚风眠，突然开口道。
<br/>　　“龙未央，不是你杀的？那是谁？”
<br/>　　冰漓长老一愣，随后看到寒月漓所指的方向。
<br/>　　“什么？你说龙未央是他杀的？”
<br/>　　冰漓长老开口之际，也是满脸震惊的看着楚风眠。
<br/>　　“是你，杀了龙未央？”
<br/>　　冰漓长老的话中，也是充满了不可置信。
<br/>　　龙未央，可是龙党领袖，在这北邙圣域中，能够真正击败龙未央的人，都是寥寥无几，更何况是将龙未央，直接斩杀。
<br/>　　冰漓长老本来还以为是寒月漓做的，以寒月漓现在的实力，做到这一点，并不稀奇。
<br/>　　但是寒月漓居然说是楚风眠杀了龙未央，这让冰漓长老极为惊讶。
<br/>　　“不错。”
<br/>　　楚风眠点了点头，大手一挥，在他的面前，便是出现了水镜。
<br/>　　这水镜，正是楚风眠轰杀了龙未央的景象。
<br/>　　眼见为实，楚风眠也懒得在多做任何解释。
<br/>　　一直是看到这水镜之中，龙未央最终死在楚风眠的手中的景象，冰漓长老的才缓过神来。
<br/>　　楚风眠在其中表现的实力，实在是太可怕了。
<br/>　　尤其是轰杀龙未央的时候，好像是龙未央在楚风眠的手中，都是毫无还手之力，这种实力放在北邙圣域中，也是真正的巨头了。
<br/>　　冰漓长老看向楚风眠，这次她的目光，便是没有之前的那般冷淡，倒是缓和了许多。
<br/>　　“很好，月漓看上的夫君，果然不会是一个废物，能够拥有这样的实力，在北邙圣域中，也足以立足了。”
<br/>　　“对了师尊，师尊这次急忙叫我回来，可是为了司空家的事？”
<br/>　　寒月漓突然开口道。
<br/>　　“徒儿已经拒绝无数次，这司空弘还是不死心，提亲的事，还要师尊帮忙出面拒绝才行。”
<br/>　　“不错，为师这次叫你回来，正是司空家的事。”
<br/>　　冰漓长老缓缓的开口道。
<br/>　　“司空家的老祖，司空景胜，跟为师有着不错的交情，这一次他亲自带人过来，我也不好出面。”

<br/>　　楚风眠平静的开口道。

<br/>　　楚风眠的话，极为平淡，但是其中蕴含的话音，却是说到做到，极为坚决，话语中极为自信。

<br/>　　冰漓长老开口道。
