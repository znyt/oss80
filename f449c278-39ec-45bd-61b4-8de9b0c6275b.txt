　　听闻苏莫的话语，柴管事眼眸微眯，冷冷的道：“小辈，你是否觉得老夫脾气很好，可以随意糊弄？”
<br/>　　柴管事是真的有些怒了，他本就有些不信，苏莫会是帝级炼器师，现在苏莫的如此的姿态，更是让他认定了心中的想法。
<br/>　　他本还有一丝念想，或许苏莫真是万年难得一见的炼器奇才，原来还是自己多想了。
<br/>　　“不敢，晚辈说的句句属实！还请前辈给晚辈一个机会！”苏莫立刻摇了摇头，声音诚恳的说道。
<br/>　　柴管事闻言，略一沉吟，冷淡的道：“我现在就给你一个机会，若是你把握不住，便别怪老夫无情了！”
<br/>　　“前辈请说！”苏莫闻言心中微颤，急忙说道，他现在也是硬着头皮上了。
<br/>　　“那我问你，你会炼制什么样的帝级宝器？”柴管事问道。
<br/>　　苏莫闻言，立刻心念一转，开始搜寻脑中的炼器传承。
<br/>　　瞬息之间，他就找到了一种适合的帝级兵器。
<br/>　　“前辈，晚辈会炼制几种帝级兵器，其中最拿手的是一种战斧，此斧名为重渊斧，乃是帝级下品兵器。”苏莫沉声说道。
<br/>　　他说的这种重渊斧，乃是炼器传承中无数帝级兵器中，炼制方法最简单的一种。
<br/>　　“重渊斧？”柴管事闻言点了点头，他乃是尊级中品炼器师，自然知道重渊斧，以及其炼制之法。
<br/>　　“那你说说重渊斧的详细炼制之法。”柴管事说道。
<br/>　　苏莫点了点头，徐徐说道：“重渊斧的炼制，需要三百六十三种材料，分别是庚金、炎狼砂、天钢藤、天罗枝……！”
<br/>　　苏莫从重渊斧需要的材料开始讲起，然后是详细的炼制之法，以及其中需要注意的地方。
<br/>　　从头到尾，他将整个重渊斧的炼制过程，完全讲述了一遍。
<br/>　　柴管事听的暗暗点头，心中很是满意，因为苏莫的讲述，并不是简单的理论知识，而是将其中的细节，都讲述的非常详细。
<br/>　　能讲述的如此详细，头头是道，那就肯定是亲手炼制过这种兵器了。
<br/>　　卢云和秦不死两人，则是听的一愣一愣的。
<br/>　　秦不死心中震撼不已，苏莫居然有如此高深的炼器之术？
<br/>　　他虽然和苏莫认识的时间不长，但他对于苍穹世界可是很了解。
<br/>　　苍穹世界在最近几千年来，基本上没有出现过帝级炼器师，苏莫这么年轻，不仅实力在苍穹世界无敌，居然还有帝级的炼器之术！
<br/>　　秦不死发现，他对于苏莫的了解，真的真是冰山一角。
<br/>　　“这个小子，或许以后在天荒星域，也能掀起一番风浪吧！”秦不死暗道。
<br/>　　而高瘦炼器师卢云，则是有些傻眼了，他怔怔的望着苏莫，心中惊骇不已。
<br/>　　不会吧？难道真是帝级炼器师？
<br/>　　想到自己之前的态度，并且还收了苏莫的礼物，卢云心中有些忐忑。
<br/>　　这苏莫要是真成了公会的炼器师，那他这个炼器学徒，绝对没有好果子吃。
<br/>　　“这不太可能吧？”卢云满脸忐忑的想道，他还是不认为苏莫会是帝级炼器师。
<br/>　　毕竟，能说出来帝级兵器的炼制之法，并不一定能炼制出来！
<br/>　　以苏莫的年纪，就算在娘胎里就学习炼器，也不太可能这么短的时间内成为帝级炼器师。
<br/>　　而且，苏莫之前也曾给他一些帝级炼器师的知识，说是偶然所得，都只是理论而已。
<br/>　　“柴管事，我说完了，重渊斧就是如此炼制！”苏莫讲解完之后，向柴管事抱拳说道。
<br/>　　他面色淡然，眸中蕴含着自信，仿佛炼制这种重渊斧只是小菜一碟而已。
<br/>　　“不错！”
<br/>　　柴管事面上露出了一丝笑容，点头道：“你能将重渊斧的炼制之法，说的如此详细，各种细节没有丝毫遗漏，看来确实有些本事！”
<br/>　　言罢，柴管事略一沉吟，道：“这样吧！老夫就给你一个机会，你在公会中暂住几日，先恢复精神力，待精神力恢复之后再行考核！”
<br/>　　柴管事给了苏莫机会，这也是因为苏莫较为年轻，他才会给这个机会。
<br/>　　因为以苏莫的年纪，就算达不到帝级的炼器水平，能有皇级上品炼器师的水平，也已经非常难得了。
<br/>　　苏莫闻言心中大喜，急忙向对方抱拳道谢：“多谢柴管事！”
<br/>　　柴管事摆了摆手，转头看向卢云，道：“卢云，你去为这位小兄弟安排个住处，好生静养！”
<br/>　　“是！”卢云不敢怠慢，立刻应声。
<br/>　　随即，卢云便带着苏莫告别了柴管事，就在三楼，距柴管事住所不远的地方，为苏莫和秦不死安排了一间房间。
<br/>　　“兄台，好好休养吧！希望你能成功！”
<br/>　　在和苏莫告别之时，卢云面上勉强挤出了一丝强笑。
<br/>　　随即，卢云便满脸忐忑的离开了，他虽然言语说希望苏莫成功，心中却是正好相反，希望苏莫到时候炼制失败，被轰出炼器师公会。
<br/>　　房间中。
<br/>　　待卢云离开之后，苏莫和秦不死两人，均是一屁.股坐在了房间内的蒲团上。
<br/>　　“苏莫，我们紧张了几个月，现在终于能放松放松了！”秦不死满脸唏嘘之色。
<br/>　　四个月的逃亡，时刻心神紧绷，他非常的疲累。
<br/>　　“好好休息一段时间吧！在这里应该足够安全了！”苏莫深深的吸了口气，脸上亦是露出配备之色。
<br/>　　随即，苏莫打量了一番这个房间，非常的大，有练功室，还有炼器室。
<br/>　　或许是因为此城有‘巨人’的缘故，此城之中，无论是外界的建筑，还是炼器师公会中的房间，均是修建的高大无比。
<br/>　　休息了一个时辰之后，苏莫盘膝而坐，开始思索他以后的路。
<br/>　　细心的规划，是他能活到今天的原因，他必须提前计划好一切。
<br/>　　首先，巫族的两名尊者肯定就在城中，说不定就在炼器师公会之外，他无论如何也不能出去。
<br/>　　其次，他想留在炼器师公会，必须要尽快将自身的炼器水平达到帝级的地步，最起码他要能炼制出一件帝级兵器。
<br/>　　再然后，他要尽快了解这个世界，毕竟，他和秦不死刚从苍穹世界逃出来，对于这天荒星域，简直就是两眼一摸黑。
<br/>　　还有就是，他要打探清楚，巫族在天荒星域到底是什么样的存在？
<br/>　　他杀了巫族那么多天才子弟，可不会认为巫族不追究了。
<br/>　　就算他逃过了两名巫族尊者的杀机，日后肯定也会遭到整个巫族的追杀。
