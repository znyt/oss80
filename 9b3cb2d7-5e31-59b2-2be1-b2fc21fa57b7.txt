        两人在楼道里拉拉扯扯，何明旭怕影响到自己的形象，索性牵住诺筱颖的手，拉着她飞快地下了楼，去了一条人少的后街。
<br/>
<br/>        后街的路灯，只开了一盏，昏昏暗暗的一条路上，压根就看不清来往的行人。
<br/>
<br/>        诺筱颖甩开了何明旭的手，什么话也不问，就呆呆地站在墙边，一言不发。
<br/>
<br/>        何明旭双手插在裤袋里，低着头，深沉地叹了口气：“其实……我一直把你当妹妹看待。”
<br/>
<br/>        “以前不说，现在才来跟我说，是打算破罐子破摔了吗？”诺筱颖声音嘶哑地质问。
<br/>
<br/>        何明旭顿时皱起了眉头，不耐烦地回答道：“筱颍，不要把话说得这么难听！我两虽然是高中同学，又都是从农村里走出来的。但是，我是本科生，前途无量，你只不过是个大专生而已，将来也只能找到一份安稳微薄收入的工作。而且，这一年里，你拿钱给我花。只不过是想在我身上投资，将来等我有成就了，好娶你为妻，带你在这大城市里生活罢了！”
<br/>
<br/>        “你就是这么想我的？”诺筱颖难以置信地看着何明旭。
<br/>
<br/>        何明旭抬眸看着诺筱颖，反问：“不然呢？你本身就是这样的女孩子啊！”
<br/>
<br/>        “既然如此，我要你把我这一年来拿给你的钱都还给我！连本带利一起还给我！”诺筱颖气恼地向何明旭伸出手来。
<br/>
<br/>        何明旭瞥了诺筱颖一眼，瘪了瘪嘴，微微侧了侧身：“瞧瞧吧！我看人一点都没错。你果真是那种女孩子！算了，谁叫你是农村里出来的女孩子了！人没志气也就算了，眼里果真就只有钱钱钱，也是没办法的了。”
<br/>
<br/>        “你……”诺筱颖抬起手来，气得咬牙切齿，直指何明旭的鼻子，欲言又止。
<br/>
<br/>        何明旭顺手从裤兜里掏出一个钱包，将里面所有的钱都拿了出来，塞到了诺筱颖指着他鼻子的这只手中。
<br/>
<br/>        “筱颍，你配不上我。你的身份，你的学历，你的家世，统统都配不上我了。而且，以后，我若是娶了你，就相当于娶了你一家子。不仅要养你，还要养你吝啬的妈妈，残疾爸爸，连着你那个没出息的哥哥也要一起养着。这样一来，我将来会很累。这点钱你先拿着，就当是我给你的补偿。等以后，我研究生毕业了，找到好的工作后，再把钱连本带利的还给你。”何明旭一边叹息着，一边感慨地摇了摇头。
<br/>
<br/>        诺筱颖却直接将钱甩在了何明旭的脸上。
<br/>
<br/>        何明旭一脸懵然地看着诺筱颖，并未因为她如此放肆地侮辱他而生气，而是蹲下身去，一张一张地将钱给捡了起来。
<br/>
<br/>        诺筱颖怔怔地看着何明旭将捡起来的钱，重新塞回了他自己的钱包里。
<br/>
<br/>        何明旭还一脸不耐烦地碎碎念叨：“这是你自己不要的，算了，是我有错在先，你生气是应该的。”
<br/>
<br/>        “何明旭，我们高中同学三年，大学恋爱快满一年。我真的没想到，你除了会脚踏两条船以外，还会如此虚情假意！”诺筱颖声音哽咽道。
<br/>
<br/>        她真的没想到，她在为了他勤工俭学，每个周末都来照顾他的时候，他却瞒着她在学校里交了另一个女朋友，而且还同居了！
<br/>
<br/>        何明旭瞥了诺筱颖一眼，沉默了片刻，自己不知道说什么是好，于是闷不吭声地转身就走。
<br/>
<br/>        他何必在这里跟这个女人计较呢？
<br/>
<br/>        反正已经东窗事发，从今以后，他和她就不会再有什么交集。
<br/>
<br/>        其实，诺筱颖长得比林若琴漂亮，但是她的出身没林若琴好。
<br/>
<br/>        林若琴就是这临海城里的人，家里在临海城还有好几套房子，她爸妈在临海城里还有个关系很硬背景很强大的大人物关照着。
<br/>
<br/>        倘若，将来他娶了林若琴，他可以少奋斗十年。
<br/>
<br/>        倘若，将来他娶了诺筱颖，他那得辛苦一辈子！
<br/>
<br/>        如此对比一来，何明旭自然是嫌弃诺筱颖，喜欢林若琴。
<br/>
<br/>        诺筱颖看着何明旭渐行渐远的背影，直至黑暗将他吞噬，她的眼泪终于从眼眶里掉了下来。
<br/>
<br/>        最痛心的那一刻，她哭不出来，那痛过后，她终于哭出来了。
<br/>
<br/>        明明，爱情那么美好，为什么，他要将这美好的爱，给摧毁得一丝余温也不剩。
<br/>
<br/>        何明旭并非没爱过诺筱颖，只是，这样的爱，经不起现实的考验。
<br/>
<br/>        他不可能陪她一起跑马拉松式的柏拉图爱情，所以，为了能让自己更好，他必须抛弃她。
<br/>
<br/>        诺筱颖去了何明旭的寝室，拿回了自己的行李箱。
<br/>
<br/>        何明旭的室友章海昌和方浩博还很替诺筱颖担心，但看到诺筱颖脸上那坚强的笑容后，就安慰了诺筱颖几句，便和马智杰一起送诺筱颖去了长途汽车站。
<br/>
<br/>        临别前，马智杰深沉地说道：“筱颍，别怪我。你是个好姑娘，我不想你被瞒在鼓里。将来，你一定会再遇上一个你爱的同时又爱你的，对你好的男人。”
<br/>
<br/>        “就是！何明旭就是个渣男！你别太伤心！”章海昌随声附和道。
<br/>
<br/>        方浩博也忍不住插嘴安慰了一句：“何明旭渣，我们三不渣！这世界上，还是有好男人的。对爱情，千万别灰心。筱颍，你要保持初心啊！”
<br/>
<br/>        “嗯，谢谢你们。”诺筱颖欣慰地笑了笑。
<br/>
<br/>        这三个大男孩，算是对她重情重义了吧！
<br/>
<br/>        “看来，以后没人再帮我们三洗衣服喽！”章海昌又感慨道。
<br/>
<br/>        方浩博立即拍了一下章海昌的头顶：“你这个混球，衣服自己不会洗吗？”
<br/>
<br/>        诺筱颖忍不住噗嗤一笑。
<br/>
<br/>        “车要开了，筱颍，你上车吧！路上小心，注意安全。”马智杰叮嘱道。
<br/>
<br/>        诺筱颖点了点头，挥手和他们三告别后，转身上了大巴车，踏上回家的路途。
<br/>
<br/>        大巴车驶动后，慢慢地出了站。
<br/>
<br/>        方浩博一手绕过马智杰的肩头，一手挂在方浩博的肩膀上，三人并排走在了一起。
<br/>
<br/>        “智杰，你是不是喜欢筱颍啊？”方浩博一边走，一边问。
<br/>
<br/>        章海昌随之起哄，侧身拍了拍马智杰的胸膛：“肯定是喜欢。从认识筱颍开始，这一年来，你这暗恋的滋味，一定不好受吧！”
<br/>
<br/>        “你们想多了，我只不过是看不惯何明旭而已。”马智杰淡淡地回答。
<br/>
<br/>        方浩博和章海昌不约而同地相视而笑。
<br/>
<br/>        大家彼此，就当心照不宣了吧！
