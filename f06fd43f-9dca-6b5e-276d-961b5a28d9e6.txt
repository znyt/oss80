        而在这透着沐浴乳香气的水线下，连星儿已经忘却了昨天在自习室里所发生的那个遭遇。
<br/>
<br/>        直到洗完澡后，她穿上明司寒的衬衫，无意间看到自己脱下来的衣物，又回想起了昨晚自习室里的那件事。
<br/>
<br/>        连星儿莫名其妙地打了个冷噤，柳眉不经意间地紧蹙。
<br/>
<br/>        怔愣地看了那一堆衣物几秒钟后，她突然疾步上前，抱起那堆衣服从浴室里出来，径直走到厨房中岛台旁的垃圾桶前，直接将衣物给全部扔进了垃圾桶中。
<br/>
<br/>        她是一个女孩子，在那种情形下，差点被一个陌生男人玷污，不管她内心素质有多高，终究还是会留下一些无法抹平的心理阴影。
<br/>
<br/>        明司寒回来的时候，看到连星儿站在垃圾桶旁发呆，于是走过去看了看。
<br/>
<br/>        当他看到垃圾桶里有她昨天所穿那些衣服，心里顿时很是担忧地看着她。
<br/>
<br/>        “星儿。”
<br/>
<br/>        明司寒的呼唤声，忽然在身后响起。
<br/>
<br/>        连星儿身心一怔，回过神来后，一转身，撞了明司寒一个满怀。
<br/>
<br/>        明司寒下意识地给了连星儿一个安慰的拥抱，然后替她理了理粘在脸颊上那几绺湿漉漉的发丝，温柔地说道：“我去拿吹风机给你吹头发吧！”
<br/>
<br/>        连星儿没点头，也没吭声。
<br/>
<br/>        明司寒轻轻地拍了拍她的手臂，然后转身去拿来了吹风机。
<br/>
<br/>        随后，连星儿坐在了餐桌前喝粥，明司寒站在她的身后，拿着吹风机替她吹头发。
<br/>
<br/>        她的发丝很柔软，握在手中，就像丝绸那般顺滑。
<br/>
<br/>        明司寒有些笨拙地替她拨弄着长发，这是他有史以来，第一次拿吹风机给女孩子吹头发。
<br/>
<br/>        连星儿喝粥喝到一半，突然意识到一个问题，猛然从椅子上起身将明司寒吓了一跳。
<br/>
<br/>        “我把你弄疼了吗？”明司寒连忙关了吹风机，看着此刻已经转过身来面对着自己的连星儿，担忧地问道。
<br/>
<br/>        连星儿直勾勾地对上明司寒的目光，反问：“你从昨晚到现在，对我这么好，该不会是在同情我吧？”
<br/>
<br/>        “我同情你？！”明司寒顿时一脸茫然。
<br/>
<br/>        连星儿扬起下巴地接着说道：“你是不是以为我被……”
<br/>
<br/>        话到这里，她又欲言而止。
<br/>
<br/>        明司寒却忍俊不禁起来。
<br/>
<br/>        连星儿拧着眉头，撅了噘嘴，问道：“你笑什么？”
<br/>
<br/>        “我笑你傻啊！”明司寒咧嘴道。
<br/>
<br/>        连星儿眨巴着眼睛，不明所以然地看着明司寒。
<br/>
<br/>        明司寒索性放下手中的吹风机，伸手将连星儿揽入了自己的怀中紧紧地抱着。
<br/>
<br/>        倘若他抱着她的感觉没错的话，她此刻，隔着他的衬衫，里面可是什么也没穿。
<br/>
<br/>        “你放开我！”连星儿不满地皱着眉头推搡着明司寒的胸膛。
<br/>
<br/>        明司寒反而越抱越紧，咧嘴道：“从昨晚到现在，我一直都在想方设法地哄你开心，你却认为我是在同情你。你说你傻不傻？”
<br/>
<br/>        “你……”连星儿顿时气急败坏地抬起手来，直指明司寒的鼻子。
<br/>
<br/>        明司寒顺势握住她这只软软的小手，忽然一脸认真地接着说道：“我只是不想你对昨天的事情留下心理阴影，怕你睡不着做噩梦，怕你以后不愿再跟我接触。”
<br/>
<br/>        而这一刻，连星儿瞬间安静了下来。
<br/>
<br/>        明司寒微微一笑，放开了她的手，莫名其妙地开始脱起了上衣。
<br/>
<br/>        连星儿怔愣地眨了眨眼睛，看着明司寒那修长的手指，在他的衬衫上，将一粒粒纽扣松开，直到露出里面精瘦却健硕的胸膛。
<br/>
<br/>        她的目光顺着他的指尖往下，衣襟向两边拨开时，是一、二、三、四、五、六！六块肌理分明的腹肌。
<br/>
<br/>        “你要不要摸摸看？”明司寒微微勾起唇角。
<br/>
<br/>        连星儿一脸懵然地微微仰头，对上明司寒的目光。
<br/>
<br/>        明司寒索性直接拉起连星儿的手，按在自己的胸膛上。
<br/>
<br/>        “星儿，我只是希望你从昨晚的那件事的阴影里走出来。”他担心她因此害怕接触异性，所以以身试教，让她不再有所抵触。
<br/>
<br/>        连星儿明白了明司寒的用心良苦后，忍不住甜甜地笑了起来：“你当我连星儿是谁啊？就昨晚那种小事能把我吓到？不过……”
<br/>
<br/>        “不过什么？”明司寒狐疑地反问。
<br/>
<br/>        连星儿的指尖从他的胸膛上一路滑到了腹部的腹肌上，然后收手抱臂，一本正经地摸着下巴说教：“啧啧啧——你这健身不到位啊！我爹地都有八块腹肌，你才六块！”
<br/>
<br/>        “？！”明司寒顿时语塞，额冒青筋，“……”
<br/>
<br/>        像她这种没心没肺的丫头，他在这儿瞎担心个啥？
