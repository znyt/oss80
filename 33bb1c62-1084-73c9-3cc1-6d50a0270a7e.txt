　　不错！巫尘自爆了！
<br/>　　苏莫此行调查宏青璇下落的目标，已经完成了，所以，无论是本体还是分身，都无需再继续留在巫族了。
<br/>　　而且，巫玛和巫幽兰以及一众巫族天才的死，根本不可能瞒过去。
<br/>　　巫尘这个身份，本来就已经引起了大祭司怀疑，经此一役，若是再回到巫族，必会遭到灭顶之灾。
<br/>　　所以，苏莫索性让分身自爆了，因为分身根本没有逃出去的可能。
<br/>　　有少司令牌在身上，分身逃无可逃。
<br/>　　反正对于苏莫而言，一个分身而已，没有多大的损失。
<br/>　　轰！
<br/>　　巫尘的自爆非常恐怖，蕴含强大魔力的神丹爆开，强犹如核弹爆炸，升起了一团巨大的黑色蘑菇云，毁灭方圆万余里。
<br/>　　啊啊啊！！
<br/>　　五名巫族天才，瞬间便被蘑菇云吞噬，被炸的尸骨无存。
<br/>　　这几人都只有武帝境四重、或五重的境界，又毫无防备，哪里能挡住巫尘的自爆。
<br/>　　几乎是一瞬间，一切全部毁灭了，所有人都烟消云散。
<br/>　　一息！
<br/>　　两息！
<br/>　　三息！
<br/>　　仅仅是三个呼吸之后，巫尘自爆之地，空间突兀的波动了一下，一名高大的老者突然出现。
<br/>　　老者不是别人，正是大祭司。
<br/>　　大祭司脸色阴沉如水，静静在伫立在半空中，片刻之后，他大手猛然一探，直接插进了无尽虚空深处，跨越了无尽的距离。
<br/>　　唰！
<br/>　　少倾，大祭司收回了手掌，手中捏着一块少司令牌，正是巫尘的那块少司令牌，被他从无尽虚空深处找了回来。
<br/>　　“是谁？”大祭司的脸色，阴沉的几乎要滴出水来，没想到巫尘居然是一个奸细！
<br/>　　他早就发现巫尘有问题，但也没有太过在意，现在居然发生了这种事，导致族中大量天才惨死！
<br/>　　这是我送给你的礼物！
<br/>　　这是挑衅吗？
<br/>　　对方到底是谁？
<br/>　　大祭司心中有些疑惑，到底是何人敢来巫族撒野，简直不可饶恕，必须以灭族来偿还！
<br/>　　不过，这么多年来，他得罪的人实在是太多了，具体是何人，还需仔细彻查一番。
<br/>　　片刻之后，大祭司身形一闪，消失无踪。
<br/>　　接下来的时间里，整个天冥星都被封锁了，严禁任何人外出，陷入了紧锣密鼓的排查之中。
<br/>　　经过一段时间的调查，最终得出的结果，巫尘背后之人就是苏莫！
<br/>　　因为巫尘这段时间以来，不止一次侧面打探宏青璇之事，尤其是和巫长远的交谈之中，多次打探宏青璇的下落。
<br/>　　得知结果的大祭司，彻底暴怒了。
<br/>　　一个蝼蚁一般的土著，居然敢混进巫族之中，残杀巫族多名天才，简直胆大包天。
<br/>　　为此，大祭司派出了数百名武尊境的长老，前去追查苏莫的动向，誓要查出苏莫逃向何处。
<br/>　　……
<br/>　　经过几个月的时间，辗转了两百余颗星辰，苏莫和聂乾、韩盈三人，终于回到了古周星。
<br/>　　他们返回的路线，和去天冥星之时完全不同，故意兜了个圈子。
<br/>　　苏莫也是谨慎，怕被巫族之人追查到塔的逃走路线。
<br/>　　虽然这样可能也会被查到，但等到巫族查到之时，估计几年已经过去了。
<br/>　　“终于回来了！”走出古周星的星空传送大阵，聂乾满脸的唏嘘之色。
<br/>　　这一次历练，基本上是不费吹灰之力，就赚到了海量的财富资源。
<br/>　　“以后很长一段时间，都不用为资源发愁了！”韩盈亦是满脸笑意。
<br/>　　“两位，你们先回神武学府吧！我还有点事！”苏莫脸庞蠕动，恢复了本来的容貌，对两人笑着说道。
<br/>　　“苏兄，你不和我们一起回去？”两人有些诧异。
<br/>　　“不了，我过几日再回去！”苏莫道。
<br/>　　随即，苏莫向两人告别，身形冲天而起，向途山镇飞了过去。
<br/>　　此去天冥星，一来一回，足足耗费了将近两年的时间，他必须要去途山镇看看了。
<br/>　　疾驰在虚空之中，苏莫脑中思索，回到神武学府之后，必须要仔细了解一下太阴族。
<br/>　　青璇在太阴族，他日后一定要去寻找对方。
<br/>　　苏莫对于太阴族，完全的不了解，只知晓这是太古万族榜上排名第八十六的种族，非常的强大。
<br/>　　青璇被一名太阴族的女人带走，想必应该危险性不大了，毕竟，一个女人不可能拿青璇作为修炼炉鼎。
<br/>　　“苏莫，汝答应吾的事情，是不是可以兑现了？”
<br/>　　这时，阎魔鬼圣的声音在苏莫脑海中响起。
<br/>　　苏莫闻言微怔，随即顿时明白了，对方这是要自己放其自由啊！
<br/>　　不过，既然答应了对方，他自然不会食言。
<br/>　　嗖！
<br/>　　而后，苏莫身形下落，落在了下方的一片山林之中。
<br/>　　“前辈，这段时间你帮我良多，我自然不会食言！”站在山林之中，苏莫轻声说道。
<br/>　　言罢，他意念一动，立刻将阎魔鬼圣的魔魂从吞噬战魂中吐了出来。
<br/>　　阎魔鬼圣的魔魂高大魁梧，但此时，虚幻无比，仿佛是一道黑影，连面目都看不清了。
<br/>　　这是因为上次在天冥圣城之时，对方封住储物戒中的少司令牌，消耗了大量的魂力。
<br/>　　“苏莫，吾多谢了！”恢复自由的阎魔鬼圣，向苏莫微微抱拳，此时，不知为何，他并没有获得自由的喜悦。
<br/>　　见识了苏莫的逆天天赋之后，他知道自己离开苏莫，或许是天大的损失。
<br/>　　但是，相比于损失，还是自由更重要。
<br/>　　“前辈，应该是我多谢你才对，多谢你的帮助和指导！”苏莫摇了摇头说道，这短时间以来，对方的确是帮助他良多。
<br/>　　阎魔鬼圣叹了口气，略一沉吟，沉声道：“现在，吾还有一事，需要汝帮忙！”
<br/>　　“何事？”
<br/>　　“汝身上有很多巫族之人的肉身，给吾一具好点的吧！”阎魔鬼圣道。
<br/>　　“这个简单！”苏莫闻言一笑，随即一挥手，十几具巫族之人的肉身，出现了身前的地面上。
<br/>　　他身上巫族之人的肉身很多，别说对方只要一具肉身，就算要一百具，也没有什么问题。
<br/>　　“前辈，你自己选吧！”苏莫说道。
<br/>　　阎魔鬼圣瞥了一眼这十几具肉身，随即魔魂一闪，瞬间没入了一具肉身的体内，消失无踪。
<br/>　　苏莫见此，顿时有些愕然，因为对方所选的肉身，正是巫天御的肉身。
