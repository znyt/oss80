        “要培养新人进来，就得放弃爷爷带过的那批老员工。人事变动会很大，爷爷不肯辞退那些老员工，筱颖会怎么做？”傅夜沉嘴角微扬，意味深长地问道。
<br/>
<br/>        他之所以一直扩展集团旗下业务领域版图，也是为了招纳各界“新鲜的血液”进来中和掉公司里的那些“老血”。
<br/>
<br/>        因为顾及爷爷的情面不能辞退老员工，所以他一直是在养着爷爷经手时的一批工龄有十年以上的老员工，哪怕他们已经跟不上时代的步伐了，他也没忍心将他们辞退。
<br/>
<br/>        爷爷说，公司在，能养得起，就一直养着好了。
<br/>
<br/>        可他傅夜沉，不是慈善家，他会养着他们，但不会白养一群混日子的人，他早有对策，只是想听听筱颖的想法。
<br/>
<br/>        “大哥为何不把老员工，剥离出去？既然要养，那就要养得有价值点。大哥如果有足够的周转资金，不如再成立一个分公司，将爷爷的那批老员工渡过去，然后取消保底工资制度，实行多做多得，按业绩算分红与提成的工资制度。集团里如果有一个大业务，旗下有两个业务范畴相同的子公司竞争，就会让内部的员工有动力。这样一来，老员工们自食其力，就不会再止步不前了。”童筱颖慢条斯理地说。
<br/>

<br/>
<br/>        “我瞎说的，大哥别当真。”童筱颖尴尬地笑了笑，为了一个落以柔，她觉得自己是豁出去了，什么话都敢乱讲，“总而言之，大哥，你要好好重视落秘书，为己所用才对！”

<br/>        希望，郑忠威能买她这个好意，以后继续为大哥的公司卖力就好。

<br/>
<br/>        说来说去，筱颖的重点还是落以柔。
<br/>
<br/>        这事情是不是太巧合了点？
<br/>
<br/>        他要调走落以柔，筱颖就主动约他出来吃饭，还有意无意地跟他谈落以柔的事情。
<br/>
<br/>        三年前，筱颖可以“纵容”苏漫雪，现在是不是又可以“纵容”落以柔？
<br/>
<br/>        她到底是足够相信他，还是她压根就不在乎他？
<br/>
<br/>        话题到这里，童筱颖不再找其他话题，而是静默地拿起碗筷吃饭。

<br/>        晚饭过后，她被他牵着回到了傅氏集团大厦前坪的停车场。
<br/>
<br/>        “我们开一台车回去就行了。”童筱颖把自己今天开出来的车的车钥匙递给了傅夜沉。
<br/>
<br/>        傅夜沉接过车钥匙，打开了副驾驶座的车门，大手挡住车门顶框，防止她撞头，并非常有绅士风度地等她坐进去后，他去了后座，将那篮红玫瑰放进去后，才去了驾驶座。
<br/>
<br/>        傅夜沉上车后，却没急着发动车子，而是俯过身来，亲自替童筱颖系好安全带。
<br/>
<br/>        这个时候，他才看到她怀里抱着的那束几近枯萎的粉玫瑰。
<br/>
<br/>        “要不要扔了？”童筱颖问。

<br/>        傅夜沉抿了抿薄唇，俊脸上虽然没有任何表情，但是一双黑眸深邃又深情地凝视着她，耐人寻味地说道：“那天……我想拿这束花，哄你开心的，但是你不在家。”
<br/>
<br/>        童筱颖怀里抱着这束花，指尖摩挲着枯萎的花瓣，眸光晃动，随即黯然失色地垂下眼帘，却不曾动容地低声喃喃细语：“大哥，为什么在我最需要帮助的时候，出现的人不是你？

<br/>
<br/>        那个时候，大哥你又在哪里？

<br/>        在偌大的滨江城里，我找工作受挫，在夜店里卖唱，差点遭人侮辱，而我需要的第一份正经的工作，是昼景给的。

<br/>        那个时候，大哥你又在哪里？
<br/>
<br/>        我为了完成设计图，想加班，买笔记本电脑所需的钱，是昼景给的。

<br/>        那个时候，大哥你又在哪里？
