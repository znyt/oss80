　　这传闻之中的极道仙兵碎片，楚风眠猜想之中，多半可能性值得就是人皇罐碎片。
<br/>　　人皇罐一分为九，楚风眠现在得到了其中五枚碎片，剩下的四块，一些是没有出世，一些应该是被其他人所得到了。
<br/>　　看来这年轻男子，就是得到了其中一枚人皇罐碎片。
<br/>　　人皇罐碎片，只要是得到，就可以用来寻找其他人皇罐碎片，人皇罐碎片之中，有着一种独特的吸引力。
<br/>　　“不管了，追过去再说，跟在他们的后面什么好处也得不到，就算是得不到传闻之中的那一枚，得到了此人手中的人皇罐碎片，也足以让人皇罐威力增强无数倍！”
<br/>　　不管是这年轻男子到底是什么人物，这人皇罐碎片楚风眠一定要得到。
<br/>　　这人皇罐碎片现在仅仅只有五枚，都已经是楚风眠最强的底牌之一了。
<br/>　　每得到一枚人皇罐碎片，重塑其中，人皇罐的威力都会恐怖无数倍，在得到一枚，甚至是在得到两枚碎片的话。
<br/>　　就算是现在楚风眠的境界不晋升，不踏入帝尊境，凭借人皇罐就足以击伤九劫古帝。
<br/>　　更何况一旦是楚风眠境界晋升之后，凭借人皇罐，灭杀九劫古帝都有可能。想到这里楚风眠便是丝毫没有犹豫。
<br/>　　楚风眠大手一挥，将这水镜到，随后运转灵力，化为了一道遁光，向着那一群人飞走的方向追了过去。
<br/>　　那一群人虽然比起楚风眠早来了五个时辰的时间，可这一群人，应该是一边搜刮这赤红大陆上的宝藏，一边才走的。
<br/>　　楚风眠想要追上这一群人，并不困难。
<br/>　　一路上，所有的宝藏几乎都是被收取一空了，留下的东西，也都算是一些没有什么价值的东西。
<br/>　　楚风眠自然是明白，这是已经被那一群人给收取走了，这一群人的实力及其恐怖。
<br/>　　就算是八劫古帝，在这一群人的面前，都会被随意围杀，十几名古帝，其中四名八劫古帝，甚至是还有两名八劫巅峰古帝。
<br/>　　这样一群人的实力，就连九劫古帝见到，都只能够是退却，不敢应战。
<br/>　　在这赤红大陆之上栖息的神兽，荒兽等，根本不是这一群人的对手，都被连连轰杀。
<br/>　　大量的宝藏，都是落入到了他们的手中。
<br/>　　楚风眠催动遁光，足足是追了三个时辰的时间，在他的灵识之内，便是突然出现了这一群人的身影。
<br/>　　这一群人，几乎是出身自同一方势力，他们身上的灵力都是无比相近，修炼的是同一种功法。
<br/>　　楚风眠的灵识笼罩过去，这一群人之间的交谈，都被楚风眠清晰的听在耳中。
<br/>　　“君少，这归墟天可真是一处宝地，处处都是宝藏，那火麟，战龙等等神兽，几乎都是已经在我们哪里消逝了万年了，这里居然还有存在，这些神兽的血，都是大补之物啊，足以炼制圣丹！”
<br/>　　一群人之中，一名年轻古帝，兴奋的喊道。
<br/>　　他刚刚轰杀了一尊古帝层次的战龙，将这战龙的精血全部收取，炼制成丹，足以是让他的实力提升数成。
<br/>　　“这样一处宝地，我们严居然才知道，真的是失算了，据说陈家的人，比我们来的早得多，只怕是早就得到了大量的宝藏了。”
<br/>　　又一名年轻古帝开口道。
<br/>　　“哼，陈家？算什么东西，也配跟我们严家并称四大家族？就那么小猫两三只，我们想要灭了他们，轻而易举！”
<br/>　　那被称为君少的严君，冷笑连连的开口道。
<br/>　　“这一方赤红大陆，看来是我们所发现的，只要是我们扫荡了这赤红大陆上的所有宝藏，什么陈家，想灭就灭，就算是吴家，我们也能毁灭！倒时候整个星域，以我们严家为尊！”
<br/>　　这严君的语气嚣张无比，只要是一切没有臣服他的，都要被他所毁灭。
<br/>　　“君少无敌，等君少踏入了九劫古帝，下一任家主的位置，也定然是属于君少的！”
<br/>　　几名严家弟子不由兴奋的说道。
<br/>　　“等我成为了家主，你们未来都是严家的长老！还有严苍，未来大长老之位，必定是你的！”
<br/>　　那严君大笑起来，对着另一名八劫古帝巅峰的年轻男子说道。
<br/>　　那严苍只是点了点头，并没有开口。
<br/>　　“我们继续去扫荡，我感觉到那极道仙兵的碎片，也在这一座大陆之内，那一枚碎片我也要得到！”
<br/>　　“走！”
<br/>　　嗖嗖嗖。
<br/>　　一道道遁光继续向着赤红大陆的中心飞遁而去。
<br/>　　其中的为首的两名八劫巅峰古帝，分别被名为严君，严苍。
<br/>　　看来这一群人的中心，就是那君少，严君。
<br/>　　“严家？吴家，陈家？莫非这一群人是来自于阎罗星域四大家族之一的严家？”
<br/>　　楚风眠杀了陈家的一群人，也是抽取了他们的记忆，对于阎罗星域有着了一定的了解。
<br/>　　阎罗星域之中，一共是有着四大家族，分别是吴，严，慕，陈，四大家族，陈家最弱，吴家最强。
<br/>　　至于严家，虽然说是第二，但是隐隐约约之间已经是不亚于吴家，是阎罗星域之中崛起的最快的家族。
<br/>　　从这一群人的交谈之中来看，这一群人，多半是来自于阎罗星域的严家。
<br/>　　这看来严家比起陈家，绝对是要强得多，这一群人只是严家年轻一代的人物，都有着如此实力。
<br/>　　“看来只怕是阎罗星域的各大家族，也都来了九域了！”
<br/>　　楚风眠眉头紧锁，单单是大罗天星域的强者，都够让楚风眠头疼的了，现在不止是大罗天星域，阎罗星域也都插手在了其中了。
<br/>　　而且魔族也已经是大肆入侵了大罗天星域。
<br/>　　九域数万年前，都未曾发生过如此之大的动荡。
<br/>　　“跟上去看看！”
<br/>　　贸然动手，楚风眠想要以一人之力将这一群严家的人灭杀，可不容易。
<br/>　　那些七劫古帝，楚风眠不在乎，但是严君，严苍二人，可都是八劫古帝巅峰的人物。
<br/>　　楚风眠对他们出手，也是危险无比，稍有不慎，就可能会自身毁灭！
<br/>　　他的身躯一闪，化为一道遁光，刚刚追逐而去。
<br/>　　突然之间，一道拳风破空而来，这一道拳风的目标，直指楚风眠。
