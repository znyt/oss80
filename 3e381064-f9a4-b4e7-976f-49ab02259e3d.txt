　　褚彦望着不远处的苏莫，面带不屑之色。
<br/>　　“褚彦哥，我觉的这人彬彬有礼，还算挺不错的！”娇小女子憋了憋嘴道。
<br/>　　“沁儿，你还年轻，有所不知，越是这种彬彬有礼，一脸人畜无害模样的人，越是心机深沉，喜欢暗箭伤人！”褚彦告诫道。
<br/>　　“或许吧！”沁儿微微耸了耸肩，她并不完全赞同对方的话。
<br/>　　人心，怎么可能和面容有直接的关系呢？
<br/>　　随即，他也不再多言，安心等待时空之城开启。
<br/>　　苏莫伫立在人群之中，望着高空之上，那时空之城的两座门户，静静的等待。
<br/>　　据闻，时空之城，只有这两座门户显露在外，至于时空之城到底在哪里，没有人知晓。
<br/>　　有人说，时空之城，隐藏在无尽的空间深处。
<br/>　　也有人说，时空之城，隐藏在时间的长河之中，不在这片世界。
<br/>　　但具体如何，无人可知。
<br/>　　周围的武者越来越多，人数足有十几万之众，且大多数都是真神境的修为。
<br/>　　所有人都在等待，等待时空之城的开启。
<br/>　　嗡嗡~~~
<br/>　　良久之后，时空之城两扇紧闭的大门，突然光芒大放，光晕炫目。
<br/>　　大门，打开了。
<br/>　　“终于开了！”
<br/>　　“不知道这次有多人陨落？”
<br/>　　“陨落在所难免！
<br/>　　众人议论纷纷，不过，也并没有多少人太过激动，因为时空之城九天开启一次，不缺进入的机会。
<br/>　　嗖嗖嗖！
<br/>　　就在此刻，一道道身影，从两座门户之中飞了出来。
<br/>　　显然，这些人是在九天前进入的时空之城，现在则出来了。
<br/>　　“少爷，你终于出来了！”
<br/>　　“师兄，你没事吧？”
<br/>　　“张大哥，你有无进展！”
<br/>　　无数的人纷纷迎了上去，个个面带振奋之色。
<br/>　　此地的人虽多，但是，要进入时空之城的人，却是只有一部分。
<br/>　　待大量的人从时空之城出来，很多要进入的人，开始纷纷向两座门户之中飞去。
<br/>　　苏莫看到，那轮回神朝的一男两女，进入了时门之中。
<br/>　　时门和空门，没有任何的区别，但是，苏莫等了这么久，他早已清除哪座是时门，哪座是空门。
<br/>　　嗖！
<br/>　　苏莫身形腾空而起，同样向时门之中飞去，这是他早就做好了打算。
<br/>　　时门和空门，分别蕴含时间法则和空间法则，本来苏莫是想要进入空门的。
<br/>　　毕竟，他一直想要参悟空间奥义，但是，他选择进入时门，也是有原因的。
<br/>　　因为，时门之内，更加的玄妙，据闻有逆转时空之力。
<br/>　　苏莫想要看看，能不能靠这种神奇的力量，找到拯救芊寻月的方法。
<br/>　　至于空门，他若是想要进入，下一次再进也无不可。
<br/>　　嗖！
<br/>　　苏莫的身形，跟随着人流，进入了时门之中。
<br/>　　时间仿佛静止，思维仿似凝固，一切好似都不存在了。
<br/>　　不知道过去了多久，可能只是一瞬间，也可能是千百年。
<br/>　　苏莫恢复了过来，他发现，此时此刻，他正身处在一片古怪的建筑之前。
<br/>　　这些建筑，全部都是宫殿，仿似都是由虚空之力形成，显得虚幻而朦胧。
<br/>　　这些宫殿，足有三十三座，第一座最矮，高约十数丈，占地方圆数千丈。
<br/>　　第二座高约三十余丈，第三座高约五十余丈，后面一些宫殿，均是越来越高。
<br/>　　最后一座宫殿，如同擎天神柱，根本看不到其顶端高达几许。
<br/>　　三十三座宫殿，彼此相连，宛若是一个整体，散发着玄妙莫测的气息。
<br/>　　“这就是三十三重时光神殿？”苏莫眼眸之中露出惊讶之色。
<br/>　　之前，他早已听闻，时门之中，有三十三重时光神殿。
<br/>　　时光神殿之中，蕴含无比神奇的时光之力，玄妙而又莫测。
<br/>　　据闻，每个人在时光神殿之中，经历都不同，机遇也不同。
<br/>　　有人功参大道，领悟出一丝时间法则，有人把握住时间流速，千百年只在一瞬间，更有甚者畅游时空，去往过去和未来。
<br/>　　总之，时光神殿，蕴含时空至尊惊天的时间大道，无人能一窥究竟。
<br/>　　“走，我们去第一重神殿！”
<br/>　　“时间有限，九日后必须离开，我们走！”
<br/>　　“快进去吧！”
<br/>　　嘈杂的声音此起彼伏，进入时门的武者，足有万人以上。
<br/>　　嗖嗖嗖！
<br/>　　破空声不断，众人纷纷腾空而起，向第一重时光神殿之中飞去。
<br/>　　第一重时光神殿，大门大开，其内黑漆漆一片，根本无法窥探其内的真容。
<br/>　　眨眼之间，万名武者，便个个身形如电，极力向时光神殿飞去。
<br/>　　苏莫跟在众人的身后，同样向第一重时光神殿飞去。
<br/>　　不过，让苏莫无语的是，其他人很快便纷纷进入了第一重神殿之中，而他却是怎么也飞不到。
<br/>　　“怎么回事？”苏莫心中惊疑不定。
<br/>　　第一重神殿，明明就在前方数里之处，他呼吸之间，应该就能飞到。
<br/>　　而现在，他飞行了几个呼吸的时间，感觉距离并没有缩短，依旧距离第一重神殿数里。
<br/>　　“呵呵，可笑不自量！”
<br/>　　褚彦进入第一重神殿之前，回头看了苏莫一眼，不屑一笑。
<br/>　　进入时门之人，基本上都是真神境的修为，就算少数不是真神，也是很强大的虚神境四变武者。
<br/>　　而这个段惊天，修为气息，只是虚神境三变，居然敢进入时门，真是不知死活啊！
<br/>　　没有理会苏莫，褚彦和沁儿、骆妤一起，进入了已经光芒大放的第一重时光神殿之中。
<br/>　　“奇怪！”
<br/>　　苏莫停了下来，此时此刻，所有的人都是进入了第一重神殿之中，唯有他一人在外面。
<br/>　　“这些人肯定都知道原因，所以能这么快进去！”苏莫面色凝重了下来。
<br/>　　时空之城存在了这么久，九天开启一次，其它人肯定早已知道进入第一重神殿的方法，而他却是不知晓。
<br/>　　仔细观察了一番周围，苏莫没有发现什么异样，便再次向第一重神殿飞去。
<br/>　　但是，结果还是一样，神殿就在前方，他的速度很快，但是飞了半刻钟的功夫，都无法飞到。
<br/>　　“是空间的原因？”苏莫再次的停了下来，不过，很快他就否决了这个猜测，因为，这里是时门之内，只会与时间有关。
<br/>　　“时间？”苏莫面带思索之色，难道是时间便慢了？
<br/>　　感觉过去了很久，实则是时间的问题？根本没有过去这么久？
<br/>　　苏莫想不太明白，随即，他闭上了眼眸，意念固守心神，默默计算着时间，再次向第一重神殿飞去。
