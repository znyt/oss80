　　九天焚身诀提升到了第二重，叶尘的肉身强度比之前又提升了数倍，周围强大的高温，也顿时对他构不成任何威胁。
<br/>　　拨开周围的高温岩浆，叶尘的身体再次下沉。
<br/>　　大约又下沉了三四十米，周围岩浆的温度，已经达到了令人胆寒的一百五十万度！
<br/>　　“看来这次是没戏了！”
<br/>　　叶尘感受到自己的皮肤，已经又开始有了融化的趋势，不由得摇了摇头，准备打道回府。
<br/>　　然而，
<br/>　　就在这时
<br/>　　轰隆！
<br/>　　下方忽然传来一道剧烈的震动，继而下方的岩浆，竟然开始飞速旋转起来，然后形成了一个巨大的漩涡，犹如一条通往地底的通道。
<br/>　　“嗯？”
<br/>　　叶尘看到这一幕，顿时眉头微微一挑，思索了片刻之后，最终还是没能忍住自己的好奇心，直接一头扎入那通道之中。
<br/>　　哗！
<br/>　　沿着那条通道，又往下潜行了数十米，前方突然出现了一道紫色光幕。
<br/>　　“难怪之前我一直无法感应下面的情况，原来是被人设下了禁制！”
<br/>　　叶尘正要出手将这道禁制破开，不料那都光幕竟然自己裂开了一条缝！
<br/>　　叶尘虽然觉得有些诡异，不过都已经走到这一步了，他自然不可能放弃，于是一咬牙便踏入裂缝之中。
<br/>　　哗啦啦！
<br/>　　叶尘这一步踏出，顿时感觉周围的空间一阵变幻。
<br/>　　下一刻，
<br/>　　他发现自己，已经进入了一处神秘而且恢弘壮阔的宫殿之中。
<br/>　　“吼！！”
<br/>　　还没等叶尘反应过来怎么回事，忽然一道犹如金属撞击般的嘶吼声，在整个大殿之中回荡开来。
<br/>　　继而，
<br/>　　自那大殿深处，陡然窜出一条，长达近千米的赤色火龙！
<br/>　　那火龙气势滔天，而且携带着惊人的毁灭气息，所过之处，仿佛连空间都要崩塌了一般。
<br/>　　火龙自那宫殿深处窜出来之后，便直奔叶尘狠狠扑了过来。
<br/>　　“嗯？”
<br/>　　叶尘也不及多想，当即一拳砸了出去，
<br/>　　轰隆！
<br/>　　拳头狠狠砸落在那火龙的脑袋之上，后者顿时倒飞了出去。
<br/>　　而叶尘也不由得连连倒退，拳头一阵酸麻生疼，就仿佛打在了世上最坚硬的岩石之上，心中不由得一阵骇然。
<br/>　　要知道，以他如今的肉身强度，已经足以媲美仙器，即便是化神天君，恐怕也不敢硬抗他这一拳。
<br/>　　而那头火龙，只是在地上打了一个滚，便再次咆哮着，朝也扑了过来。
<br/>　　轰轰轰！
<br/>　　叶尘又一连十几拳打出，每一拳都落在那头火龙的身上，却愣是没能伤到它分毫。
<br/>　　“哼！”
<br/>　　一来二去，叶尘顿时也来了脾气，暗暗运转逆生百转秘术，将真元提升了数倍之后，再次一拳砸了出去，
<br/>　　“八极镇魔神拳！！”
<br/>　　轰隆隆！
<br/>　　八道硕大的拳头，从四面八方，将那头火龙包围在其中，
<br/>　　砰！砰！砰！砰！
<br/>　　“嗷！！”
<br/>　　那火龙同时被来自八方的拳头击中，顿时发出一道哀嚎，
<br/>　　继而，“彭！”的一声，
<br/>　　火龙那长达千米的庞大身躯，竟然破碎开来，自那火龙的腹中，竟然飞出一把三米来长的残破大刀，静静地悬浮在宫殿中央上空。
<br/>　　“果不其然！”
<br/>　　叶尘见状，顿时脸上露出一抹浓浓的喜色。
<br/>　　其实刚刚他和那火龙交手几次之后，便已经大致猜到，方才的火龙，应该灭世魔刀的器灵所化。
<br/>　　达到仙器级别以上的法宝，已经具备不低的灵智。
<br/>　　到了这个时候，已经不是人来挑选法宝，而是法宝来挑选人，只有用自己的实力，获得法宝的认可，法宝器灵才会认可你这个主人。
<br/>　　只不过，让叶尘没有想到的是，眼前那把长刀的形象，看上去实在太普通了，甚至有些惨不忍睹，就像是少了一半似得，居然是一把残刀！
<br/>　　即便是叶尘，如果不是刚刚和它交过手，都根本不会注意到它！
<br/>　　“这就是大名鼎鼎的灭世魔刀吗？也太寒酸了吧？”
<br/>　　叶尘忍不住吐槽了一句。
<br/>　　嗡嗡~~~
<br/>　　空中那把长刀，似乎听懂了叶尘这话，顿时一阵剧烈的颤动，似乎在表达着自己强烈的不满。
<br/>　　叶尘暗暗好笑，连忙话锋一转，又道：
<br/>　　“虽然看上去又点破，不过的确是一件难得旷世神兵！”
<br/>　　那长刀听到这话，顿时发出一道清脆的振鸣声，显然十分欢喜。
<br/>　　叶尘哈哈一笑，感觉自己应该已经得到了灭世魔刀的认可，于是直接虚手一抓，
<br/>　　“来吧！从今以后，就由我来做你的新主人！！”
<br/>　　叶尘自信满满，本以为能够手到擒来。
<br/>　　却不料，
<br/>　　那灭世魔刀竟然一阵巨颤，拼命的挣扎起来，似乎并不愿意成为他的兵刃。
<br/>　　叶尘顿时眉头微微一皱，显然没有想到，这灭世魔刀，竟然如此难以收服！
<br/>　　“怎么？你是想再和我打一场吗？”
<br/>　　叶尘故意脸色一板，再次挥舞起自己的拳头。
<br/>　　就在这个时候，
<br/>　　宫殿深处，陡然再次响起了一道声音，充满着无尽的沧桑。
<br/>　　“有缘人！你想要成为灭世魔刀的主人，还须答应本座一个条件！！”
<br/>　　声音落下，
<br/>　　一道高大的金色虚影，从宫殿深处缓步走了出来。
<br/>　　这是一名须发尽白老者，不过并非实体，应该只是一道残魂，但饶是如此，却透着一股难以形容的强悍毁灭气息，让人不由自主的感到心底发寒，仿佛他一个眼神，便可以将整个世界毁灭一般。
<br/>　　叶尘也不由得微微一惊，不过很快便恢复了淡然之色，
<br/>　　“阁下应该就是灭世仙君吧？晚辈叶尘拜见！”
<br/>　　叶尘恭恭敬敬冲着那老者行了一礼，不过脸上无悲无喜，姿态也不卑不亢。
<br/>　　倘若这灭世仙君还活着，此时的叶尘，就算是一百个绑在一块，肯定也不会是对方的对手。
<br/>　　但，仅仅只是一道残魂而已，还不足以对叶尘构成威胁。
<br/>　　当然了，灭世仙君乃是几千年的声名赫赫的大人物，叶尘作为后辈，自然也不能失了礼数。
<br/>　　那金色虚影老者看到叶尘的表现，眼眸中顿时露出一抹欣赏之色，
<br/>　　“好！好得很呐！本座等了两千多年，终于等到一个还算不错的传承者了！”
<br/>　　“年轻人，你可愿意入我门下？”
