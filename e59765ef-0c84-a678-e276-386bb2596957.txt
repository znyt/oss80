        见童筱颖不吭声，苏漫雪不依不饶地接着撒娇：“想想我们俩曾经是那么要好的闺蜜，无话不谈。在我生病的时候，也是你任劳任怨地照顾着我。筱颍，我错了。你可不可以，再给我一次机会呀？”

<br/>        没错，她在她面前装可怜，而不是在跟她诚恳地在道歉。

<br/>        童筱颖看着苏漫雪的这张脸，心灰意冷地将自己的手从苏漫雪的手中抽了出来。

<br/>        “我去做早餐了。”童筱颖淡淡地应了声，眼神里也透着一股子的清冷。

<br/>        语毕，她转身去了厨房。
<br/>
<br/>        苏漫雪瘪了瘪红唇，恶狠狠地瞪着童筱颖进厨房的背影。

<br/>        从今以后，她决定改变自己的计划。
<br/>
<br/>        私宅里的几个人，陆陆续续地在餐桌前坐下。
<br/>
<br/>        苏漫雪就坐在童筱颖的对面，她一边吃着自己盘子里的早餐，一边仔细观察着童筱颖的一举一动、一神一态。
<br/>
<br/>        既然，自己的脸都整容成她那个样子了，就连声音都跟她童筱颖有那么七八分相似，那么，索性连她童筱颖这个人，她也一起跟着模仿了。

<br/>        傅夜沉既然爱的是童筱颖这个人，那么她苏漫雪就完完全全变成童筱颖这样的人，不就可以取代童筱颖在傅夜沉心里的地位了吗？

<br/>        傅氏集团以后是傅夜沉的，她若是把傅大少奶奶的位子坐稳了，那么，傅氏集团以后也是她苏漫雪的。
<br/>
<br/>        不仅傅氏集团，连带着傅家那座上百年的大庄园，古董级的房子，也都是她的。
<br/>
<br/>        为了以后那享之不尽的荣华富贵，她现在必须得隐忍。
<br/>
<br/>        苏漫雪说改品行，还真就开始改品行。
<br/>
<br/>        她不再对刘婶呼来喝去，说话开始客客气气起来。
<br/>
<br/>        童筱颖和傅夜沉去上班了，连华生也外出有事去了，苏漫雪便让刘婶收拾其他屋子，傅夜沉的房间由她来亲自收拾打理。
<br/>
<br/>        就算他傅夜沉的心是石头做的，只要她苏漫雪坚持不懈，总有一天，她会把他的心给捂热的吧！
<br/>
<br/>        刘婶见这位傅大少奶奶能想通这件事情，看着苏漫雪非常有耐心地替大少爷整理内务，不禁欣慰地笑了。
<br/>
<br/>        知错能改善莫大焉，希望这位傅大少奶奶和她家大少爷早日修成正果。
<br/>
<br/>        其实，刘婶看得出，大少爷喜欢的人是二少奶奶，但是二少奶奶毕竟已经是二少爷的合法妻子了。
<br/>
<br/>        所以，刘婶还是希望大少爷能放弃二少奶奶，好好地和这位傅大少奶奶过日子。
<br/>
<br/>        她是过来人，有这样的想法，自然是为了他们这几个年轻人好。
<br/>
<br/>        一场激烈的竞争，在一间偌大的会议室里展开。

<br/>        傅昼景和季思妍，领着自己的团队，拿着自己的标书，向甲方介绍自己施工队伍的实力。
<br/>
<br/>        既然现在，他已经有了自己的建筑公司，那么他就要把自己的公司壮大发展到最好。
<br/>
<br/>        男人，就应该要有野心。

<br/>        季思妍一脸崇拜地看着傅昼景，在这次竞标会上，将对手一一碾压，那气宇轩昂的风采，十足的男神气派。

<br/>        傅昼景看着童筱颖的照片，嘴角情不自禁地微微上扬。
