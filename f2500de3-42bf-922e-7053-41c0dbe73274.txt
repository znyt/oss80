　　第133章
<br/>　　李善言出现后横隔在二人中间，在他左右，程大雷和熊老大都拎着斧头。
<br/>　　“二位英雄，莫要动手，且莫伤了两家和气。”李善言。
<br/>　　程大雷将斧头交给身后秦蛮，斜睨着熊老大道：“今日若不是看李军师面子，我非一斧劈了你不可。”
<br/>　　“凭你。”熊老大语气十分不屑。
<br/>　　“怎么，我同神仙学艺，会五雷法的事情你没听说过。再废话，小心我一道雷劈死你。”看熊老大的样子肯定属于四肢发达，头脑简单的哪一种，程大雷干脆比划着手势，虚张声势的吓唬他。
<br/>　　“找死！”熊老大眉头拧成一个疙瘩。
<br/>　　“二位英雄的名字，幽州谁人不知，谁人不晓，即便主公也常提起二位。主公本打算今天见见二位，无奈身染风寒，只好委托我主持今日的盛事。”
<br/>　　“幽州王也知道我！”熊老大忽有几分骄傲，想想也是，杨龙停是幽州万民之主，自己是群贼之首，他知道自己也没什么奇怪。
<br/>　　“经常说起呢。”李善言爽朗笑道。
<br/>　　熊老大也咧咧嘴，突然看到程大雷竖起两个手指，冲着自己晃个不停。
<br/>　　“你干嘛？”
<br/>　　“运功。”
<br/>　　“运功干嘛？
<br/>　　“请雷。”过程中，程大雷比出的手势一直没放下。
<br/>　　“混蛋，你还想害我！”熊老大勃然大怒。
<br/>　　绿林道刀口舔血，最信这些玄里虚里的东西。关于程大雷会五雷法的事情他也听说过，只是听说得不多。可这种事，越是听说得不多，越是容易信以为真。
<br/>　　李善言拦住熊老大，冲程大雷挥挥手：“程寨主，收了神通吧。”
<br/>　　“看我天雷啊！”
<br/>　　程大雷大叫一声，冲熊老大伸手一指。熊老大下意识倒退一步，竟抬头看了看天空，发现没有丝毫动静。
<br/>　　此时程大雷哈哈大笑，已经坐到风雨亭中。熊老大看着他，眼中已经要喷出火来。
<br/>　　“诸位英雄，我今日奉命到此处，为得是请各位为国效力。只要大家答应在主公帐下听用，从前种种一概抹去，保诸位有个大好前程，未来封妻荫子也未必不可。”
<br/>　　李善言开口，点到今日的主题。他说完观察着诸人的反应，这些乌合之众也并不是傻瓜，他们绝不会痛快答应这件事，一定会想法设法提各种各样的要求。
<br/>　　然后你来我往，就是言语和智力上的交锋。
<br/>　　“好！”
<br/>　　“好！”
<br/>　　两道声音突然响起，高举手的是程大雷和老道郭烦人。二人目光相对，都有些发愣。
<br/>　　“为国为民，死不足惜。”程大雷。
<br/>　　“保国安邦，吾辈当先。”郭烦人高喊。
<br/>　　咦！
<br/>　　二人目光对视，颇有种英雄相惜的感觉，也有点遇到同行的意思。
<br/>　　程大雷迫不及待冲李善言道：“李军师，我早有报效帝国的心，自幼读圣贤书，想的不就是为国为民做些事么。如果可能，我真想把心挖出来，让你看看我的心是多么炙热。”
<br/>　　李善言倒是有些发愣，招安对山贼来说终不是太光彩的事情，为了今日的事，他准备了很多言语。可程大雷这意思，他答应得也太爽快了吧……
<br/>　　其他众贼也对程大雷和郭烦人的行为表示了鄙夷，这人没骨气也不该没骨气到这种地步吧。
<br/>　　程大雷仍喋喋不休：“李军师，你要是不信，我就让你看看我的心，我这颗心啊……”
<br/>　　“程寨主，不必激动，不必激动，且让我把话说完。”李善言清了清嗓子，道：“这次招安，主公准备将诸位英雄组成一军，从诸位中间选出两位左右将军。”
<br/>　　这时代的军衔极其复杂，统领三军可以叫做将军，管一二十人也能叫将军，慢慢『将军』成了对军人对尊称。至于杨龙停拿出这左右将军有多大权力，估计也是画大饼对意思多些。
<br/>　　“不用说了！”程大雷一挥手：“这左将军对官职我应下来了，必不负所托。”
<br/>　　“……”李善言满头黑线：“我没说是你吧。”
<br/>　　熊老大冷哼一声：“你也配么？”
<br/>　　“二位，二位。”李善言道：“且莫急着争执，这次不仅要为大家抹去匪籍，兵器装备也一应俱全，我家主公对诚意，大家是绝对可以放心的。”
<br/>　　终还是要想法设法说服这些人为己所用，那么大饼就不介意画得圆一些。只是看众人的意思，似乎还有些犹豫，不是每个人都像程大雷那样干脆。
<br/>　　“呔，你们这帮土匪山贼，给脸不要脸，还真把自己当个人了么。”
<br/>　　此刻，一道洪亮的声音突然响起，众人扭头看去，只见一年轻人大步向风雨亭行来，他身穿皮甲，腰间佩着长剑。
<br/>　　李善言一惊：“你怎么来了！”
<br/>　　这人自然就是杨龙停的义子罗铁，幽州城有名的混人。他拍着腰上佩剑叮当作响，口中大笑。
<br/>　　“我怎么不能来，谁又困的住我，哈哈。”
<br/>　　李善言：“快把他架回去，别让他在这里捣乱！”
<br/>　　“谁敢碰我！”罗铁锵地拔出佩剑，道：“我怎么是来捣乱，军师，我是来帮你做事。”
<br/>　　说着，他横剑指过在场的绿林群贼：“你们这些是敬酒不吃吃罚酒，今日只要赢得我手中这柄快剑，去留随意。”
<br/>　　罗铁的剑很快，这是幽州城有目共睹的事情。本来李善言见到这混世魔王过来，心中就一咯噔，但转念一想，这些绿林贼人，目中无人，让他们见识见识厉害也好。
<br/>　　“快剑，有多快？”马老五突然道，握住了腰上的短刀。
<br/>　　“很快的快！”
<br/>　　罗铁忽然出剑，左手甩出三枚铜板，只听锵锵之音，铜板一一落地。
<br/>　　大家再看地上的铜板，三枚铜板皆从中间被劈开，变成六瓣。
<br/>　　“你们今天谁能斩开三枚铜板，今日的事情我就不管啦！”罗铁趾高气扬。
<br/>　　众人盯着地上的铜板，下意识都摇摇头，一剑三斩，的确好快。
<br/>　　程大雷拍拍秦蛮的肩膀，又冲高飞虎和徐神机递个眼色。
<br/>　　“走啦，走啦。”
<br/>　　“大当家，不瞧会了么。”
<br/>　　“瞧什么，又不是看杂耍，风紧扯乎，扯乎。”
<br/>　　
<br/>(本章完)
